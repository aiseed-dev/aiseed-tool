歴史栽培料理

サヤインゲン料理ガイド

〜自家栽培の豆を最高に美味しく食べる知恵〜

1. 日本とイタリア、決定的な「食べ方」の違い

日本の栽培者さんが最も驚くのが、「火入れ」の概念です。

  • 🇯🇵 日本のスタイル：
    • 「シャキシャキ感（歯ごたえ）」を重視。
    • たっぷりの湯で短時間（2〜3分）茹で、冷水に取る「色止め」が基本。
  • 🇮🇹 イタリアのスタイル：
    • 「クタッとするまで（Tenerezza）」煮込むのが美徳。
    • 15分〜30分以上加熱することも珍しくありません。
    • 目的は食感ではなく、「豆のサヤに、オイルやソースの旨味を吸わせること」です。

ヴェネト地方には「テガ・ニョッカ（ニョッキのように柔らかいサヤ）」という褒め言葉があるほど。自家栽培の新鮮な豆なら、煮崩れる寸前のトロトロの状態こそが、イタリア流の贅沢なのです。

2. 自然の味を活かす！地域別・伝統レシピ

畑で採れたてのサヤインゲンを使って、イタリアのマンマ（お母さん）の味を再現してみましょう。

🥘 北イタリア（ヴェネト州）の蒸し煮

【テゴリーネ・イン・テッチャ (Tegoline in Tecia)】

「テッチャ」は方言で「鍋」のこと。お湯で茹でるのではなく、鍋の中で野菜の水分とオイルを乳化させながら蒸し煮にする技法です。特に平サヤインゲン（モロッコインゲンなど）におすすめです。

  • 材料: サヤインゲン、ニンニク、オリーブオイル、トマトピューレ（少量）、水
  • 作り方:
    1. 厚手の鍋にオリーブオイルと潰したニンニクを入れ、香りを出す。
    2. 生のままのサヤインゲンを投入し、油を馴染ませる。
    3. 少量のトマトピューレ（または刻んだトマト）と、ひたひたより少なめの水を加える。
    4. 蓋をして弱火で20〜30分、クタクタになるまで蒸し煮にする。
    5. 水分が飛び、サヤがソースを吸って艶やかになったら完成。

🍅 南イタリア（ナポリ・プーリア州）のトマト和え

【コルネッティ・アル・ポモドーロ (Cornetti al Pomodoro)】

太陽の味がする南部の定番。こちらは煮込みすぎず、フレッシュなトマトソースで「和える」感覚です。丸サヤのインゲンによく合います。

  • 材料: サヤインゲン、ミニトマト（完熟）、ニンニク、オリーブオイル、バジル（大量）
  • 作り方:
    1. サヤインゲンは下茹でしておく（イタリア風にするなら柔らかめに）。
    2. フライパンでニンニクとオリーブオイルを熱し、半分に切ったミニトマトを入れて強火で炒め、ソース状にする。
    3. 茹でたインゲンを加え、ソースを絡める。
    4. 火を止めてから、手でちぎったバジルをたっぷりと加える。
  • 楽しみ方: お皿に残った美味しいソースは、パンで拭って（スカルペッタ）最後の一滴まで楽しみます。

🍋 ローマ（ラツィオ州）のシンプルサラダ

【ファジョーリニ・アッラグロ (Fagiolini all'Agro)】

「アッラグロ」は酸味を効かせたスタイルのこと。採れたてで香りの良い、最高品質の豆が手に入った時はこれが一番です。

  • 材料: サヤインゲン、レモン汁（またはワインビネガー）、上質なオリーブオイル、塩
  • 作り方:
    1. インゲンを塩茹でする。
    2. ザルにあげ、水にはさらさずそのまま粗熱を取る（水っぽくなるのを防ぐため）。
    3. 温かいうちに、レモン汁とたっぷりのオリーブオイルを回しかけて馴染ませる。
    4. 冷めると味が染み込み、より美味しくなります。

3. 収穫祭の楽しみ！保存食（コンセルヴァ）の技法

収穫最盛期、食べきれないほどのサヤインゲンが採れたら、イタリアの伝統的な保存食にして冬の楽しみにしましょう。

🫙 オイル漬け (Fagiolini Sott'olio)

イタリアの家庭に必ずある常備菜です。アンティパスト（前菜）や、ワインのおつまみに最高です。

  1. 下処理: インゲンの筋を取り、洗い、適当な長さに切る。
  2. 茹でる: 水と白ワインビネガーを「2:1」の割合で鍋に入れ、沸騰させる。インゲンを入れ、少し歯ごたえが残る程度（5〜7分）茹でる。
    • ポイント: お酢で茹でることで保存性を高め、シャキッとした食感を残します。
  3. 乾燥（最重要）: ザルにあげた後、清潔な布巾の上に広げ、半日ほど陰干しして完全に水分を飛ばす。
    • 注意: 水分が残っているとカビやボツリヌス菌の原因になります。
  4. 瓶詰め: 煮沸消毒した瓶にインゲンを詰める。お好みで「刻んだニンニク」「ミントの葉（またはオレガノ）」「唐辛子」を挟む。
  5. 注ぐ: インゲンが完全に隠れるまで、上質なオリーブオイルを注ぐ。空気を抜いて密閉し、冷暗所で1ヶ月ほど寝かせてから食べる。

🍂 ドライ保存（乾燥豆への移行）

もし収穫のタイミングを逃してサヤが硬くなってしまったり、中の豆が膨らんでしまった場合でも、捨てないでください！

  • マンジャトゥットの終わり: サヤが硬くなったら、そのまま畑で（あるいは収穫して軒下で）カラカラになるまで乾燥させます。
  • 種子を食べる: サヤから豆を取り出せば、自家製の「乾燥インゲン豆」になります。冬の間のスープ（ミネストローネ）の具材として大活躍します。

4. 栽培者さんへのアドバイス

  • 品種の個性:
    • 平サヤ（モロッコ系）: ヴェネト風の「蒸し煮」に最適。味がよく染みます。
    • 紫インゲン: 茹でると緑になりますが、柔らかいのでサラダ向きです。
    • 丸サヤ: トマト煮込みやオイル漬けで食感を楽しんでください。
  • 収穫のタイミング:
    • 現代の品種は「スジなし（Senza filo）」が多いですが、固定種や在来種を育てている場合は、スジが硬くなる前の若採りが基本です。
    • 少し育ちすぎたかな？と思ったら、迷わず「クタッとなるまで煮込む」イタリア式調理法を選んでください。驚くほど美味しくなります。

ご自身の畑で育ったサヤインゲン。
「シャキシャキ」だけでなく、イタリア流の「トロトロ」や「オイル漬け」で、その可能性をすべて味わい尽くしてくださいね。

Buon Appetito!（召し上がれ！）
