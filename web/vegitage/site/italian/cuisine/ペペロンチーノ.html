ペペロンチーノ料理ガイド

1. 日本とイタリア：食べ方の決定的な違い

日本の家庭料理におけるトウガラシは、主に「乾燥させたものを輪切りにして、辛味だけを油に移す」使い方が主流です。しかし、本場イタリア（特に南部）ではアプローチが異なります。

  • 「野菜」として食べる: 辛味の少ない品種や未熟果は、シシトウやピーマンのように炒めたり揚げたりして、果肉そのものを味わいます。
  • 「香り」を食べる: 辛さ（カプサイシン）だけでなく、品種ごとのフルーティーな香りや、ハーブのようなニュアンスを重視します。
  • 「色」を食べる: 料理全体を鮮やかな赤（カーマインレッド）に染め上げ、食欲と生命力を刺激します。

3. 自然栽培の味を活かす！イタリア郷土料理レシピ

化学肥料を使わず育てたトウガラシは、エグみが少なく、クリアな旨味があります。そのポテンシャルを引き出すシンプルなレシピをご紹介します。

【バジリカータ州】 ペペローニ・クルスキ (Peperoni Cruschi)

～驚きのサクサク食感！「赤い黄金」のスナック～

セニーゼ種のような「皮が薄く、辛くない（または微辛）」品種が豊作だった場合にぜひ試してほしい、バジリカータ州の象徴的な料理です。

  • 適した品種: セニーゼ、または皮の薄い甘トウガラシ
  • 作り方:
    1. 収穫したトウガラシを完全に乾燥させます（カラカラになるまで）。
    2. ヘタと種を取り除きます。
    3. フライパンに多めのオリーブオイルを熱します（中温）。
    4. 乾燥トウガラシを投入し、数秒間だけ揚げます（焦げやすいので注意！色が鮮やかになったらすぐ引き上げる）。
    5. 冷めると、ポテトチップスのように驚くほど「サクサク」になります。
  • 食べ方: そのままスナックとして、または砕いてパスタや目玉焼きのトッピングに。

【カンパニア州】 フリアリエッロの素揚げ (Friarielli Fritti)

～ナポリの定番、採れたて未熟果の楽しみ～

「フリアリエッロ」と呼ばれる小型品種を使いますが、一般的なトウガラシの「緑色の未熟果」でも代用可能です。間引き菜の有効活用にも最適です。

  • 適した品種: フリアリエッロ、またはトウガラシの未熟果（緑色）
  • 作り方:
    1. トウガラシを洗い、水気をよく拭き取ります（ヘタはついたままでOK）。
    2. フライパンにオリーブオイルとニンニクを熱し、トウガラシを丸ごと投入します。
    3. 皮がしんなりして、少し焦げ目がつくまでじっくり炒め揚げにします。
    4. 仕上げに良質な塩を振るだけ。
  • ポイント: 同名の葉野菜（チーマ・ディ・ラーパ）とは別物です。ナポリではトマトソースと絡めることもあります。

【プーリア州】 トウガラシの詰め物 (Peperoncini Ripieni)

～サレント地方の伝統、7年トウガラシのオイル漬け～

少し丸みのある中辛品種におすすめ。保存がきくので、たくさん収穫できた時の常備菜として最高です。

  • 適した品種: 7年トウガラシ、チェリータイプ（丸型）
  • 作り方:
    1. トウガラシのヘタを切り、中身（種とワタ）をスプーンでくり抜きます。
    2. 鍋に酢と水（1:1）、塩少々を入れて沸騰させ、トウガラシを数分茹でます（殺菌と食感調整）。
    3. 布の上に逆さに置いて、一晩しっかり乾かします（水分厳禁）。
    4. ツナ、ケッパー、アンチョビをフードプロセッサーでペースト状にします。
    5. トウガラシの中にペーストを詰め、煮沸消毒した瓶に入れます。
    6. トウガラシが完全に隠れるまでオリーブオイルを注ぎ、冷暗所で1ヶ月ほど熟成させます。

【アブルッツォ州】 真のアーリオ・オーリオ・エ・ペペロンチーノ

～品種の個性をダイレクトに味わう～

アブルッツォ州では「ラッツァレット」のような香りの良い品種が好まれます。自然栽培のトウガラシなら、乾燥させたものを粉砕せず、手でちぎって使うのがおすすめです。

  • 適した品種: ラッツァレット、カイエンナ系の香り高い品種
  • 作り方:
    1. フライパンにオリーブオイル、潰したニンニク、そして種ごとちぎった乾燥トウガラシを入れます。
    2. 極弱火でじっくりとオイルに香りと辛味を移します（焦がさないのが鉄則）。
    3. 茹で汁を加えて乳化させ、パスタと絡めます。
  • ポイント: シンプルだからこそ、トウガラシ自体の「甘い香り」や「大地の味」が際立ちます。

4. 収穫を無駄にしない：伝統的な保存技法

イタリアの農家は、夏の実りを冬まで楽しむために様々な保存術を持っています。

1. セルテ (Serte) - 天日干しのネックレス

南イタリアの家の軒先に吊るされている、あの赤い首飾りのことです。
* 方法: 収穫したトウガラシの柄（ヘタの軸）の部分に、太い針と丈夫な糸を通し、数珠つなぎにします。
* 効果: 風通しの良い日陰や壁面に吊るすことで、カビを防ぎながら長期保存できます。インテリアとしても美しく、使う分だけハサミで切って料理に使います。

2. クレマ・ディ・ペペロンチーノ (Crema di Peperoncino) - 生ペースト

カラブリア州の知恵。乾燥させず、生のフレッシュさを塩とオイルで閉じ込めます。
* 方法: 生の激辛トウガラシを種ごとフードプロセッサーにかけ、重量の約3%の塩と混ぜます。ザルにあけて一晩置き、余分な水分を抜きます（重要）。その後、瓶に詰め、オリーブオイルをひたひたに注ぎます。
* 用途: ブルスケッタに塗ったり、肉料理に添えたり。ンドゥイヤ（豚肉とトウガラシのペースト）のような使い方ができます。

3. オリオ・サント (Olio Santo) - 聖なるオイル

直訳すると「聖なる油」。食卓に欠かせない辛味オイルです。
* 方法: 完全に乾燥させたトウガラシ（粉末または粗挽き）を瓶に入れ、高品質なオリーブオイルを注ぎます。数週間置くと、美しい赤色と香りが移ります。
* 注意: 必ず「完全に乾燥した」トウガラシを使ってください。水分が残っているとボツリヌス菌のリスクがあります。

ご自身の手で育てたトウガラシは、イタリアの市場で売られているものにも負けない、力強い味を持っているはずです。ぜひ、日本の薬味的な使い方を超えて、イタリアのマンマのように「野菜としてのペペロンチーノ」を堪能してください。
