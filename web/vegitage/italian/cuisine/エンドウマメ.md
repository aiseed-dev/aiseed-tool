日本の自然栽培農家や家庭菜園家の皆様へ。

ご自身で育てたエンドウマメ（実エンドウ、ウスイエンドウ、グリーンピース）の収穫シーズンは、まさに「緑の宝石」を手にする喜びの瞬間かと思います。

日本では「豆ご飯」や「卵とじ」、あるいは彩りとしての塩茹でが一般的ですが、イタリアにおいてエンドウマメ（Piselli）は、パスタやリゾットの主役を張る堂々たる野菜です。

提供された記事に基づき、イタリアの伝統品種の個性を活かした、栽培者だからこそできる「エンドウマメのイタリア伝統料理ガイド」を作成しました。

# 栽培者のためのイタリア流エンドウマメ活用ガイド

## 1. 日本とイタリア：食べ方の決定的な違い

まず、調理のアプローチを変えてみましょう。

*   **日本のアプローチ（彩りと食感）**
    *   豆を別茹でして、最後に混ぜることが多い。
    *   鮮やかな「緑色」を残すことを重視する。
    *   プリッとした食感を楽しむ。

*   **イタリアのアプローチ（出汁と甘み）**
    *   **「ソフリット（香味野菜）」と一緒に最初から煮込む**ことが多い。
    *   豆の色が悪くなっても気にしない。むしろ、豆から出る甘みとデンプン質がソースと乳化することを重視する。
    *   **「鞘（サヤ）」も出汁（ブロード）に使う**（これが栽培者の特権です！）。

## 2. 品種と料理のマリアージュ

記事にある通り、イタリアのエンドウには「生食できるほど甘いもの」から「煮込みに適した乾燥用」まで多様な個性があります。お手持ちの品種に近いタイプを選んで調理法を変えてみましょう。

| 豆のタイプ | 近いイタリア品種 | おすすめの料理法 |
| :--- | :--- | :--- |
| **極甘・皮が薄い** | **Biso di Borso** (ヴェネト)<br>**Pisello di Lavagna** (リグーリア) | **生食、リジ・エ・ビジ**<br>加熱は短時間に留め、繊細な甘みを活かす。 |

| **小粒・乾燥豆** | **Nano di Zollino** (プーリア)<br>**Roveja** (ウンブリア) | **スープ、ピューレ**<br>長時間煮込んでクリーミーな食感を楽しむ。 |

## 3. イタリア各地の代表的郷土料理レシピ

自然栽培で育てた生命力あふれるエンドウマメを味わい尽くす、3つの代表的なレシピを紹介します。

### ① ヴェネト州：ドージェ（総督）の愛した「リジ・エ・ビジ (Risi e Bisi)」
*甘みの強い品種（ウスイエンドウなど）に最適*

これはリゾットでもスープでもない、その中間の「オール・オンダ（波打つような）」とろみが特徴の料理です。

**【栽培者のためのポイント】**
捨ててしまいがちな**「若くて柔らかい鞘（サヤ）」**を煮出して、ブロード（出汁）を取ってください。これが味の決め手です。

1.  **ブロード作り**: 剥いた後のサヤを鍋に入れ、水と少量の塩で20分ほど煮出し、ミキサーにかけて濾します（または単に煮出した汁を使います）。これが濃厚な緑の出汁になります。
2.  **ベース作り**: 鍋にオリーブオイルとバター、刻んだパンチェッタ（またはベーコン）、玉ねぎを入れて弱火でじっくり炒めます。
3.  **煮込み**: 生のマメと米（洗わない）を投入し、軽く炒めます。
4.  **仕上げ**: 作っておいた「サヤの出汁」を少しずつ加えながら、米がアルデンテになるまで煮ます。最後にパセリとパルメザンチーズを加え、マンテカーレ（空気を含ませるように混ぜる）して完成。

### ② カンパニア州：ナポリの定番「パスタ・エ・ピセッリ (Pasta e Piselli)」
*少し皮がしっかりした品種や、中晩生種に最適*

ナポリでは「アッゼッカータ（Azzeccata）」と呼ばれる、ねっとりと絡みつくような仕上がりが好まれます。

**【シンプルイズベスト】**
1.  鍋にオリーブオイル、ニンニク、刻んだパンチェッタを入れて炒めます。
2.  たっぷりのエンドウマメを入れ、ひたひたの水を加えて豆が柔らかくなるまで煮ます。
3.  **ここがポイント**: 別の鍋でパスタを茹でるのではなく、**豆を煮ている鍋に直接ショートパスタ（ミスタ・コルタなど）を投入**します。
4.  パスタが水分と豆の旨味を吸い込みながら茹で上がります。水分が飛び、クリーム状になったら完成。ペコリーノチーズと黒胡椒をたっぷりと。

### ③ プーリア州：乾燥豆の煮込み「チェカマリティ (Cecamariti)」
*収穫遅れで硬くなった豆や、乾燥させた豆に最適*

「夫の目をくらます（ほど美味しい）」というユニークな名前の料理。本来は「Pisello Riccio」などの伝統品種を使いますが、完熟した豆でも再現できます。

1.  乾燥豆（または完熟豆）を一晩水で戻します。
2.  土鍋（あれば）に豆、水、ローリエ、玉ねぎ、トマト少々を入れて、薪の残り火（弱火）で豆が崩れるほど柔らかくなるまで煮込みます。
3.  仕上げに揚げたパン（クルトン）と、たっぷりの生オリーブオイルを回しかけます。
4.  野菜の甘みだけの、驚くほど滋味深いスープになります。

## 4. 保存食の伝統技法：旬を閉じ込める

一斉に収穫期を迎えて食べきれない場合は、イタリアのマンマの知恵で保存しましょう。

### エンドウマメのオイル漬け (Piselli Sott'olio)
冷凍保存が普及する前の伝統的な保存法です。アンティパスト（前菜）や、冬場のパスタの具材になります。

**【材料】**
*   エンドウマメ（新鮮なもの）
*   白ワインビネガー：水 ＝ 1：1
*   塩、黒胡椒（粒）、ローリエ、ニンニク
*   上質なオリーブオイル

**【作り方】**
1.  鍋にビネガー、水、塩を入れて沸騰させます。
2.  エンドウマメを入れ、再沸騰してから3〜4分（硬さを見ながら）茹でます。
3.  清潔な布の上に広げ、**完全に水分を乾かします**（水分が残るとカビの原因になります）。
4.  煮沸消毒した瓶に豆を詰め、間にローリエ、ニンニク、黒胡椒を挟みます。
5.  豆が完全に隠れるまでオリーブオイルを注ぎます。気泡が入らないように注意し、冷暗所で1ヶ月ほど馴染ませてから食べます。

## 5. 最後に：歴史を食べる

記事にあったように、エンドウマメはかつて「貧者の肉」であり、同時に「王侯貴族の熱狂」の対象でもありました。

ご自身で育てたエンドウマメを食べる時、ナポリの農民のようにクタクタに煮込んでパンと一緒に食べるのも良し、ヴェネツィアの貴族のように新鮮な甘みを生で楽しむのも良しです。

特に自然栽培のエンドウは、化学肥料を使ったものに比べて「エグみ」が少なく、豆本来の野性味と甘みのバランスが良いはずです。ぜひ、サヤから出したての生の豆を数粒、畑でそのまま味わってみてください。それが、11,000年の歴史を持つこの作物の、最も贅沢な楽しみ方かもしれません。

どうぞ、イタリアの風を感じながら、豊かな収穫の食卓をお楽しみください。
