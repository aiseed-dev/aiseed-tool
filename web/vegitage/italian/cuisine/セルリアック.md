# 畑から食卓へ：イタリア流セルリアック（セーダノ・ラーパ）の楽しみ方

イタリアでは「セーダノ・ラーパ（Sedano Rapa）」と呼ばれ、特に北イタリアの冬の食卓に欠かせないセルリアック。
日本ではまだ「セロリ味の芋」といった認識で、スープやサラダの脇役になりがちですが、本場イタリアでは**「主役級の冬野菜」**であり、時には**「癒やしの食材」**としても扱われます。

自然栽培でじっくり育てたセルリアックは、香りが凝縮され、肉質も緻密です。そのポテンシャルを引き出すイタリアの知恵をご紹介します。

---

## 1. イタリアの伝統と日本の食べ方：ここが違う

日本の一般的なレシピ（フレンチの影響が強い「レムラード」など）と、イタリアの伝統的なアプローチには明確な違いがあります。

| 特徴 | 日本での一般的な食べ方 | **イタリアの伝統的な流儀** |
| :--- | :--- | :--- |
| **香り** | クセとして消そうとすることが多い | **精油成分（d-リモネン等）を「芳香」として楽しむ** |
| **食感** | 生のシャキシャキ感が中心 | **加熱して「ねっとり・ホクホク」させる食感も重視** |
| **役割** | サラダやスープの具材 | **肉料理の付け合わせ（コントルノ）や出汁のベース** |
| **部位** | 白い実の部分のみ使用 | **皮や端材も「ソッフリット」として旨味出しに使う** |

イタリア、特にヴェネト州の農家にとって、セルリアックは単なる野菜ではなく、厳しい冬を乗り越えるための**「栄養と香りのカプセル」**なのです。

---

## 2. 北イタリアの郷土料理を味わう

記事にも登場した、名産地ならではの食べ方をご紹介します。

### ① ヴェローナ風サラダ (Insalata di Verona)
**地域：ヴェネト州**
フランスの「セロリ・レムラード」に似ていますが、イタリアではよりシンプルに、素材の味を活かします。
*   **特徴:** マヨネーズだけでなく、**「マスタード」の酸味**と**「レモン」**を効かせるのがヴェローナ流。
*   **食べ方:** 千切りにした生のセルリアックを和え、少し寝かせて味を馴染ませます。戦時中の修道院では、これを栄養源として擦り下ろして食べていたという歴史があります。

### ② セルリアックのピュレ (Purè di Sedano Rapa)
**地域：北イタリア全般**
ジャガイモのピュレとは一味違う、大人の付け合わせです。
*   **特徴:** 茹でるのではなく、**蒸して（または少量の牛乳で煮て）**柔らかくし、バターと共に滑らかにします。
*   **相性:** ヴェネト州名物の豚肉ソーセージ「コテキーノ」や、脂の乗ったローストポークの下に敷くと、セルリアックの爽やかさが脂っこさを中和し、絶妙なハーモニーを生みます。

### ③ ヴェルタータ (Vellutata)
**地域：トレンティーノ / アルプス地方**
「ビロードのような」という意味のクリームスープ。
*   **特徴:** 玉ねぎやジャガイモと共にクタクタに煮込み、ブレンダーで撹拌します。
*   **ポイント:** 寒い山岳地帯の料理です。仕上げにオリーブオイルを回しかけ、クルトンを浮かべて「食べるスープ」として楽しみます。

## 3. 自然栽培の味を活かす「シンプル・レシピ」

肥料を控えて育てた自然栽培のセルリアックは、えぐみが少なく、甘みが強いのが特徴です。その個性をダイレクトに味わうレシピです。

### 【生で味わう】セルリアックのカルパッチョ
記事にある通り、古代ローマでは「薬草」でした。そのフレッシュな生命力を頂きます。
1.  皮を厚めに剥き、スライサーで極薄にスライスする。
2.  皿に並べ、良質な**EXVオリーブオイル**と**岩塩**、**レモン汁**をかける。
3.  仕上げに**砕いたクルミ**とパルミジャーノチーズを散らす。
    *   *ポイント：記事にある「ナッツのような風味」が、本物のナッツと共鳴して香りが倍増します。*

### 【焼いて味わう】セルリアックのアル・フォルノ（オーブン焼き）
1.  皮付きのまま（またはアルミホイルで包んで）、丸ごとオーブンでじっくり焼く。
2.  串がスッと通るまで火を通したら、カットして塩とオリーブオイルだけで食べる。
    *   *ポイント：茹でると逃げてしまう水溶性の香り成分やカリウムを、内部に閉じ込めます。驚くほど甘くなります。*

### 【隠し味として】冬のソッフリット (Soffritto Invernale)
イタリア料理の基本、香味野菜の炒めベース「ソッフリット」。
通常はセロリの茎を使いますが、冬場は**「セルリアックのみじん切り」**に変えてみてください。
*   ミネストローネやラグーソースのコクと深みが劇的に変わります。
*   形が悪いものや、皮に近い部分の有効活用として最適です。

## 4. 長期保存の知恵：イタリアの保存食（コンセルヴァ）

かつては砂の中に埋めて保存していましたが、現代の家庭で楽しめるイタリア伝統の保存法をご紹介します。収穫量が多い時にぜひお試しください。

### ① オイル漬け (Sedano Rapa Sott'olio)
イタリアのアンティパスト（前菜）の定番です。
1.  セルリアックを拍子木切り、またはサイコロ状にカットする。
2.  **白ワインビネガーと水（1:1）**に塩を加え、少し歯ごたえが残る程度に数分茹でる（これが殺菌と食感の決め手）。
3.  ザルにあげ、清潔な布巾の上で**完全に乾かす**（水分はカビの原因になります）。
4.  消毒した瓶に詰め、ニンニク、黒胡椒、ローリエと共に、具材が隠れるまでオリーブオイルを注ぐ。
    *   *1ヶ月後くらいからが食べ頃です。*

### ② 酢漬け (Sedano Rapa Sott'aceto)
さっぱりとしたピクルス風。
1.  薄切りにしたセルリアックを塩もみして水分を抜く。
2.  熱した甘酢（ワインビネガー、砂糖、水、スパイス類）を注ぐ。
    *   *肉料理の脂を流す付け合わせとして重宝します。*

### ③ ドライ・ブイヨン (Essiccato / Dado vegetale)
記事中で言及されたペッレグリーノ・アルトゥージの『厨房の科学』にも通じる、旨味の凝縮法です。
1.  セルリアックを薄くスライスし、天日またはフードドライヤーでカラカラになるまで乾燥させる。
2.  ミルで粉末にする。
3.  塩と混ぜて**「自家製セロリ塩」**にするか、玉ねぎや人参のドライパウダーと混ぜて**「即席ブイヨンの素」**にする。
    *   *スープに一振りするだけで、本格的なイタリアンの香りが立ち上ります。*
