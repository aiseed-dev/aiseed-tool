自然栽培でフダンソウ（スイスチャード）を育てている皆さま、こんにちは。
畑で元気に育つフダンソウ、日本ではお浸しや炒め物が定番ですが、実はイタリアでは**「Bieta（ビエタ）」**と呼ばれ、古代ローマ時代から愛され続けている歴史ある野菜です。

かつては「鍛冶屋の食事（労働者の質素な飯）」と揶揄されたこともありましたが、今ではその栄養価と多様な品種が再評価され、イタリアの食卓に欠かせない「緑の遺産」となっています。

ここでは、ご自身で育てたフダンソウを余すことなく、イタリア流に美味しく食べるためのガイドをお届けします。

# イタリア流「フダンソウ（Bieta）」の伝統料理ガイド
～畑の恵みをイタリアの知恵で味わう～

## 1. 日本とイタリア：食べ方の決定的な違い

日本のフダンソウ料理とイタリア料理の最大の違いは、**「油（オイル）」と「クタクタ感」**にあります。

*   **日本のアプローチ（シャキシャキ・水煮）**
    *   サッと茹でて冷水にとり、醤油や出汁で食べる。
    *   食感（歯ごたえ）を残すことが良しとされる。
*   **イタリアのアプローチ（クタクタ・オイル煮）**
    *   **「Remotio nocumenti（害の除去）」の思想：** 中世の医学書『タクイヌム・サニタティス』には、フダンソウの「えぐみ」や「冷やす性質」を中和するために、**油やスパイスと共に調理すべき**と記されています。
    *   しっかりと茹でた後、さらにオリーブオイルとニンニクで炒め煮（リパッサータ）にするのが基本。
    *   野菜の繊維が崩れるくらい柔らかくすることで、甘みと旨みを引き出します。

**★栽培者へのアドバイス**
自然栽培で育ったフダンソウは味が濃厚です。イタリア流にしっかりと火を通し、良質なオリーブオイルをたっぷり合わせることで、土の香りが「旨み」へと変わります。

## 2. 地域別：イタリアの郷土料理とフダンソウ

イタリアは南北に長く、地域ごとに全く異なるフダンソウ料理が存在します。畑で採れた品種やサイズに合わせて使い分けてみてください。

### 【北イタリア・リグーリア州】
**料理名：トルタ・パスクアリーナ（Torta Pasqualina）**
*   **特徴：** 復活祭（イースター）に食べる象徴的なパイです。
*   **使い方：** 主に葉の部分（Erbette/エルベッテ）を使います。
*   **ここが面白い：** 伝統的にはパイ生地を**33層**（キリストの生涯の年数）重ねます。中にはフダンソウとリコッタチーズ、そして**生の全卵**を割り入れて焼き上げます。カットすると現れるゆで卵の断面が「再生」を象徴しています。

### 【北イタリア・エミリア＝ロマーニャ州】
**料理名：エルバッツォーネ（Erbazzone）**
*   **特徴：** 「貧者の料理」の精神が息づく、塩味の野菜パイです。
*   **使い方：** 葉だけでなく、**白い茎（コスタ）**を主役にします。
*   **ここが面白い：** 方言で茎を「Scarpa（靴）」と呼ぶことから、かつては「スカルパッツォーネ」と呼ばれました。貴族が葉を食べた残りの茎を、農民が美味しく食べるために工夫したのが始まりです。パン生地に茎と葉の炒め物をたっぷり詰め込みます。

### 【北イタリア・ヴェネト州】
**料理名：バッサーノ風ビエタ（Bietola di Bassano）**
*   **特徴：** バッサーノ・デル・グラッパ周辺の食べ方。
*   **使い方：** **「根っこ」**まで食べます。
*   **ここが面白い：** この地域の品種は根がカブのように肥大します。根と葉を一緒に蒸したり茹でたりして、甘い玉ねぎと共に食べます。
*   **★栽培者メモ：** もし間引きを忘れて根が太くなってしまったフダンソウがあれば、捨てずに根の皮を剥いて食べてみてください。驚くほど甘みがあります。

### 【南イタリア・プーリア州＆シチリア州】
**料理名：野生ビエタの煮込み（Gira / Bietola di campagna）**
*   **特徴：** 栽培種ではなく、自生する野生種（原種に近いもの）を好みます。
*   **使い方：** 味の濃い葉を、トマトや豆と一緒にクタクタに煮込みます。
*   **ここが面白い：** プーリア州では「ソラマメのピューレ」の付け合わせとして、苦味のある野生フダンソウのオイル煮が添えられます。シチリアでは「Gira（ジーラ）」と呼ばれ、ニンニクとトマトで煮込みます。

## 3. 素材の味を活かすシンプル・レシピ

収穫したてのフダンソウを味わうための、最も基本的かつ美味しいイタリア家庭料理のレシピです。

### ① ビエタ・リパッサータ（Bieta Ripassata）
～フダンソウのニンニクオイル炒め～
「Ripassata」は「もう一度鍋に通す」という意味。一度茹でてから炒めるのがコツです。

1.  フダンソウは茎と葉に分け、塩茹でする（茎は長めに、葉はさっと）。
2.  水気を**手でしっかりと絞る**（ここが重要！水っぽいと美味しくありません）。
3.  フライパンに多めのオリーブオイル、潰したニンニク、唐辛子（お好みで）を入れ、弱火で香りを出す。
4.  茹でたフダンソウを入れ、オイルを吸わせるように中火で炒め合わせる。
5.  塩で味を調える。

**★ポイント：** これが基本形です。肉料理の付け合わせに最高ですし、パスタと和えれば立派なプリモ・ピアットになります。

### ② コスタのグラタン（Coste al Gratin）
～太い茎を主役にするご馳走～
茎が立派に育つ「スイスチャード」タイプにおすすめです。

1.  太い茎（コスタ）を5cm幅に切り、柔らかくなるまで茹でる。
2.  耐熱皿にバターを塗り、茹でた茎を並べる。
3.  上からベシャメルソース（ホワイトソース）をかけ、パルミジャーノチーズをたっぷり振る。
4.  オーブンで表面がきつね色になるまで焼く。

**★ポイント：** トロトロになった茎は、まるでアスパラガスやアーティチョークのような上品な風味になります。「茎こそがご馳走」だと実感できる一品です。

## 4. イタリアの保存食（Conservazione）の知恵

たくさん収穫できた時は、イタリアのマンマの知恵で保存食にしましょう。

### フダンソウのソットーリオ（Sott'olio / オイル漬け）
イタリアではあらゆる野菜をオイル漬けにします。フダンソウの太い茎はこれに最適です。

1.  **下準備：** 茎を一口大に切る。
2.  **酢茹で：** 鍋に「水：白ワインビネガー＝1：1」の液を沸かし、塩少々を入れる。茎を入れて3〜5分（少し歯ごたえが残る程度）茹でる。
    *   *※酢で茹でることで保存性が高まり、ボツリヌス菌のリスクを抑えます。*
3.  **乾燥：** 清潔な布巾の上に広げ、数時間〜半日置いて**完全に水分を飛ばす**。
4.  **瓶詰め：** 煮沸消毒した瓶に茎を詰め、ニンニク、黒胡椒、ローリエなどを加える。
5.  **注油：** 高品質なオリーブオイルを、具材が完全に隠れるまで注ぐ。気泡が入らないように注意。
6.  **熟成：** 冷暗所で1ヶ月ほど馴染ませてから食べる。

### 冷凍保存（トルテッリの具材用）
1.  葉を柔らかく茹でて、水気を極限まで絞る。
2.  ゴルフボール状に丸めてラップし、冷凍する。
3.  使う時は解凍し、リコッタチーズやパルミジャーノと混ぜれば、イタリア風ラビオリ（トルテッリ）の中身や、オムレツの具材としてすぐに使えます。
