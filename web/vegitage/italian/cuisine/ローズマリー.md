自然栽培や家庭菜園で野菜を育てる皆様へ。

ご自身で育てた「ローズマリー」を、単なる「飾りの葉」で終わらせていませんか？
イタリアにおいてローズマリーは、料理の風味を決定づける「建築的な土台」であり、時には主役にもなる力強い野菜です。

提供された解説記事に基づき、イタリアの伝統的な知恵と、栽培者だからこそ楽しめる贅沢な食べ方をガイドにまとめました。

# イタリア伝統野菜ガイド：ローズマリー（Rosmarino）
**〜育てた香りを余すことなく味わう、イタリアの知恵〜**

## 1. 日本とイタリア：使い方の決定的な違い

日本の家庭料理では、ローズマリーは「最後に少し乗せる飾り」や「臭み消し」として控えめに使われがちです。しかし、イタリアのマンマ（お母さん）たちの使い方はもっと大胆で、機能的です。

*   **「香り」ではなく「味」の一部：**
    イタリアでは、調理の最初（ソフリット）からニンニクやオイルと共に加熱し、油に香りを完全に移します。これにより、ローズマリーは具材の一部として料理全体に統合されます。
*   **枝（木質部）も調理器具：**
    葉だけでなく、硬くなった枝も捨てません。串焼きの串（スピエディーニ）として使ったり、炭火の中に投げ込んで燻製香をつけたりと、植物全体を使い切ります。
*   **脂の消化を助ける機能性：**
    豚肉や羊肉など脂の多い肉料理に必ず合わせるのは、単に美味しいからだけではなく、ローズマリーが持つ「脂肪の酸化を防ぎ、消化を助ける（肝臓ケア）」という薬効が、経験的に知られているからです。

## 2. 品種と風土に合わせた料理の使い分け

イタリアの伝統的農産物（PAT）のデータに基づき、ローズマリーのタイプによる使い分けをご提案します。ご自身の畑のローズマリーがどちらのタイプに近いか、想像しながら選んでみてください。

### 【北イタリア・冷涼タイプ】
**イメージ：ロズマリーノ・ディ・モンテヴェッキア（ロンバルディア州）**
*   **特徴：** 段々畑で育ち、寒暖差がある環境。香りが繊細で上品。
*   **おすすめの料理：**
    *   **リゾット（Risotto）：** お米、バター、パルミジャーノチーズだけのシンプルなリゾットに。繊細な香りが乳製品のコクを引き立てます。
    *   **白身肉のロースト：** 鶏肉やウサギ肉、白身魚など、淡白な素材の風味を殺さずに引き立てます。

### 【南イタリア・乾燥タイプ】
**イメージ：ロズマリーノ・ディ・ヴィッラローザ（シチリア州）**
*   **特徴：** 強烈な日差しと乾燥した岩場で育つ。精油成分が非常に濃厚で、パンチがある。
*   **おすすめの料理：**
    *   **赤身肉・ジビエ料理：** 羊（ラム）やイノシシなど、個性の強い肉料理に負けない力強さがあります。
    *   **トマト煮込み：** 南イタリア風の濃厚なトマトソースや、カポナータのアクセントに。

## 3. 栽培者だけが楽しめる「イタリア郷土料理」レシピ

収穫したての新鮮なローズマリーがあるからこそ作れる、現地のマニアックかつ絶品な料理を紹介します。

### ① ナポリの驚き「ローズマリーのフリット」
*(Rosmarino fritto / Napoli)*
「えっ、そのまま食べるの？」と驚かれるかもしれませんが、ナポリの貧しい人々の知恵が生んだ傑作です。新芽の柔らかい部分を使います。

1.  **収穫：** 春先などの、先端の柔らかい新芽（5〜7cmほど）を摘みます。
2.  **衣づけ：** 小麦粉と卵、少量の水で緩めの衣（パステッタ）を作ります。
3.  **揚げる：** ローズマリーを衣にくぐらせ、高温の油でカラッと揚げます。
4.  **仕上げ：** 熱いうちに塩を振ります。
*   **味：** サクサクの衣の中でハーブの香りが爆発します。ビールや白ワインのおつまみに最高です。

### ② リグーリアの野草料理「プレボッジオン風ソテー」
*(Preboggion / Liguria)*
畑の雑草（タンポポやノゲシなど）や葉野菜を美味しく食べるための、リグーリア州の技法です。

1.  **準備：** 畑で採れた葉野菜（フダンソウ、ホウレンソウ、野生のルッコラなど）を茹でて水気を絞ります。
2.  **ソフリット：** フライパンにオリーブオイル、潰したニンニク、**たっぷりのローズマリー**を入れて弱火で熱し、香りを引き出します。
3.  **炒める：** 茹でた野菜を投入し、オイルを絡ませるように炒めます。
4.  **隠し味：** 最後に**少量の「お酢」**を回しかけて酸味を飛ばします。
*   **ポイント：** ローズマリーとお酢の組み合わせが、野菜の土臭さを消し、驚くほど洗練された味に変えます。

### ③ 北イタリアの極致「ローズマリーのリゾット」
*(Risotto al rosmarino / Lombardia)*
具材なし。お米とハーブの香りを食べる、禅のような一皿です。

1.  **ブロード：** 野菜くずでとった出汁（または湯）の中に、ローズマリーの枝を一本入れて煮出し、香りを移しておきます。
2.  **調理：** 玉ねぎのみじん切りをバターで炒め、お米（洗わない）を加えて透き通るまで炒めます。
3.  **煮る：** 1のブロードを少しずつ加えながら、お米をアルデンテに煮ます。
4.  **仕上げ：** 火を止め、刻んだ生のローズマリー（極少量）とバター、パルミジャーノを加えて混ぜ合わせ（マンテカトゥーラ）、蓋をして2分蒸らします。

## 4. 保存の知恵（伝統的保存食）

たくさん収穫できた時は、イタリアの伝統的な方法で保存しましょう。

### 伝統的な「日陰干し」 (Essiccazione all'ombra)
シチリアのヴィッラローザ（PAT認定地）では、直射日光を避けて乾燥させます。
*   **方法：** 風通しの良い日陰に束ねて吊るします。
*   **理由：** 直射日光は葉を変色させ、揮発性の精油成分を飛ばしてしまいます。日陰でゆっくり乾燥させることで、生に近い鮮やかな緑色と香りをキープできます。

### ローズマリー・ソットーリオ (Sott'olio)
「オイル漬け」です。調味料としてだけでなく、漬けたオイル自体が極上のソースになります。
1.  洗って完全に水気を切ったローズマリー（半日ほど乾かすと良い）を用意します。
2.  清潔な瓶にローズマリーを詰め、全体が浸かるようにEXVオリーブオイルを注ぎます。
3.  お好みで、唐辛子、黒胡椒の粒、ニンニク一片を一緒に漬け込みます。
    *   **使い方：** 1ヶ月後からが食べ頃。肉を焼く前のマリネ液や、仕上げの香りづけに。

### 魔法の調味料「ローズマリー塩」 (Sale aromatico)
古代ローマ人が「記憶のハーブ」と呼んだ香りを閉じ込めます。
1.  生のローズマリーの葉と、粗塩を 1:3 程度の割合で用意します。
2.  フードプロセッサーで撹拌するか、すり鉢で葉が細かくなるまで混ぜ合わせます。
3.  瓶に入れて保存。
    *   **使い方：** 天ぷら、フライドポテト、肉のグリルに。これだけでイタリアの味になります。

ご自身で育てたローズマリーは、スーパーで売られているパック詰めとは生命力が違います。ぜひ、葉を刻むだけでなく、枝ごと煮込み、花を味わい、その香りを胸いっぱいに吸い込んでみてください。それは3000年の歴史を持つ、イタリアの風景そのものの香りです。
