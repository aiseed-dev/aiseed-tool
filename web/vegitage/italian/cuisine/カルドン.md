自然栽培で野菜を育てる皆様、こんにちは。
畑でひときわ存在感を放つ巨大なアザミ、「カルドン」。その生命力あふれる姿を収穫したものの、「どうやって食べれば一番おいしいのか？」と迷われてはいませんか？

アーティチョークの野生種に近いカルドンは、イタリアでは**「冬の王様」**と呼ばれるほど愛されている野菜です。その魅力は、独特のほろ苦さと、手をかけることで生まれる深い甘味、そしてシャキシャキとした食感にあります。

ここでは、イタリアの歴史と伝統に基づいた、カルドンを楽しみ尽くすための料理ガイドをご紹介します。

# イタリア伝統野菜「カルドン」料理ガイド
**～土の力を食卓へ～**

## 1. はじめに：イタリア流と日本流の違い
日本でフキやゴボウなどの繊維質な野菜を調理する場合、私たちはよく「醤油と出汁」で煮含めます。しかし、カルドンの野性味あふれる風味は、**「油脂（オリーブオイル・バター）」**や**「乳製品（チーズ・ミルク）」**と合わせることで真価を発揮します。

*   **日本のイメージ:** アク抜きをして、さっぱりと煮る。
*   **イタリアの流儀:** アク（苦味）を旨味に変えるため、動物性のコク（アンチョビ、チーズ、肉の出汁）をぶつける。あるいは、小麦粉やミルクを使って茹でこぼし、まろやかさを引き出す。

## 2. 下処理の鉄則（全レシピ共通）
カルドンは「変色」との戦いです。包丁を入れた瞬間から黒ずんでいくため、準備が肝心です。

1.  **レモン水を用意:** 大きなボウルに水を張り、レモン汁（または酢）をたっぷり入れます。
2.  **筋を取る:** セロリやフキのように、茎の表面にある硬い筋をナイフで剥ぎ取ります。
3.  **産毛を取る:** 表面の白い産毛も苦味のもとになるため、こそげ落とします。
4.  **すぐに水へ:** 5〜10cmの長さに切ったら、即座にレモン水へ放り込みます。

> **栽培者へのアドバイス：**
> 畑で「軟白栽培（遮光）」をしていないカルドンは、野生味が強く苦いです。その場合、下茹での際に**「少量の小麦粉」**と**「レモン汁」**を熱湯に加えると、色が白く仕上がり、アクも抜けやすくなります。

## 3. イタリア各地の郷土料理レシピ

### 【ピエモンテ州】バーニャ・カウダ (Bagna Cauda)
記事にもある通り、ピエモンテ州、特にニッツァ・モンフェッラート周辺では、カルドンは「生」で食べるのが最高のご馳走です。

**自然栽培野菜のポイント:**
もし畑で土寄せや遮光をして「軟白化」させた株なら、ぜひ生で。そうでない場合は、牛乳で柔らかく煮てからこのソースをつけてください。

*   **材料:**
    *   カルドン（中心部の柔らかい茎）
    *   ニンニク：1株（たっぷり使います）
    *   アンチョビ（フィレ）：50g〜
    *   EXVオリーブオイル：100ml〜
    *   バター：少々（まろやかさを出す隠し味）
*   **作り方:**
    1.  小鍋にみじん切りにしたニンニクと牛乳少々（分量外）を入れ、ニンニクが柔らかくなるまで煮ます（臭み消し）。
    2.  牛乳を捨て、オリーブオイルとアンチョビを加え、弱火でアンチョビを溶かします。
    3.  最後にバターを溶かし、熱々のソースを卓上へ。
    4.  冷水で締めてパリパリにした生のカルドンを、熱いソースに浸して頬張ります。

### 【トスカーナ・エミリア地方】カルドンのグラタン (Cardi Gratinati)
16世紀の料理長スカッピも推奨した、カルドンとチーズの黄金の組み合わせ。苦味とベシャメルソースの甘味が絶妙に調和します。

*   **材料:**
    *   下茹でしたカルドン
    *   ベシャメルソース（ホワイトソース）
    *   パルミジャーノ・レッジャーノ（粉チーズ）
    *   バター
    *   ナツメグ（重要！）
*   **作り方:**
    1.  下茹でしたカルドンを耐熱皿に並べます。
    2.  たっぷりのベシャメルソースをかけます。ナツメグを効かせると本格的になります。
    3.  パルミジャーノを雪のようにたっぷりと振りかけ、バターのかけらを散らします。
    4.  200℃のオーブンで表面が黄金色（狐色）になるまで焼き上げます。

### 【アブルッツォ・モリーゼ地方】カルドンのスープ (Zuppa di Cardi)
イタリア中部から南部にかけて、クリスマスの定番料理です。鶏の出汁で煮込むことで、カルドンが滋味深い味わいになります。

*   **材料:**
    *   カルドン
    *   鶏ガラスープ（ブロード）
    *   卵：2個
    *   パルミジャーノチーズ
    *   小さなミートボール（牛豚合挽き）
*   **作り方:**
    1.  鶏のブロードの中で、下処理したカルドンと小さなミートボールを柔らかくなるまで煮込みます。
    2.  ボウルに卵とパルミジャーノを混ぜ合わせておきます。
    3.  スープが沸騰しているところに卵液を回し入れ、「かき玉汁」のようにふんわりと仕上げます。
    4.  体に染み渡る、冬の養生スープの完成です。

## 4. 素材の味を活かすシンプル調理

### カルドンのフリット (Cardi Fritti)
自然栽培で育てた野菜の力強さをダイレクトに感じるなら、揚げ物が一番です。ローマ時代から愛される調理法です。

*   **作り方:**
    1.  カルドンは少し硬さが残る程度に下茹でし、水気をしっかり拭き取ります。
    2.  小麦粉を薄くまぶし、溶き卵にくぐらせます（パン粉をつける派とつけない派がいますが、卵だけの衣が伝統的で軽やかです）。
    3.  オリーブオイルでカリッとなるまで揚げ、熱いうちに美味しい塩とレモンを絞って食べます。
    *   **味の特徴:** 外はカリッ、中はトロッとした食感になり、アーティチョークのような香りが口いっぱいに広がります。

## 5. 保存食の知恵

### カルドンのオイル漬け (Cardi Sott'olio)
たくさん収穫できた時は、イタリアのマンマの知恵「ソットーリオ（オイル漬け）」で保存しましょう。春まで楽しめます。

*   **材料:**
    *   カルドン
    *   白ワインビネガー：水 ＝ 1：1
    *   塩、粒胡椒、ローリエ、ニンニク
    *   ひまわり油またはオリーブオイル
*   **作り方:**
    1.  鍋にビネガー、水、塩を入れて沸騰させ、筋を取って一口大に切ったカルドンを入れます（約3〜5分、歯ごたえが残る程度に茹でる）。
    2.  清潔な布巾の上に広げ、数時間〜半日ほど置いて**完全に乾かします**（水分が残るとカビの原因になります）。
    3.  煮沸消毒した瓶にカルドンを詰め、ニンニク、胡椒、ローリエを加えます。
    4.  カルドンが完全に隠れるまでオイルを注ぎます。気泡が入らないように注意し、冷暗所で1ヶ月ほど寝かせると食べ頃です。

## むすびに
カルドンは、種を撒いてから収穫まで長い時間を要し、調理にも手間がかかる野菜です。しかし、その手間こそが、古代ローマから現代に至るまでイタリアの人々を魅了し続けてきた理由でもあります。

ご自身で育てたカルドンを、熱々のバーニャカウダソースに浸したり、チーズと共に焼き上げたりする時、皆様の食卓にはイタリアの冬の風景が広がることでしょう。どうぞ、その「苦味」と「甘味」のハーモニーを存分にお楽しみください。
