日本の自然栽培・有機栽培に取り組む皆様へ。

イタリア野菜と聞いて、トマトやズッキーニを思い浮かべる方は多いでしょう。しかし、南イタリアの食文化の魂（Anima）とも言える野菜、それが**「ペペロンチーノ（トウガラシ）」**です。

日本では「ペペロンチーノ」というとパスタ料理を指すことが一般的ですが、イタリア語ではトウガラシそのものを指します。かつてコロンブスの交換によってイタリアにもたらされたこの「小さな悪魔」は、数世紀をかけてイタリアの風土に根付き、単なる辛味づけを超えた**「旨味と香りの野菜」**へと進化しました。

本ガイドでは、日本の一般的なトウガラシとは一線を画す、イタリア伝統品種の魅力と選び方、そして栽培の勘所を解説します。

# イタリア伝統野菜「ペペロンチーノ」栽培ガイド

## 1. 日本の品種と何が違うのか？

日本の家庭菜園や直売所で一般的な「鷹の爪（タカノツメ）」と、イタリアの在来種には決定的な違いがあります。それは**「果肉の厚み」と「香りの複雑さ」**です。

*   **日本のトウガラシ（鷹の爪など）:**
    *   **特徴:** 皮も果肉も非常に薄い。
    *   **味:** 突き抜けるような直線的な辛さ。香りは控えめ。
    *   **用途:** 主に乾燥させて「辛味」を足すスパイスとして使用。

*   **イタリアのペペロンチーノ:**
    *   **特徴:** 品種によりますが、**果肉に厚みがあるものが多い**です。
    *   **味:** 辛さの中に、フルーティーな甘み、スモーキーな香り、ハーブのようなニュアンスが含まれます。「辛いだけではない」のが最大の特徴です。
    *   **用途:** 乾燥だけでなく、生食、オリーブオイル漬け、詰め物、ペースト（ンドゥイヤ等）など、**「野菜」として料理の具材**になります。

自然栽培で作る場合、この「果肉の旨味」と「香り」がより顕著に表れるため、イタリア品種を選ぶ意義は非常に大きいです。

## 2. 品種選びのポイント：個性豊かなイタリアの在来種

提供されたデータを基に、代表的な4つのカテゴリー（品種群）を紹介します。ご自身の畑の土質や、料理の好みに合わせて選んでみてください。

### ① カラブラ（Calabria）系：辛さと香りの王道
**代表:** *Peperoncino di Calabria IGP*（認定品種群）

*   **特徴:** 「イタリアのトウガラシの首都」カラブリア州の品種群。形状は「サクランボ型（Ciliegia）」や「円錐形（Corno）」など多様です。
*   **味と食感:** 中程度〜強烈な辛さがありますが、同時に**非常に香りが強い**のが特徴。乾燥させても色が鮮やかな赤（カーマインレッド）を保ちます。
*   **おすすめの用途:** 辛いものが好きな方へ。生で刻んでパスタに、乾燥させて粉末に、またはサラミや肉料理の保存料兼スパイスとして。

### ② セニーゼ（Senise）系：甘くてサクサク、揚げて食べる
**代表:** *Peperone di Senise IGP*（バジリカータ州）

*   **特徴:** 別名「クルスキ（Cruschi）」。これは**辛くない（または微辛）**品種です。最大の特徴は**「果皮が極薄」**であること。
*   **味と食感:** 糖度が高く、乾燥させるとパリパリになります。
*   **おすすめの用途:** **「揚げトウガラシ」専用**と言っても過言ではありません。乾燥させたものをサッと油で揚げると、スナック菓子のようにサクサクになります。日本のトウガラシでは再現できない食感です。

### ③ 詰め物・保存食系：肉厚でジューシー
**代表:** *Peperoncino di 7 anni*（プーリア州）、*Ciliegia*（サクランボ型）

*   **特徴:** 「7年トウガラシ」の名は、原産地で株が何年も生き続けることに由来します（日本では冬越し対策が必要）。果実は丸みを帯びており、果肉が厚いです。
*   **味と食感:** じっくりとした辛さと、パプリカのようなジューシーさがあります。
*   **おすすめの用途:** 中の種をくり抜き、ツナやアンチョビ、ケッパーを詰めてオリーブオイルに漬け込む**「詰め物（Ripieni）」**に最適です。鷹の爪では不可能な調理法です。

### ④ フライ・炒め物系：緑のうちに食べる
**代表:** *Friariello Napoletano*（カンパニア州）、*Lazzaretto Abruzzese*（アブルッツォ州）

*   **特徴:**
    *   **フリアリエッロ:** ナポリの小型甘唐辛子（シシトウに似ていますが別物）。苦味が少なく甘みが強い。
    *   **ラッツァレット:** 細長く皺がある。中程度の辛さで、アブルッツォ料理（パスタなど）のベースに使われます。
*   **おすすめの用途:**
    *   フリアリエッロは完熟を待たず、緑色のうちに素揚げにして塩を振って食べます。
    *   ラッツァレットは、ニンニクとオイルで炒める「アーリオ・オーリオ」の真髄を味わうための品種です。

## 3. 自然栽培でのポイント

日本の気候（特に梅雨と湿気）は、乾燥した地中海性気候とは異なります。イタリア品種のポテンシャルを引き出すための栽培のコツです。

### ① 「水はけ」が命
イタリアのトウガラシ、特にセニーゼ種のような乾燥地帯に適応した品種は、過湿を嫌います。
*   **高畝にする:** 水はけを良くするため、畝は高く立ててください。
*   **土作り:** 肥沃すぎる土壌よりも、やや痩せ地〜中程度の肥沃度の方が、香りが凝縮されやすくなります。窒素過多は「葉ばかり茂って実がつかない（木ボケ）」や、病気の原因になります。

### ② 完熟を待つ勇気
日本のシシトウやピーマンは未熟果（緑）で収穫しますが、イタリアのトウガラシ（フリアリエッロを除く）は、**真っ赤に完熟するまで待つ**のが基本です。
*   完熟することで、カプサイシン（辛味）だけでなく、糖度と旨味成分がピークに達します。
*   自然栽培ではゆっくり育つため、焦らず赤くなるのを待ちましょう。秋の深まりとともに赤くなった実は、味が濃厚になります。

### ③ 乾燥の工夫
イタリア南部のように「軒先に吊るして天日乾燥」は、湿度の高い日本ではカビのリスクがあります。
*   **セニーゼ種など:** 収穫後、雨の当たらない風通しの良い場所でしっかり乾燥させるか、食品乾燥機（ドライフードメーカー）の助けを借りるのも一つの手です。カビさせずに乾燥できれば、長期保存可能な「赤い黄金」となります。

### ④ 交雑に注意（自家採種する場合）
トウガラシ（*Capsicum annuum*）は非常に交雑しやすい植物です。
*   甘い品種（セニーゼなど）と辛い品種（カラブリアなど）を隣同士で植えると、翌年の種から育ったものが「激辛のセニーゼ」になってしまうことがあります。
*   種を採る場合は、花が開く前に袋をかけるか、距離を十分に離して栽培してください。

## 栽培データ

イタリアのトウガラシは基本的に *Capsicum annuum* 種であり、栽培方法は日本のピーマンやトウガラシに準じますが、生育期間が長い傾向にあります。

| 項目 | データ・内容 |
| :--- | :--- |
| **収穫期** | **8月下旬 〜 11月（霜が降りるまで）**<br>※青果（フリアリエッロ等）は7月から収穫可能ですが、赤く完熟させる品種は晩夏以降が本番です。 |
| **播種期** | **2月 〜 3月（温床育苗が必要）**<br>※発芽に高温が必要なため、早春に加温して種を蒔きます。 |
| **発芽適温** | **25℃ 〜 30℃**<br>※20℃以下では発芽が揃いにくいです。 |
| **播種法** | **ポット播き / セルトレイ播き**<br>育苗期間が70〜80日と長いため、直蒔きではなく移植栽培が基本です。 |
| **覆土** | **薄く（5mm程度）**<br>乾燥しないように注意しますが、深く埋めすぎないようにします。 |
| **生育適温** | **25℃ 〜 30℃**<br>高温を好みます。15℃以下では生育が停滞します。 |
| **栽培法** | **日当たりと水はけの良い場所**<br>株間は45〜60cm確保します。支柱を立てて倒伏を防ぎます。<br>一番花より下の脇芽は基本的に除去し、風通しを良くします。 |
| **採種法** | **完熟果採種**<br>株の上で完全に赤く（または品種固有の色に）熟し、少しシワが寄るくらいまで置いた果実を収穫します。<br>果実を割り、種を取り出して日陰で十分に乾燥させます。 |
| **種子寿命** | **約 2 〜 3年**<br>条件が良ければ長持ちしますが、発芽率は徐々に落ちます。 |
| **休眠** | 特になし（乾燥後、適温があれば発芽します） |
| **種子保存法** | **低温低湿**<br>紙封筒に入れ、乾燥剤と共に密閉容器（瓶や缶）に入れて冷蔵庫の野菜室などで保管するのが理想です。 |
| **増やし方** | **種子繁殖**<br>トウガラシ類は基本的に種で増やします。<br>※「7年トウガラシ」も植物学的には種子で更新しますが、日本の冬（0℃以下）では枯れるため、越冬させる場合は鉢上げして室内（10℃以上）に取り込む必要があります。 |
