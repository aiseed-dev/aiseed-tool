イタリアの伝統野菜、特に「ブロッコリー」の深淵な世界へようこそ。

日本のスーパーマーケットで見かける丸くて大きなブロッコリーは、実はイタリア野菜のほんの一側面に過ぎません。イタリアの伝統的なブロッコリーは、花蕾（つぼみ）だけでなく、茎や葉、そして脇芽を味わう「野性味あふれる冬の味覚」です。

ここでは、自然に寄り添う栽培者の皆様に向けて、イタリア伝統ブロッコリーの魅力と、その多様な品種、そして栽培の勘所をご案内します。

# イタリア伝統ブロッコリー栽培ガイド
**～地中海の遺産、葉と茎を味わう冬の愉しみ～**

## 1. 日本のブロッコリーとイタリア在来種の違い

私たちが普段「ブロッコリー」と呼んでいるものは、イタリアの「ブロッコロ・カラブレーゼ（カラブリアのブロッコリー）」がアメリカで改良され、巨大な頂花蕾（てっぺんの大きな塊）を作るように固定された品種群がベースになっています。

しかし、イタリアの在来種（伝統品種）は、これとは異なる特徴を持っています。

| 特徴 | 一般的なF1品種（日本・北米流） | イタリア伝統品種（在来種） |
| :--- | :--- | :--- |
| **形状** | 大きな頂花蕾が一つできる「頂花蕾型」 | 脇芽が次々と出る「スプラウティング（側枝）型」や、葉を食べるタイプが多い |
| **可食部** | 主に花蕾（つぼみ）と太い茎 | **花蕾、茎、若葉、脇芽**すべて |
| **味・香り** | マイルドで甘みがある、均一な味 | **濃厚な旨味、独特の苦味や甘み**、品種ごとの個性が強い |
| **収穫期間** | 一発勝負（頂花蕾を獲って終わりが多い） | **冬から春まで長く収穫し続けられる** |
| **料理** | サラダ、茹で野菜 | クタクタに煮る、パスタソース、スープ、ソテー |

イタリア語の語源「ブロッコ（Brocco）」が「突き出た若枝」を意味するように、本来は**次々と伸びてくる若い茎（Cyma）を楽しむ野菜**なのです。

## 2. 知っておくべきイタリアの銘品種

ここでは、イタリア各地で愛され、スローフードの箱舟（プレシディオ）や伝統的特産品（PAT）にも登録されている代表的な品種を紹介します。これらは種が入手できれば、日本の自然菜園でも栽培可能です。

### ① クレアッツォのフィオラーロ（Broccolo Fiolaro di Creazzo）
*   **特徴:** **「花蕾がない（ように見える）」**のが最大の特徴です。頂点に大きな蕾を作るのではなく、葉と茎、そして小さな脇芽（現地語でFioi＝子供たち）を収穫します。
*   **味:** 霜に当たることで劇的に甘くなります。苦味と甘みのバランスが絶妙で、イタリアで最も美味しい葉野菜の一つと言われます。
*   **栽培の狙い:** ヴェネト州の冬の寒さに適応した品種です。日本の冬の寒さにも強く、冬の間の貴重な青菜として重宝します。

### ② トルボレのブロッコリー（Broccolo di Torbole）
*   **特徴:** ガルダ湖畔の風が育てる希少種。花蕾は小さく、**黄色みを帯びた緑色**をしています。
*   **味:** アブラナ科特有の硫黄臭さがほとんどなく、非常に繊細で甘みが強いです。現地では生食されることもあるほどです。
*   **栽培の狙い:** 湿気を嫌うため、風通しの良い場所での栽培が適しています。

### ③ ブロッコロ・ロマネスコ（Broccolo Romanesco）
*   **特徴:** 日本でも知名度が上がってきた、幾何学的（フラクタル）な形状のカリフラワーとブロッコリーの中間種。色は明るい黄緑色。
*   **味:** カリフラワーのようなコリコリした食感と、ナッツのようなコクのある甘みがあります。
*   **栽培の狙い:** ローマのクリスマスの定番です。大きく育てるには肥沃な土壌が必要です。

### ④ パテルノーポリのアプリラーティコ（Broccolo Aprilatico di Paternopoli）
*   **特徴:** **「4月（April）」**に収穫する晩生品種。冬を越し、春先に一気に伸びてくる側枝を収穫します。
*   **栽培のポイント:** 頂花蕾ができたら、花が咲く前に摘芯（ピンチ）します。そうすることで脇芽が一斉に伸び、柔らかい茎を大量に収穫できます。春の端境期に収穫できる貴重な品種です。

### ⑤ ビジニャーノのブロッコリー（Broccolo di Rapa di Bisignano）
*   **特徴:** 正確にはカブの仲間（チーマ・ディ・ラーパに近い）ですが、南イタリアではブロッコリーとして扱われます。
*   **味:** 野性味が強く、独特のほろ苦さと辛味があります。
*   **料理:** オレッキエッテ（耳たぶ型のパスタ）と合わせるのが鉄板です。ニンニクと唐辛子との相性が抜群です。

## 3. 自然栽培でのポイントと品種選び

日本の自然系菜園でこれらの品種を育てる際のポイントは、**「土壌の酸度」と「寒さの利用」**です。

### ポイント1：土壌とミネラル
ブロッコリーの原産地は地中海の石灰岩質土壌です。彼らはカルシウムやミネラルが豊富な土を好みます。
日本の土壌（特に火山灰土）は酸性になりがちです。自然栽培では、**カキ殻石灰や草木灰**などを活用し、ミネラル分を補給しつつ酸度を調整（pH 6.0～6.8程度）してあげると、伝統品種本来の味が引き出されます。

### ポイント2：寒さに当てる
多くのイタリア伝統品種は、寒さから身を守るために糖分を蓄積します。
*   **早生種（年内収穫）:** 早く食べたい場合はロマネスコなどの早生系を選びます。
*   **晩生種（越冬収穫）:** フィオラーロやアプリラーティコは、冬の寒さにじっくり当てて、春先に収穫することで真価を発揮します。虫の少ない時期に生育できるメリットもあります。

### ポイント3：脇芽（側枝）を楽しむ
F1品種のように「真ん中の大きな一つ」を狙うのではなく、**「次々と出てくる脇芽を長く楽しむ」**という意識で育ててください。
主茎の頂点を早めに収穫（摘芯）して、側枝の発生を促すことで、春まで長く収穫を続けることができます。これは家庭菜園や少量多品目栽培に非常に適したスタイルです。

## 4. おすすめの食べ方

イタリア流の食べ方を知ることで、栽培のモチベーションも上がります。

1.  **クタクタ煮（Stracotto）:** 日本では「茹で過ぎ」と言われるくらい、柔らかく茹でます。オリーブオイルとニンニクで風味をつけ、野菜の甘みを引き出します。
2.  **パスタソース:** 茹でたブロッコリーをフォークで潰し、アンチョビ、ニンニク、唐辛子と炒めてパスタに絡めます。
3.  **ソテー:** 下茹でしたものを、強火で焼き色がつくまでソテーします。肉料理の付け合わせに最適です。

## 5. 栽培データ

※品種により差がありますが、一般的なイタリア系晩生種・側枝型ブロッコリーの目安です。

### 基本データ
*   **科名:** アブラナ科アブラナ属
*   **学名:** *Brassica oleracea* var. *italica* 等
*   **原産地:** 地中海沿岸

### 栽培カレンダー（中間地・暖地）
*   **播種期:**
    *   **夏まき:** 7月中旬～8月下旬（年内～冬収穫）
    *   **秋まき:** 9月上旬～10月上旬（春収穫・晩生種向け）
*   **収穫期:** 11月～翌4月（品種の早晩性による）
    *   ※アプリラーティコなどは春（3月～5月）がメイン

### 栽培条件
*   **発芽適温:** 20～25℃
*   **生育適温:** 15～20℃（冷涼な気候を好む）
*   **土壌:** 水はけが良く、有機質に富んだ土壌。pH 6.0～6.8（酸性を嫌う）。

### 栽培方法
1.  **播種法:** セルトレイやポットに種をまき、育苗してから定植するのが確実です（直まきは虫害リスクが高い）。
2.  **覆土:** 種が隠れる程度（5mm～1cm）。
3.  **育苗:** 本葉5～6枚になるまで育てます。夏場の育苗は高温と乾燥、虫に注意し、寒冷紗などを活用します。
4.  **定植:** 株間は広め（45cm～60cm）にとります。特にフィオラーロなど葉が広がるタイプはスペースが必要です。
5.  **管理:** 成長期には水分と肥料分（ボカシ肥など）を必要とします。頂花蕾タイプ以外は、主茎が伸びてきたら早めに摘芯し、側枝の成長を促すと収量が増えます。

### 採種（種採り）について
*   **増やし方:** 種子で増やします。
*   **受粉:** 他家受粉性（虫媒花）。アブラナ科の他の品種（キャベツ、カリフラワー、他のブロッコリー）と容易に交雑します。
*   **採種法:**
    1.  形質が良い株を収穫せずに残し、春に花を咲かせます。
    2.  交雑を防ぐため、周囲に他のアブラナ科の花がない環境にするか、ネットで覆うなどの隔離が必要です。
    3.  花が終わり、サヤ（莢）が茶色く乾燥するまで待ちます。
    4.  サヤを収穫し、さらに乾燥させてから種を取り出します。
*   **種子寿命:** 常温で3～4年（保存状態による）。
*   **休眠:** ほとんどありません。
*   **種子保存法:** 紙袋や密閉容器に入れ、冷蔵庫などの冷暗所で乾燥保存します。
