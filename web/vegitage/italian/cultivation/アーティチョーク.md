地中海の「食用の花」、アーティチョーク（朝鮮アザミ）。
日本ではまだ「珍しい洋野菜」という位置づけですが、イタリアでは数千年の歴史の中で、地域ごとの気候や食文化に合わせて驚くほど多様な品種が育まれてきました。

一般的に日本で流通している品種と、イタリアの伝統品種はどう違うのか。そして、日本の風土で育てる際にどの品種を選び、どう向き合えばよいのか。イタリアの伝統野菜解説記事に基づき、栽培者の視点から解説します。

# イタリア伝統アーティチョーク品種ガイド

## 1. 日本の一般品種とイタリア伝統品種の違い

日本のホームセンターや種苗店で最も一般的に見かけるのは**「グリーングローブ（Green Globe）」**や**「インペリアルスター」**といった品種です。これらはアメリカや他国で改良されたものが多く、以下の特徴があります。

*   **日本の一般種（グリーングローブ系）:**
    *   **形:** ぽってりとした球形。
    *   **色:** 全体が緑色。
    *   **味・食感:** 肉厚でホクホクしているが、繊維が強め。茹でてマヨネーズやバターで食べるのが主流。
    *   **栽培:** 比較的寒さに弱く、日本では一年草扱いされることも多い。

一方、**イタリアの伝統品種**は「多様性」そのものです。

*   **イタリア伝統品種:**
    *   **形・色:** 紫色のもの、細長いもの、鋭いトゲがあるものなど様々。
    *   **味・食感:** 単にホクホクしているだけでなく、**「生で食べられるほどアクが少なく甘い」**ものや、**「丸ごと揚げてクリスピーに食べる」**ものなど、品種ごとに明確な用途があります。
    *   **風味:** 地中海のテロワールを反映し、ナッツのようなコクや、心地よい苦味のバランスが重視されます。

## 2. イタリアの主要品種とその特徴

イタリアには「アーティチョーク四大学派」とも呼べる主要な品種群と、ヴェネツィアの希少種があります。それぞれの個性を知ることで、栽培の楽しみが広がります。

### ① ラツィオの王様：ロマネスコ種
**Carciofo Romanesco del Lazio IGP**
*   **特徴:** 「マンモラ（Mammola）」とも呼ばれる、**巨大な球形**の品種です。最大の特徴は**「トゲがない」**こと。
*   **味:** 可食部が非常に多く、柔らかい。ローマ名物「ユダヤ風アーティチョーク（丸ごと素揚げ）」はこの品種でないと作れません。
*   **栽培視点:** 晩生（おくて）の傾向があります。トゲがないため管理や収穫が楽で、ボリューム感のある収穫物が得られます。

### ② サルデーニャの野生味：スピノーソ種
**Carciofo Spinoso di Sardegna DOP**
*   **特徴:** 円錐形で、**長く鋭い黄金色のトゲ**を持ちます。葉にもトゲがあり、扱いには革手袋が必要です。
*   **味:** トゲの見た目に反して、**アク（タンニン）が少なく甘みが強い**のが特徴。イタリアでは茎や芯を薄くスライスして**「生食（サラダ）」**にします。
*   **栽培視点:** 原種に近い野性味があり、乾燥に強い傾向があります。生で食べるという最高の贅沢を味わえるのは、栽培者の特権です。

### ③ ヴェネツィアの宝石：ヴィオレット種
**Carciofo Violetto di Sant'Erasmo (Presidio)**
*   **特徴:** 細長い円筒形で、美しい**濃紫色**をしています。ヴェネツィアの干潟（塩分を含む土壌）で育つ品種です。
*   **味:** 肉厚で非常に柔らかく、独特の風味があります。
*   **栽培視点:** 最初の蕾を早めに切り取る**「カストラウーラ（去勢）」**という技術が有名です。これにより脇芽の収穫を促します。切り取った最初の小さな蕾は、極上の柔らかさを持ちます。

### ④ 南イタリアの早生種：ブリンディジーノ種 & パエストゥム種
**Carciofo Brindisino IGP / Carciofo di Paestum IGP**
*   **特徴:**
    *   **ブリンディジーノ:** 円筒形で苞葉が柔らかい。早生で冬から収穫可能。
    *   **パエストゥム:** ロマネスコ系だが赤紫色の斑紋があり、少し早生。
*   **栽培視点:** 比較的早く収穫できるため、日本の梅雨前に収穫を終えたい暖地での栽培に向いている可能性があります。

## 3. 日本の自然栽培におけるポイント

アーティチョークは地中海沿岸（冬は雨が多く温暖、夏は乾燥）の植物です。日本（冬は乾燥して寒く、夏は高温多湿）とは気候が真逆であるため、以下の点に注意が必要です。

### ① 品種選びは「耐寒性」と「湿気」で考える
*   **暖地（関東以西の太平洋側）:**
    冬越しが可能な地域では、**「スピノーソ種」**や**「ヴィオレット種」**などの多年草としての性質が強いイタリア品種に挑戦する価値があります。株を充実させれば、春に次々と脇芽を収穫できます。
*   **寒冷地:**
    冬に枯れてしまうリスクが高いため、春まきでその年の秋〜初冬に収穫を目指すか、徹底した防寒が必要です。イタリア品種は晩生のものが多いので、収穫に至らないまま冬を迎えるリスクがあります。早生系の品種を選ぶか、苗作りを早める工夫が必要です。

### ② 「水はけ」が生命線
イタリア品種、特に原種に近いものは、日本の梅雨や秋の長雨による「根腐れ」に弱いです。
*   **高畝（たかうね）:** 必須です。水はけを極限まで良くしてください。
*   **土壌:** 粘土質よりも、砂質を含んだ排水性の良い土を好みます。自然栽培では、腐植を豊かにしつつも、水が停滞しない団粒構造を目指してください。

### ③ 「種」か「株分け」か
*   **種:** 入手しやすいですが、アーティチョークは遺伝的にバラつきが出やすい植物です。「ロマネスコ」の種を蒔いても、トゲのある個体が出たり、形が揃わなかったりすることがあります。自然栽培ではその「個体差」も楽しみの一つですが、均一性を求めすぎないことが大切です。
*   **株分け（吸枝）:** イタリアの農家は、脇芽（吸枝）を株分けして増やします。これにより親株と同じ形質（味・形）を受け継ぎます。もし良い株が育ったら、その脇芽を大切に育てて更新していくのが、美味しいイタリア品種を維持する秘訣です。

### ④ 収穫のタイミングと「カストラウーラ」
日本で売られているアーティチョークは大きくなりすぎて固いものが多いです。自家栽培なら、イタリア流に**「若採り」**が可能です。
特に頂点の最初の蕾（頂花蕾）を早めに切り取る（カストラウーラ）ことで、柔らかい脇芽（側花蕾）がたくさん出てきます。小さく柔らかい実は、下処理も楽で、オイル漬けや素揚げに最適です。

## 栽培データ

アーティチョーク（朝鮮アザミ）
*学名: Cynara cardunculus var. scolymus*

| 項目 | 内容 |
| :--- | :--- |
| **収穫期** | 4月下旬〜6月上旬（暖地での越冬株の場合）。寒冷地の春まき栽培では10月〜11月頃。 |
| **播種期** | **春まき:** 2月〜3月（温床育苗）、**秋まき:** 9月〜10月（暖地のみ。冬越し管理が必要）。 |
| **発芽適温** | 20〜25℃（高温すぎると発芽率が落ちるため注意）。 |
| **播種法** | ポットに2〜3粒まき、本葉が出たら間引いて1本にする。直根性で根を傷めると弱るため、定植時は根鉢を崩さないこと。 |
| **覆土** | 種の厚みの2〜3倍程度（嫌光性種子のため、しっかり覆土する）。 |
| **生育適温** | 15〜20℃。涼しい気候を好む。耐寒性は-2℃程度まで（品種による）。 |
| **栽培法** | **【定植】** 株間は広く（1m以上）取る。巨大になるため場所の確保が必要。<br>**【土壌】** 排水性を最優先する。酸性土壌を嫌うため、必要なら石灰等で調整。<br>**【水やり】** 生育期は水を欲しがるが、停滞水は厳禁。乾燥しすぎないよう敷草（マルチ）が有効。<br>**【越冬】** 霜に弱いため、株元に藁や腐葉土を厚く敷き、寒冷紗などで防寒する。 |
| **増やし方** | **① 種子:** バラつきが出やすい。<br>**② 株分け（吸枝）:** 親株の根元から出る脇芽（吸枝）を、根をつけた状態で切り取り植え付ける。品種の特性を維持できる最良の方法。適期は秋または春。 |
| **採種法** | 初夏に収穫せずに花を咲かせ、枯れるまで放置する。綿毛が開いたら花頭を切り取り、乾燥させて種を採る。ただし、交雑しやすいため純粋な品種維持は難しい。 |
| **種子寿命** | 常温で3〜5年程度（長命種子）。 |
| **休眠** | 夏の高温期に半休眠状態になることがある（地上部が枯れることもあるが、根が生きていれば秋に復活する）。 |
| **種子保存** | 乾燥剤と共に密閉容器に入れ、冷蔵庫の野菜室などで保管。 |
