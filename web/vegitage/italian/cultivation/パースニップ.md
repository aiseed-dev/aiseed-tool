# イタリアのパースニップ（Pastinaca）：古代の甘味と現代の栽培ガイド

パースニップ（*Pastinaca sativa*）は、ジャガイモが登場する以前、ヨーロッパの冬を支える重要なエネルギー源でした。特にイタリアでは、古代ローマ皇帝ティベリウスがわざわざゲルマニア（現在のドイツ）から取り寄せたという逸話が残るほど、「寒さにあたった甘み」が珍重されてきました。

日本ではまだマイナーな野菜ですが、霜が降りる日本の冬の気候は、イタリアの伝統的なパースニップ栽培に非常に適しています。ここでは、イタリアに残る貴重な品種と、自然栽培におけるポイントを解説します。

## 1. イタリア伝統品種と、日本で流通する一般的な品種の違い

日本で種が入手できる一般的なパースニップ（「ホロークラウン」などの固定種やF1種）と、イタリアの伝統品種（在来種）には、以下のような違いがあります。

| 特徴 | 一般的な流通品種（英米系・F1など） | イタリア伝統品種（カピティニャーノなど） |
| :--- | :--- | :--- |
| **形状** | スラリと長く、均一な円錐形。 | **太く短い、または不揃い。** 力強い根張り。 |
| **食感** | 繊維がやや残りやすいものもある。 | 加熱すると**ねっとりとクリーミー**になる傾向。 |
| **味・香り** | さっぱりとした甘み。セリ科特有の香りが強い。 | **濃厚な甘み**（栗やサツマイモに近い）。香りはナッツやアニスのような複雑なニュアンス。 |
| **用途** | シチュー、スープの具材。 | **フリット（揚げ物）、オムレツ、離乳食。** 主役になれる味。 |

**最大の違いは「甘みへの執着」です。**
イタリアの伝統的な栽培では、雪の下で越冬させるなどして極限までデンプンを糖化させるため、単なる野菜というより「冬の甘味」としての性格が色濃く残っています。

## 2. 品種データ解説：アブルッツォの至宝

記事内で紹介されている、イタリアでスローフード・プレシディオ（食の箱舟）に認定されている品種について解説します。

### パスティナーカ・ディ・カピティニャーノ (Pastinaca di Capitignano)
*   **産地:** アブルッツォ州ラクイラ県（標高900m以上の山岳地帯）
*   **特徴:**
    *   この品種の真骨頂は**「越冬」**にあります。現地の農家（Custodi：種子の守り手）は、秋に収穫せず、雪の下に埋めたまま冬を越させます。
    *   厳しい寒さで根のデンプンがショ糖に分解され、果物のような甘みを持ちます。
    *   形状は太くなりやすく、肉質は緻密です。
*   **食べ方:**
    *   現地ではクリスマスイブの「肉なし料理」の主役です。塩漬けタラ（バッカラ）と一緒に揚げ物にしたり、オムレツに入れたりして、その甘みを楽しみます。

> **※注意：プーリア州の「ペスタナカ（Pestanaca）」について**
> 記事にあるプーリア州ティッジアーノの「聖イッパツィオのペスタナカ」は、名前こそパースニップ（Pastinaca）に由来しますが、植物学的には**「黄色・紫色のニンジン」**です。種子を探す際は混同しないようご注意ください。

## 3. 日本の自然栽培におけるポイント

日本の自然栽培・有機栽培の生産者が、イタリア流の「美味しいパースニップ」を育てるための要点は以下の通りです。

### ① 「寒さ」を味方につける作型
ローマ皇帝がわざわざ北国から取り寄せたように、パースニップは**「寒さ」が調味料**です。
*   **収穫のタイミング:** 葉が枯れ始めてもすぐには収穫せず、**何度か強い霜に当てること**が重要です。可能であれば、1月〜2月まで畑に置いておくことで、驚くほど甘くなります。
*   **播種時期:** 逆算して、春まき（3〜5月）または初夏まきで、じっくり時間をかけて根を太らせます。

### ② 土壌の選び方（又根を防ぐ）
自然栽培では肥料で形を作ることはしませんが、土の物理性は重要です。
*   カピティニャーノのような**砂質土壌**が理想です。粘土質や石が多い土では又根（またね）になりやすいです。
*   初期生育が非常に遅いため、雑草に負けないよう、播種後の草管理がしやすい区画を選んでください。

### ③ 生態系サービスとしての利用
記事にもある通り、パースニップの花（セリ科特有の散形花序）は、ヒラタアブや寄生バチなどの**天敵昆虫（益虫）を強力に引き寄せます**。
*   すべて収穫せず、数株を畑に残して花を咲かせることで、翌年の果菜類や葉物野菜の害虫防除（バンカープランツ）として機能します。これはアグロエコロジーの観点からも推奨されます。

### ④ 【重要】取り扱いの注意：光毒性
栽培者に必ず知っておいていただきたいのが**「光線皮膚炎」**のリスクです。
*   パースニップの茎や葉には「フラノクマリン類」が含まれています。
*   葉の汁が皮膚につき、そこに日光（紫外線）が当たると、火傷のような水ぶくれや色素沈着を起こします。
*   **対策:** 収穫や除草作業の際は、**必ず長袖・手袋を着用**してください。晴天時の作業は特に注意が必要です。

## 栽培データ

パースニップ（セリ科）は種子の寿命が短く、発芽が最大の難関です。

| 項目 | データ・内容 |
| :--- | :--- |
| **収穫期** | 11月下旬 〜 3月（霜に当ててから収穫するのがベスト） |
| **播種期** | **寒冷地:** 4月〜5月 <br> **暖地:** 3月〜4月、または9月（秋まきは翌春収穫）<br> ※生育期間が約4〜5ヶ月と長いため、春まき冬どりが一般的です。 |
| **発芽適温** | 15℃ 〜 20℃（地温が高すぎると発芽率が落ちます） |
| **播種法** | **直播き（移植不可）**。直根性のため移植すると又根になります。<br> 筋まきし、最終株間は10〜15cm程度に間引きます。 |
| **覆土** | 薄くかける程度（好光性種子に近い性質ですが、乾燥を防ぐためごく薄く土をかけ、鎮圧します）。 |
| **生育適温** | 15℃ 〜 20℃（冷涼な気候を好みます） |
| **栽培法** | **発芽まで乾燥させないこと**が最重要です。発芽まで2〜3週間かかることもあります。<br> 初期生育が遅いため除草を徹底します。土寄せを行うと青首（首部分の緑化）を防げます。 |
| **採種法** | 二年草です。冬を越し、翌年の初夏にトウ立ちして花が咲きます。<br> 種が茶色く完熟したら、花茎ごと刈り取って追熟・乾燥させます。<br> ※他のセリ科（野生のニンジンなど）とは交雑しませんが、野生化したパースニップがある場合は注意が必要です。 |
| **種子寿命** | **非常に短い（約1年）**。<br> 毎年新しい種を採るか、購入する必要があります。古い種は極端に発芽率が落ちます。 |
| **休眠** | 特になし。 |
| **種子保存法** | 乾燥剤を入れた密閉容器に入れ、冷蔵庫で保管してください。常温保存では翌年の発芽は期待できません。 |
