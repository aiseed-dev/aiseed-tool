# イタリアの伝統野菜「ボリジ（ルリジサ）」栽培ガイド

地中海の鮮やかな青色を映したような「星の花」ボリジ。
イタリア・リグーリア州では「プレボッジョン（野生ハーブミックス）」の主役として、プーリア州では「貧者の豚（捨てるところがない食材）」として愛される、ムラサキ科の一年草です。

その清涼感あるキュウリのような香りは、古来より「勇気」と「喜び」をもたらし、憂鬱を吹き飛ばすと信じられてきました。日本の自然菜園でも、ミツバチを呼ぶコンパニオンプランツとしてだけでなく、イタリア流の「野菜」として育ててみませんか。

## イタリア伝統品種と日本流通品種の違い

基本的に、イタリアで伝統野菜として扱われているボリジと、日本の園芸店でハーブとして流通しているボリジは、植物学的には同じ ** *Borago officinalis* L. ** です。しかし、その「扱い」と「収穫のタイミング」に大きな違いがあります。

### 1. 味と香りの捉え方
*   **日本（ハーブ・観賞用）:** 主に**花**をエディブルフラワーとして利用したり、葉をハーブティーにするのが一般的です。観賞用として大きく育てることが多いため、茎葉が硬くなりがちです。
*   **イタリア（野菜用）:** 葉を**野菜**として食べます。キュウリのような風味（オイスターのような滋味とも言われる）と、加熱した時のとろみのある食感が重視されます。他の野草の苦味を和らげる「つなぎ」の役割も果たします。

### 2. 食感と形態
ボリジの最大の特徴は、全草を覆う「剛毛（borra）」です。
*   **日本:** 成長しきった株は毛が硬く、口に刺さるため生食には向きません。
*   **イタリア:** 伝統料理（パンソティやフリット）では、**若く柔らかい葉**を摘むか、あるいは**加熱調理（茹でる・揚げる）**することで剛毛を無力化して食べます。加熱すると毛のチクチク感は消え、柔らかな食感に変わります。

### 3. 用途の違い
*   **日本:** 砂糖菓子、サラダの彩り、ティー。
*   **イタリア:**
    *   **詰め物パスタ（パンソティ）:** 茹でて刻み、リコッタチーズ等と混ぜてパスタに詰める。
    *   **フリット（天ぷら）:** 衣をつけて揚げる（プーリア州の伝統）。
    *   **スープ:** 刻んで煮込む。

## 日本での自然栽培のポイント

日本の気候で、イタリアのような「美味しい野菜」としてボリジを育てるためのポイントです。

### 1. 品種選びと種子の入手
日本国内で入手できるボリジの種で問題ありません。特別な「イタリア野菜用」という品種を探す必要はなく、一般的な *Borago officinalis* の種を入手してください。
ただし、もし「白花種（アルバ）」などの変種が手に入る場合は、青花種（標準）の方が薬効や伝統的な利用においては一般的です。

### 2. 栽培環境（湿気対策）
ボリジは地中海原産のため、**乾燥には強く、過湿に弱い**性質があります。
*   **土壌:** 肥沃すぎる土だと徒長して倒れやすくなります。自然栽培なら、痩せ地や荒れ地でも十分に育ちます。水はけの良さを最優先してください。
*   **場所:** 日当たりを好みます。日本の梅雨〜夏の高温多湿で溶けるように枯れることがあるため、風通しの良い場所を選びましょう。

### 3. 味を良くする収穫のコツ
*   **若葉を狙う:** 葉を食べるなら、花茎が立ち上がる前のロゼット状の時期の「若い葉」か、茎が伸びても先端付近の柔らかい葉を収穫します。これにより、剛毛の不快感を減らし、ピロリジジンアルカロイド（後述）の摂取量も抑えることができます。
*   **花:** 次々と咲くので、咲いたそばから収穫してサラダや飾りに使います。ガクの部分からポロリと外れます。

### 4. こぼれ種での循環
ボリジは繁殖力が強く、こぼれ種でよく増えます。自然菜園では、一度植えれば翌年からは勝手に生えてくることが多いです。これを活かし、果樹の下草やコンパニオンプランツ（イチゴやトマトと相性が良いとされる）として共生させるのがおすすめです。

### ※安全性について（重要）
記事にある通り、ボリジの葉や花には微量のピロリジジンアルカロイド（PA）が含まれます。
*   **日常的な大量摂取は避ける:** サプリメントのような濃縮形での摂取は推奨されません。
*   **伝統的な食べ方:** イタリアでも「時折楽しむ季節の料理」としての位置づけです。あく抜き（茹でこぼし）や、若い葉の利用など、伝統的な知恵に従って適量を楽しみましょう。

## 栽培データ

| 項目 | 内容 |
| :--- | :--- |
| **収穫期** | **春まき:** 5月〜7月<br>**秋まき:** 4月〜6月<br>葉は本葉が展開したら随時（若いうちが美味）。花は開花直後に収穫。 |
| **播種期** | **春まき:** 3月〜5月<br>**秋まき:** 9月〜10月（寒冷地を除く）<br>※秋まきの方が株が大きく育ちやすいですが、冬の寒さが厳しい地域は春まき推奨。 |
| **発芽適温** | 20℃前後 |
| **播種法** | **直播き（じかまき）推奨**。<br>直根性で移植を嫌うため、畑やプランターに直接種をまくのがベストです。ポット育苗する場合は、根が回る前に若苗のうちに定植し、根を崩さないようにします。 |
| **覆土** | **必要（嫌光性種子）**。<br>種が隠れる程度（1cm〜2cmほど）しっかりと土を被せます。発芽までは土を乾かさないように管理します。 |
| **生育適温** | 15℃〜25℃<br>冷涼な気候を好み、日本の真夏の蒸し暑さは苦手です。 |
| **栽培法** | 株間は30〜50cm確保します（かなり横に広がります）。<br>肥料は控えめで育ちます（多肥だとアブラムシがつきやすくなります）。<br>草丈が高くなり倒れやすいので、必要に応じて支柱を立てるか、土寄せをします。 |
| **採種法** | 花が終わると4つの分果ができます。緑色から黒褐色に変色し、完熟すると自然に地面に落下します。<br>こぼれる前に、黒くなったものを見つけたら手で受けて軽く触り、ポロリと落ちるものを収穫します。または、完熟直前の枝ごと刈り取って追熟・乾燥させます。 |
| **種子寿命** | 2〜3年程度（比較的短命）。 |
| **休眠** | 特に深い休眠はありません。採種後、環境が整えば発芽します。 |
| **種子保存法** | 十分に乾燥させ、紙袋や密閉容器に入れて、冷蔵庫（野菜室）などの冷暗所で保管します。 |
