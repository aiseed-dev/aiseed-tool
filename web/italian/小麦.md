# イタリアの小麦

*Triticum* spp. — 文明を育んだ黄金の穂

---

## 序論：地中海文明と小麦の不可分な関係

小麦（*Triticum*属）は、イタリア半島の歴史、文化、そして社会経済構造を形成してきた最も重要な作物です。紀元前7000年頃、「肥沃な三日月地帯」において人類が初めて野生の穀物を栽培化した瞬間から、小麦は文明の礎となりました。イタリアにおいて、この穀物は単なるカロリー源としての役割を超え、ローマ帝国の拡大を支える戦略物資であり、近代国家形成期における政治的自立の象徴、そして現代においては生物多様性と食のアイデンティティを巡る議論の中心にあります。

ここでは、イタリアにおける小麦の歴史的変遷を、植物学的、農学的、政治的、そして栄養学的視点から包括的に分析します。特に、古代の自生種（ランドレース）から、20世紀初頭のナザレーノ・ストランペッリによる遺伝学的革命、ファシズム期の「小麦の戦い」、そして現代における「古代小麦（Grani Antichi）」の再評価に至るまでのプロセスを詳述します。また、シチリアやトスカーナといった主要産地の地域固有品種の特性や、DOP（保護指定原産地表示）等の品質認証制度が果たす役割についても、最新の知見に基づき検証を行います。

## 小麦の植物学的分類とイタリアの遺伝的多様性

イタリアで栽培されてきた小麦の多様性を理解するためには、まずその遺伝的構造と植物学的分類を整理する必要があります。小麦属の進化は、異なるゲノムを持つ祖先種の交雑と倍加の歴史であり、イタリアの農耕文化はこれら異なる倍数性の種を用途に応じて使い分けてきました。

### ゲノム構成による分類体系

イタリアの伝統的農業において利用されてきた小麦は、主に以下の三つの倍数性グループに分類されます。

イタリアにおける主要な小麦種の植物学的分類と用途：

| 倍数性 | ゲノム構成 | 植物学的名称 | イタリア語通称 | 歴史的・現代的用途 |
| :---- | :---- | :---- | :---- | :---- |
| 二倍体 | AA (2n=14) | *Triticum monococcum* | Piccolo farro / Monococco | 人類最古の栽培種。現在は極めて限定的な地域（一部の山岳地帯）でのみ栽培されるスペシャリティ穀物。 |
| 四倍体 | BBAA (2n=28) | *Triticum dicoccum* | Farro medio / Emmer | 古代ローマの主食。トスカーナ州ガルファニャーナ地方などでIGP産品として保護されています。 |
|  |  | *Triticum turgidum subsp. durum* | Grano duro | イタリア南部の主要作物。パスタやアルタムーラ・パンの原料となる硬質小麦。 |
|  |  | *Triticum turgidum subsp. turgidum* | Grano del miracolo / Rivet wheat | 枝分かれする穂を持つ珍しい品種。歴史的な遺伝資源として保存。 |
| 六倍体 | BBAADD (2n=42) | *Triticum aestivum subsp. aestivum* | Grano tenero | パン用軟質小麦。イタリア中部・北部で優勢。製パン適性が高い。 |
|  |  | *Triticum spelta* | Farro grande / Spelta | 北方系（ゲルマン系）のスペルト小麦。イタリアでは比較的新しい導入か、北部国境地帯に限られます。 |

### 「古代小麦」とランドレースの定義

現代の文脈において「古代小麦」や「伝統品種」と呼ばれるものは、厳密には20世紀半ばの集約的育種（緑の革命）以前に成立した品種群を指します。これらは、科学的な交配プログラムによって作出されたものと、数世紀にわたり農民が自家採種を繰り返す中で地域の環境に適応した「ランドレース（在来種）」の双方を含みます。

これらの品種群に共通する農学的特徴は以下の通りです：

1. 草丈の高さ: 多くの伝統品種は150cmから180cmに達します。これは光合成能力を高め、雑草との競合において物理的に優位に立つための適応でしたが、同時に強風や豪雨による「倒伏（lodging）」のリスクを抱えていました。
2. 広範な遺伝的多様性: 個体群の中に遺伝的なばらつきがあり、これが病害虫の蔓延や気候変動に対する緩衝材（レジリエンス）として機能します。
3. 低投入環境への適応: 化学肥料が存在しなかった時代に選抜されたため、貧栄養土壌でも根を深く張り、養分を吸収する能力が高いです。現代の有機農業において再評価されている最大の理由がここにあります。

## イタリア小麦の歴史的展開：古代から19世紀末まで

### ローマの穀倉と中世の農耕

イタリアにおける小麦栽培の歴史は、ローマ帝国の拡大と軌を一にします。シチリア島は「ローマの穀倉」としての役割を担い、帝国の巨大な人口を支えるための戦略的供給基地でした。古代ローマ時代には、殻付きの「ファッロ（二粒小麦）」が主食でしたが、脱穀の容易な裸麦（デュラム小麦や普通小麦）への転換が徐々に進行しました。

中世からルネサンス、そして近代に至るまで、イタリアの農業社会は、北部・中部の小作制度（メッザドリア）や南部のラティフォンド（大土地所有制）といった社会構造と結びついていました。この時代、農民たちは地域ごとの微気象（マイクロクライメート）に適応した独自の品種（ランドレース）を選抜・維持しており、これが現代に残るイタリアの豊かな遺伝資源の基盤となっています。

### 19世紀末の危機と「リエーティ」種の限界

19世紀末、イタリアの小麦栽培は人口増加に伴う需要拡大に対し、生産性の限界に直面していました。当時、イタリア中部で最も信頼され、広く栽培されていた品種の一つに「リエーティ（Rieti）」があります。この品種は赤さび病（rust）に対して優れた抵抗性を持っていましたが、致命的な欠点がありました。それは、草丈が高すぎるために倒伏しやすく、また成熟が晩生（おくて）であるため、収穫期にイタリア特有の夏の干ばつや熱波（stretia）の被害を受けやすいことでした。

当時の平均的な小麦収量はヘクタールあたり1.0トン程度に過ぎず、イタリアは慢性的な食糧輸入国でした。この状況を打破するためには、従来の「優良個体の選抜」という手法を超えた、抜本的な遺伝的改良が必要とされていました。

## ナザレーノ・ストランペッリと遺伝学的革命

20世紀初頭、イタリアの農学者ナザレーノ・ストランペッリ（Nazareno Strampelli, 1866-1942）によって、世界の農業史における最初の「緑の革命」とも呼ぶべき飛躍的な品種改良が行われました。彼の業績は、ノーベル賞受賞者ノーマン・ボーローグによる後の多収量品種開発に数十年先駆けるものでした。

### 交雑育種への挑戦と「アカコムギ」の導入

1900年、ストランペッリは「リエーティ」種の耐病性を維持しつつ、倒伏耐性と早生性を付与するための交配実験を開始しました。当時、異なる品種を人為的に掛け合わせる「交雑（Hybridization）」は、メンデルの法則が再発見されたばかりの時代にあって、極めて前衛的かつ懐疑的に見られる手法でした。

ストランペッリの最大の功績は、日本の品種「アカコムギ（Akakomugi）」を育種親として導入したことにあります。アカコムギは草丈が低く、極めて早生でしたが、栽培価値自体はないと見なされていました。しかしストランペッリは、この品種が持つ矮性遺伝子（*Rht8*）と日長反応性遺伝子（*Ppd-D1*）に着目しました。

### 「セメンティ・エレッテ（精選種）」の系譜

ストランペッリは複雑な交配プログラム（リエーティ × ウィルヘルミナ・タルウェ × アカコムギ）を実行し、一連の画期的な品種群「セメンティ・エレッテ（Sementi Elette）」を作出しました。これらは現代のイタリア小麦の基礎となっています。

ストランペッリによって作出された主要品種とその系譜：

| 品種名 | 分類 | 交配親 / 系譜 | 特性・革新点 |
| :---- | :---- | :---- | :---- |
| Ardito | 普通小麦 | Rieti × Wilhelmina × Akakomugi | 「アルディート（大胆な）」の名を持ちます。極早生で背が低く、倒伏に強いです。夏の干ばつが来る前に収穫可能となり、山間部でも栽培が可能になりました。 |
| Mentana | 普通小麦 | Rieti × Wilhelmina × Akakomugi | Arditoの兄弟系統。環境適応能力が極めて高く、アルゼンチンや中国など世界的に普及しました。 |
| San Pastore | 普通小麦 | Balilla × Villa Glori | 1930年代から50年代にかけてイタリアで最も広く栽培された品種の一つ。安定した高収量を誇りました。 |
| Senatore Cappelli | デュラム小麦 | Jean Retifah（チュニジア在来種）からの純系選抜 | 南イタリアの乾燥地帯に適応。その品質の高さから「選ばれし種（Razza Eletta）」と称されました。 |
| Carlotta | 普通小麦 | Rieti × Massy | ストランペッリの妻の名を冠した初期の品種。多収量ですが倒伏耐性はArditoほどではありませんでした。 |

## ファシズム下の「小麦の戦い」と農業近代化

1925年、ベニート・ムッソリーニ政権は、増大する小麦輸入による貿易赤字を解消し、国家の自給自足（アウタルキー）を達成するために「小麦の戦い（Battaglia del Grano）」を宣言しました。この国家的キャンペーンにおいて、ストランペッリの科学的成果は政治的な武器として動員されました。

### 科学とプロパガンダの融合

ムッソリーニは、土地所有構造の改革（リフォルマ・アグラリア）という社会的に困難な課題を回避しつつ食糧増産を達成するため、単位面積当たりの収量増加に注力しました。ストランペッリは「全国穀物遺伝学研究所」の所長に任命され、彼が開発した新品種（特にArditoやMentana）の普及が国策として強力に推進されました。

また、ストランペッリは農民が自身の保有する古い種子と、研究所が開発した改良種子を等量交換できる「種子交換法」の制定を働きかけ、新品種の普及速度を劇的に高めました。この結果、イタリアの小麦収量は19世紀初頭の1.0トン/haから、1930年代には1.5トン/haへと約50%向上し、パン用小麦のほぼ完全自給を達成するに至りました。

### 「セナトーレ・カッペッリ」の覇権

この時期、デュラム小麦の分野で決定的な地位を確立したのが「セナトーレ・カッペッリ（Senatore Cappelli）」です。ストランペッリがプーリア州フォッジャの試験場で選抜したこの品種は、1.8メートル近い草丈を持ちながらも、南イタリアの過酷な乾燥気候に適応し、優れた製麺特性（パスタ適性）を持っていました。1930年代から1960年代にかけて、イタリアのデュラム小麦作付面積の60%以上を占める独占的な品種となり、現代のイタリアンパスタの黄金時代の基礎を築きました。

## 地域固有の伝統品種（ランドレース）の特性

イタリアには、ストランペッリ以前から存在する、あるいは特定の地域環境で独自に進化・維持されてきた数多くの在来品種が存在します。これらは現代の均質化された市場において、「生物多様性」と「テロワール（土地の味）」を体現する存在として再評価されています。

### シチリア島の遺伝資源

シチリア島は、その多様な微気象条件により、特異な小麦品種の宝庫となっています。

主要なシチリア産古代小麦の特性：

| 品種名 | 分類 | 特性・用途 |
| :---- | :---- | :---- |
| Timilia (Tumminia) | デュラム小麦 | 「Marzuolo（3月蒔き）」とも呼ばれる春播き品種。短期間（約3ヶ月）で成熟するため、干ばつ回避能力が高いです。製粉すると独特の灰色の粉となり、カステルヴェトラーノの黒パン（Pane Nero）に用いられます。 |
| Russello | デュラム小麦 | 「赤い」を意味する名を持ち、穂が赤みを帯びます。シチリア東部（イブレイ山地など）で栽培され、グルテンは弱いですが香りが強く、硬質パンの製造に適しています。 |
| Perciasacchi | デュラム小麦 | 名前の由来は「袋を突き破る（Percia-sacchi）」。粒が細長く鋭いため、麻袋を貫通することから名付けられました。現代のカムット（Khorasan）小麦に類似した特性を持ち、パスタにした際の食感が優れています。 |
| Maiorca | 普通小麦 | シチリアでは珍しい軟質小麦。数世紀にわたり栽培され、白くきめ細かい粉が得られるため、カンノーロの皮やビスケットなどの製菓用に重宝されます。 |
| Realforte | デュラム小麦 | 煮崩れしにくい良質なセモリナ粉が得られる歴史的品種。 |

### トスカーナおよび中部イタリアの伝統品種

トスカーナ州を中心とする中部イタリアでは、製パンに適した軟質小麦の伝統が色濃く残っています。

* Verna（ヴェルナ）: 1953年にトスカーナ州立農業研究所のミカエレス教授によって開発された比較的新しい「伝統品種」です。標高の高い地域や寒冷地に適応し、グルテン含有量は低い（W値が低い）ですが、発酵の安定性と独特の風味を持ちます。
* Gentil Rosso（ジェンティル・ロッソ）: 19世紀中頃から20世紀初頭にかけてトスカーナで最も普及していた品種です。成熟すると穂が赤紫色になります。タンパク質とグルテンの質は現代品種に劣りますが、石臼挽きにすることで香りと栄養価が際立ちます。
* Frassineto（フラッシネート）: アレッツォ県のフラッシネート地域に由来します。多収量かつ環境適応性が高く、トスカーナの無塩パン（Pane Sciocco）の原料として歴史的に重要です。

### 二粒小麦（ファッロ）の聖地：ガルファニャーナ

イタリアは、古代穀物であるファッロ（エンマー小麦）の栽培が途絶えることなく継続されてきた稀有な国です。特にトスカーナ州ルッカ県のガルファニャーナ地方では、Farro della Garfagnana IGPとして、地域固有の二粒小麦（*Triticum dicoccum*）が保護されています。この地域のファッロは、標高300mから1000mの山間部で化学肥料を使わずに栽培され、その強靭な根系により土壌流出を防ぐ環境保全的な役割も果たしています。

## 品質認証制度（DOP/IGP）による保護と生産規定

イタリアはEUの地理的表示保護制度を最も積極的に活用している国の一つであり、特定の伝統的小麦品種の使用を義務付けた製品が数多く存在します。これらの「生産規定書（Disciplinare）」は、品種の保存における法的な防波堤となっています。

### アルタムーラ・パン（Pane di Altamura DOP）

プーリア州アルタムーラで作られるこのパンは、パンとして欧州初のDOP認定を受けました。その生産規定は極めて厳格であり、原料となるデュラム小麦の品種を特定しています。

* 使用品種: ムルジャ北西部（Alta Murgia）で栽培された「Appulo」「Arcangelo」「Duilio」「Simeto」の4品種を、単独または混合で少なくとも80%以上使用しなければなりません。
* 製法: 天然酵母（Pasta Madre）、海塩、水のみを使用し、厚いクラスト（外皮）を形成するための薪窯焼成（またはそれに準ずる方法）が義務付けられています。
* 気候変動への対応: 2024年の深刻な干ばつの際、指定品種の収穫量が激減したため、一時的に品種規定を緩和する特例措置が申請されました。これは、伝統的生産規定と気候変動の現実との衝突を示す象徴的な事例です。

### グラニャーノのパスタ（Pasta di Gragnano IGP）

ナポリ近郊のグラニャーノは「パスタの首都」として知られています。IGP認定を受けるためには、以下の基準を満たす必要があります。

* 原料: タンパク質含有量が乾燥重量比で13%以上の高品質なデュラム小麦セモリナ粉を使用すること。
* 成形: ブロンズダイス（青銅製の抜き型）を使用すること。これにより表面に微細な凹凸が生まれ、ソースの絡みが良くなります。
* 乾燥: 40℃から80℃の範囲で、6時間から60時間をかけて徐々に乾燥させること。現代の高温短時間乾燥（100℃以上で数時間）とは異なり、小麦の風味と栄養価を保持します。

## 栄養学的評価と健康への影響：科学的視点

近年、「古代小麦は現代小麦よりも健康的である」という言説が広まっていますが、これに対する科学的評価は慎重に行う必要があります。

### グルテンの質とセリアック病

「古代小麦はグルテンフリーである」あるいは「セリアック病患者でも食べられる」という誤解が存在しますが、科学的にはすべての小麦品種（古代・現代問わず）はセリアック病患者にとって有害なグルテンを含んでいます。しかし、グルテンの「質」には違いがあります。

* グルテン指数: 伝統品種（特にデュラム小麦の古品種）は、現代品種に比べてグルテン指数（グルテンの強さ・弾力）が低い傾向にあります。これは、製パンや製麺においては生地が切れやすいという欠点になりますが、消化管内での分解・消化が容易である可能性を示唆しています。
* 非セリアック・グルテン感作（NGCS）: セリアック病ではないが小麦摂取により不調をきたす人々（NGCS）を対象とした研究において、セナトーレ・カッペッリ種などの古代小麦を使用したパスタを摂取した場合、現代品種に比べて消化器症状が軽減されたという報告があります。

### 機能性成分と抗酸化作用

伝統品種の多くは、現代品種と比較して二次代謝産物（ポリフェノール、フラボノイド、カロテノイド）を豊富に含むことが確認されています。

* 微量栄養素: スペルト小麦や一粒小麦は、現代の普通小麦よりも亜鉛、鉄、マグネシウム、セレンなどのミネラル含有量が高い傾向にあります。
* 加工の影響: 伝統品種は石臼で全粒粉（Integrale）または半全粒粉（Tipo 1, 2）として製粉されることが多く、胚芽やふすまに含まれる食物繊維や抗酸化物質が製品に残存しやすいです。これが健康上の利点として認識されている側面も大きいです。

## 現代農業における課題と展望：サステナビリティと市場

### 低投入型農業への適性

現代の集約農業は、高い収量を得るために多量の窒素肥料と農薬を必要とします。これに対し、古代小麦・伝統品種は「低投入（Low-input）」システムにおいてその真価を発揮します。

* 雑草抑制: 高い草丈は、除草剤を使用せずとも物理的に雑草を被圧（遮光）する効果があります。
* 環境適応: ストランペッリの時代から続く「乾燥地適応」の遺伝子は、気候変動による水不足が深刻化するイタリア農業において、灌漑用水を節約できる持続可能な選択肢となっています。

### 経済的側面とスローフードの役割

伝統品種の栽培は、収量が現代品種の30〜50%程度（1.5〜2.5トン/ha）にとどまるため、経済的な持続可能性を確保するには高付加価値化が不可欠です。

* 価格: 伝統品種の小麦粉は、一般市場で3.00ユーロから5.10ユーロ/kgで取引されており、これは標準的な小麦粉の2倍以上の価格です。
* スローフード: スローフード協会は「プレシディオ（Presidi）」プロジェクトを通じて、アブルッツォのソリーナ小麦やシチリアの伝統的穀物などを認定し、生産者と消費者を直接結ぶ支援を行っています。これにより、絶滅の危機に瀕していた品種が「地域の誇り」として復活し、小規模農家の生計を支えるモデルが確立されました。

---

## 結論

イタリアにおける小麦の歴史は、自然環境への適応と人為的な改良の絶え間ない相互作用の記録です。ローマ時代の穀倉から、ストランペッリによる科学的品種改良、ファシズム期の増産体制、そして戦後の工業化を経て、現在イタリアは再び「多様性」の価値に回帰しています。

「古代小麦」ブームは、単なる懐古趣味ではなく、気候変動への適応、化学肥料への依存脱却、そして食と健康の関係再考という、現代社会が直面する課題に対する一つの回答です。イタリアの伝統品種は、過去の遺産であると同時に、持続可能な未来の農業システムを構築するための不可欠な遺伝資源（リソース）として、今後も重要な役割を果たし続けるでしょう。
