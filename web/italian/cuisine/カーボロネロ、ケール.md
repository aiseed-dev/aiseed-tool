自然栽培でカーボロネロやケールを育てている皆様、こんにちは。

日本では、ケールといえば「青汁」や「スムージー」、あるいは「生のままサラダ」で食べるイメージが強いかもしれません。しかし、本場イタリアでは、これらの野菜は**「クタクタになるまで煮込む」**ことで真価を発揮する食材です。

特に自然栽培で育った野菜は、繊維がしっかりとしており、味が濃厚です。その力強い味を最大限に引き出す、イタリアの農民たちが何世紀にもわたって受け継いできた「伝統的な食べ方」をご紹介します。

# 畑から食卓へ：カーボロネロとケールのイタリア伝統料理ガイド

## 1. イタリア流と日本流の決定的な違い

日本のスーパーフードとしての食べ方と、イタリアの伝統的な食べ方には大きな違いがあります。

*   **日本（現代風）：** 「酵素を壊さないように生で」「苦味をフルーツで消す」
*   **イタリア（伝統流）：** **「加熱して甘みを引き出す」「オイルと合わせて吸収率を高める」**

イタリア、特にトスカーナやプーリアの農村では、これらの野菜は「硬い繊維を柔らかくし、消化を良くする」ために加熱調理が基本です。特に霜が降りた後の収穫物は、加熱することで驚くほど甘く、旨味の強い出汁（だし）が出ます。

## 2. トスカーナの王様「カーボロネロ（黒キャベツ）」の楽しみ方

トスカーナ地方では、カーボロネロは「冬のスープの主役」です。

### 代表料理：リボッリータ (Ribollita)
**地域：トスカーナ州（フィレンツェ周辺）**

「再び（Ri）煮込んだ（Bollita）」という意味の、農民の知恵が詰まった料理です。本来は古くなったパンを消費するための料理ですが、主役はカーボロネロです。

**【自然栽培野菜を活かすレシピ】**

1.  **材料**: カーボロネロ（たっぷり）、白インゲン豆（乾燥を戻したもの、または水煮）、玉ねぎ、人参、セロリ、トマト（少量）、硬くなったパン、高品質なオリーブオイル。
2.  **ソフリット**: 玉ねぎ、人参、セロリのみじん切りをオリーブオイルでじっくり炒め、甘みを出します。
3.  **煮込み**: ざく切りにしたカーボロネロと白インゲン豆、トマト、水を加えて煮込みます。カーボロネロがクタクタになるまで、弱火で1時間以上煮るのがコツです。
4.  **パンの投入**: スープの水分が減ってきたら、薄切りのパンを鍋に敷き詰め、スープを吸わせます。
5.  **「リ」ボッリータ**: 一晩置き、翌日に再び火を入れて温め直します。全体が粥状にトロトロになったら完成。
6.  **仕上げ**: 皿に盛り、**生のオリーブオイル（Olio crudo）**をたっぷりと回しかけます。

> **栽培者へのアドバイス**:
> トスカーナには「カーボロネロは霜に当たってから収穫せよ」という鉄則があります。寒さで葉が縮み、糖度が増した時がリボッリータの作り時です。

### 簡易版：フェットゥンタ (Fettunta con Cavolo Nero)
スープを作る時間がない時は、茹でたカーボロネロをニンニクとオリーブオイルで炒め（リパッサータ）、焼いたパンに乗せるだけの「ブルスケッタ」がおすすめです。シンプルですが、野菜の味がダイレクトに伝わります。

## 3. プーリアの魂「カーボロ・リッチョ（縮れケール）」の楽しみ方

南イタリアのプーリア州では、縮れたケール（現地名：Cole Rizze）はパスタや豆の付け合わせとして愛されています。

### 代表料理：ファーヴェ・エ・フォリエ (Fave e Foglie)
**地域：プーリア州（バーリ周辺）**

「空豆（Fave）と葉野菜（Foglie）」という名の通り、乾燥そら豆のピューレと、苦味のある野菜を合わせた料理です。甘い豆のペーストと、ケールの苦味・塩気が絶妙なコントラストを生みます。

**【自然栽培野菜を活かすレシピ】**

1.  **ピューレ作り**: 乾燥そら豆（皮なし）を、崩れるまで水で煮込みます。木べらで混ぜてペースト状にし、オリーブオイルと塩で味を調えます。
2.  **ケールの準備**: カーボロ・リッチョ（ケール）を塩茹でします。日本の一般的な茹で方より少し長めに、茎が柔らかくなるまで茹でてください。
3.  **盛り付け**: 皿の半分にそら豆のピューレ、もう半分に茹でたケールを盛ります。
4.  **仕上げ**: 全体に最高級のオリーブオイルを回しかけ、お好みで赤唐辛子を添えます。パンですくいながら食べます。

> **栽培者へのアドバイス**:
> プーリアの農家は、大きな葉だけでなく、葉の付け根から出る**脇芽（Cimoni / チモーニ）**を非常に大切にします。柔らかく甘みが強いこの部分は、栽培者だけが味わえる特権です。捨てずに必ず食べてください。

## 4. 収穫過多の時の「保存食（Conserva）」技法

畑で一斉に収穫時期を迎えた時、イタリアのマンマたちは「オイル漬け」にして保存します。瓶詰めは見た目も美しく、冬の間の貴重な常備菜になります。

### ケールのソットーリオ（Sott'olio / オイル漬け）

自然栽培のケールを数ヶ月楽しむための伝統的な方法です。

**【作り方】**

1.  **洗浄とカット**: 葉を洗い、硬い茎を取り除いて適当な大きさに切ります。
2.  **酢茹で（重要）**: 水とワインビネガーを「2：1」の割合で鍋に入れ、沸騰させます（酢を入れることで保存性を高め、ボツリヌス菌のリスクを下げます）。ここで3〜4分茹でます。
3.  **乾燥**: ザルにあげ、清潔な布巾の上で**完全に**水気を乾かします。水分が残っているとカビの原因になります。一晩置くのが理想です。
4.  **瓶詰め**: 煮沸消毒した瓶に、ケール、スライスしたニンニク、唐辛子、お好みでミントやオレガノを詰めます。
5.  **注油**: 具材が完全に隠れるまでオリーブオイルを注ぎます。気泡が入らないように箸で突きながら入れます。
6.  **熟成**: 冷暗所で1ヶ月ほど寝かせると味が馴染みます。

## 5. 自然栽培農家へ送る「美味しく食べるための3つの鉄則」

最後に、ご自身で育てたカーボロネロやケールを最高に美味しく食べるためのポイントをまとめます。

1.  **茎（コスタ）を捨てない**
    葉脈や茎の部分は非常に甘みが強いです。硬い場合は、細かく刻んでスープのベース（ソフリット）と一緒に炒めたり、先に茹でてから葉と合わせたりしてください。イタリアでは「食感を残す」ことも楽しみの一つです。

2.  **「クタクタ」を恐れない**
    日本の「シャキシャキ」文化から一度離れてみてください。オリーブオイルと少量の水で蒸し煮（Stufato）にし、葉の色が鮮やかな緑から「深い暗緑色」に変わるまで火を通すと、青臭さが消え、奥深い旨味が現れます。

3.  **オリーブオイルはケチらない**
    アブラナ科の野菜は油と相性が抜群です。調理用には普通のオイルでも構いませんが、食べる直前の「仕上げのオイル（Olio a crudo）」は、ぜひ香りの良いエキストラバージンオリーブオイルを使ってください。これが料理の味を決定づけます。

古代ローマの時代から「医者いらず」と言われたこれらの野菜。
