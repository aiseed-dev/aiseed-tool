# イタリア流「レモン」の伝統料理ガイド
## 〜自家栽培だからこそできる、皮ごと食べる贅沢〜

### 1. 日本とイタリア：レモンの常識の違い

日本の食卓におけるレモンは「酸味を加える液体（果汁）」としての役割が主ですが、イタリアでは**「香りを楽しむ固形物（果実・皮・葉）」**として扱われます。

*   **日本の使い方:** 絞って捨てる、薄切りを飾る。
*   **イタリアの使い方:** 皮を刻んでパスタに混ぜる、白い部分（アルベド）をサラダにする、葉で肉を包んで焼く。

特に自然栽培で育てたレモンは、皮にワックスや農薬の心配がないため、イタリアの伝統的な食べ方を実践するのに最適です。

### 2. 品種別：料理への適性使い分け

お手元のレモンがどのようなタイプかによって、最適な調理法が異なります。イタリアのIGP（地理的表示保護）品種の特性を参考に、使い分けてみましょう。

| レモンのタイプ | 近いイタリア品種 | おすすめの調理法 |
| :--- | :--- | :--- |
| **皮が厚く、ゴツゴツしている** | **アマルフィ** (Sfusato Amalfitano) | **【生食・リキュール】**<br>香りが強いため、皮ごとスライスしてサラダやカルパッチョに。リモンチェッロ作りにも最適。 |
| **果汁が多く、酸味が強い** | **シラクーサ** / **ソレント** | **【加熱料理・ドレッシング】**<br>魚介類のソテーや、レモンパスタのソースに。酸味と香りのバランスが良い万能選手。 |
| **皮が薄く、マイルドな味** | **インテルドナート** / **マイヤー** | **【そのまま食べる】**<br>苦味が少ないため、皮ごと紅茶に入れたり、薄切りにしてお肉と一緒に焼いたりするのに向きます。 |
| **巨大で皮が非常に厚い** | **サ・ポンピア** / **獅子柚子など** | **【お菓子・保存食】**<br>生食は難しいため、皮をシロップで煮込む「蜜煮」やピール菓子に。 |

### 3. 産地直送：イタリア郷土料理レシピ

農家の方が畑から採ってきたばかりのレモンで作るべき、シンプルかつ絶品の郷土料理をご紹介します。

#### ① プロチダ島の「レモンサラダ」 (Insalata di Limoni)
**【ナポリ湾・プロチダ島】**
記事にある通り、船乗りたちが壊血病予防のために食べた「パンのレモン」の食べ方です。白い部分（アルベド）が厚いレモンが収穫できた時にぜひ試してください。

*   **材料:** レモン（皮が厚いもの）、ニンニク、ミント、唐辛子、オリーブオイル、塩。
*   **作り方:**
    1.  レモンは皮をむかず、白い部分ごと一口大の乱切り（サイコロ状）にします。
    2.  ボウルに入れ、みじん切りのニンニク、ちぎったミント、唐辛子を加えます。
    3.  たっぷりのオリーブオイルと塩で和え、少し置いて味を馴染ませます。
*   **ポイント:** 日本のレモンで苦味が気になる場合は、切った後に一度サッと湯通しするか、塩もみして洗うと食べやすくなります。パンに乗せて食べると最高の前菜になります。

#### ② アマルフィ風「レモンパスタ」 (Tagliolini al Limone)
**【カンパニア州・アマルフィ海岸】**
クリームを使わず、レモンの香りとチーズの乳化だけで作る、生産者ならではの贅沢なパスタです。

*   **材料:** パスタ（タリオリーニやスパゲッティ）、レモンの皮（すりおろし）、レモン果汁、パルミジャーノチーズ、オリーブオイル。
*   **作り方:**
    1.  ボウルにオリーブオイル、レモンの皮、果汁を入れます。
    2.  茹で上がったパスタをボウルに入れ、茹で汁を少々加えます。
    3.  ここが重要です。**チーズを加えながら激しく混ぜ合わせ、とろっとしたソース状（乳化）にします。**
*   **ポイント:** 加熱しすぎると香りが飛ぶため、フライパンで炒めず、ボウルの中で和えるのがコツです。

#### ③ カラブリア風「レモンの葉包み焼き」 (Polpette nelle foglie di limone)
**【シチリア州・カラブリア州】**
レモンの木を持っている人だけの特権。葉（Foglie）をハーブとして使います。

*   **材料:** ひき肉（またはイワシのすり身）、パン粉、卵、チーズ、レモンの葉。
*   **作り方:**
    1.  ハンバーグのタネのような肉団子を作ります。
    2.  洗ったレモンの葉で肉団子を挟みます。
    3.  オーブンやフライパンで焼きます。
*   **味わい:** 葉の精油成分が熱で肉に移り、爽やかな香りがつきます。葉は食べませんが、香り付けの役割は絶大です。

### 4. 収穫を長く楽しむ：伝統的な保存技術

一度にたくさん収穫できた時は、イタリアのマンマの知恵を借りて保存食にしましょう。

#### ① ソット・サーレ（塩レモン / Limoni sotto sale）
北アフリカの影響を受けたシチリアなどで見られる保存法です。
*   **方法:** レモンに十字の切り込みを入れ、たっぷりの粗塩と共に瓶に詰め込みます。
*   **活用:** 1ヶ月後から、皮が柔らかくなり旨味が増します。刻んで煮込み料理やパスタの隠し味に。

#### ② マルメラータ（ジャム / Marmellata di Limoni）
シチリア・シラクーサ地方の定番。
*   **こだわり:** 自然栽培なら、皮を何度も茹でこぼす必要はありません（苦味も薬効と考えます）。皮、果肉、種（お茶パックに入れてペクチン抽出用にする）を丸ごと使い、砂糖と一緒に煮込みます。パンだけでなく、熟成したチーズに添えるのがイタリア流です。

#### ③ サ・ポンピア風「蜜煮」 (Simil-Pompia Intrea)
サルデーニャ島の「怪物」レモンの調理法を応用します。皮が厚すぎて食べにくい巨大レモンや、獅子柚子などが手に入った場合に。
*   **方法:** 皮の表面の黄色い部分を薄く削ぎ落とし、分厚い白い部分だけを残します。これを蜂蜜やシロップで数時間〜半日、飴色になるまで弱火で煮込みます。
*   **味:** 苦味がキャラメルのようなコクに変わり、高級なデザートになります。
