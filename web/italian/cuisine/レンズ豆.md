# イタリア流・自家栽培レンズ豆の伝統料理ガイド
〜地中海の知恵で味わう「大地の黄金」〜

## 1. 日本の食べ方 vs イタリアの食べ方：決定的な違い

日本の家庭料理とイタリアの伝統料理で、最も大きく異なるのは**「煮込み方」**と**「相棒」**です。

| 日本の一般的な扱い | イタリアの伝統的な扱い |
| :--- | :--- |
| **下茹でする** | **下茹でせず、スープの中で直接煮る** |
| 湯で茹でてから、サラダや料理に混ぜる。 | 香味野菜（ソフリット）の旨味を吸わせながら煮る。 |
| **脇役** | **主役（プリモ・ピアット）** |
| 彩りや栄養のアクセント。 | パスタやパンと合わせ、肉の代わりのメインディッシュになる。 |
| **さっぱり** | **コクと旨味** |
| ドレッシングで食べる。 | 動物性油脂（豚肉の脂、パンチェッタ）やオリーブ油でコクを足す。 |

**【農家の皆様へのポイント】**
特に小粒の品種（カステッルッチョなど）は、**水戻し（浸水）が不要**です。洗ってそのまま鍋に入れられる手軽さが、イタリアのレンズ豆の最大の魅力です。

---

## 2. 品種別・料理の使い分けガイド

育てた（あるいは入手した）レンズ豆のタイプによって、最適な調理法が異なります。イタリアのIGP（地理的表示保護）品種の特性に合わせた使い分けです。

### ① 小粒・薄皮タイプ（カステッルッチョ系）
*   **特徴:** 皮が薄く、火の通りが早い。煮崩れしにくいが、中はクリーミー。
*   **おすすめ料理:** **スープ、煮込み（In Umido）**
*   **理由:** 浸水なしで調理でき、皮が口に残らないため、スープとして飲むのに最適です。

### ② 大粒・緑色タイプ（アルタムラ系）
*   **特徴:** 「ジャイアント・グリーン」と呼ばれ、存在感がある。ホクホクとした食感。
*   **おすすめ料理:** **付け合わせ（Contorno）、サラダ、ピューレ**
*   **理由:** 形がしっかり残るので、お肉料理の横に添えたり、茹でてオリーブオイルで和える温サラダに向いています。

### ③ 極小粒・火山性土壌タイプ（ウスティカ島系）
*   **特徴:** 濃厚な味わい。煮るとすぐにトロトロになる。
*   **おすすめ料理:** **濃厚なポタージュ、パスタソース**
*   **理由:** 短時間で煮崩れて自然なとろみがつくため、パスタによく絡みます。

## 3. イタリア各地の代表的な郷土料理

### 【ナポリ・南部】パスタ・エ・レンティッキエ (Pasta e Lenticchie)
イタリア全土で愛される、「豆とパスタ」の料理です。スープパスタとリゾットの中間のような、とろりとした食感が命です。

*   **使う品種:** 小粒種（カステッルッチョなど）またはウスティカ種
*   **作り方:**
    1.  鍋でニンニク、唐辛子をオリーブオイルで炒める。
    2.  洗ったレンズ豆と水（またはブロード）を加えて柔らかくなるまで煮る。
    3.  **同じ鍋の中に**、ミックスパスタ（余ったパスタを折ったものでOK）を直接投入する。
    4.  パスタがアルデンテになり、水分が減って全体がクリーミーになるまで煮込む。
*   **農家の知恵:** 穀物（パスタ）と豆を合わせることで、アミノ酸バランスが整い「貧者の肉」としての真価を発揮します。

### 【北イタリア・全土】コテキーノ・コン・レンティッキエ (Cotechino con Lenticchie)
大晦日に「金運」を願って食べる伝統料理です。

*   **使う品種:** 形が残りやすい中〜大粒種（アルタムラなど）
*   **作り方:**
    1.  玉ねぎ、人参、セロリのみじん切り（ソフリット）をじっくり炒める。
    2.  レンズ豆を加え、ひたひたの水で煮含める。
    3.  脂の乗った豚肉のソーセージ（コテキーノ）の輪切りを乗せて提供する。
*   **農家の知恵:** レンズ豆は淡白なので、**動物性の脂（豚の脂身やソーセージの茹で汁）**を吸わせることで、驚くほど濃厚なメインディッシュに変わります。

### 【シチリア島】ウスティカ風レンズ豆のスープ (Zuppa di Lenticchie di Ustica)
火山島ウスティカの素朴な料理です。

*   **使う品種:** ウスティカ種（なければ小粒種）
*   **作り方:**
    1.  レンズ豆、トマト、ズッキーニなどの季節野菜を水から煮る。
    2.  **野生のフェンネル（フィノッキエット）**とバジルを加える。
    3.  最後に折ったスパゲッティを加えて煮込む。
*   **農家の知恵:** 畑のハーブ（特にフェンネル系）を効かせるのがシチリア流。肉を使わずとも満足感があります。

## 4. 素材の味を活かす「シンプル・レシピ」

自然栽培で育てたレンズ豆自体の味を楽しむなら、19世紀の料理研究家ペッレグリーノ・アルトゥージも推奨するこの方法がベストです。

### レンズ豆の「イン・ウミド」（含め煮）
**～Artusi Recipe No. 414 アレンジ～**

和食の「煮豆」に近いですが、砂糖は使いません。野菜の甘みだけで仕上げます。

**材料:**
*   乾燥レンズ豆：200g
*   香味野菜（玉ねぎ、セロリ、人参）：各少量、みじん切り
*   パンチェッタ（またはベーコン）：30g（あれば）
*   バターまたは良質なオリーブオイル：適量
*   塩、胡椒：適量

**手順:**
1.  レンズ豆をたっぷりの水で茹でる（小粒なら20〜30分）。少し芯が残るくらいで湯を切る。
2.  別の鍋でバター（またはオイル）を熱し、香味野菜とパンチェッタをしんなりするまで炒める。
3.  1のレンズ豆を2の鍋に入れ、ひたひたのブロード（または水）を加える。
4.  水分が飛び、豆がふっくらして艶が出るまで弱火で煮含める。
5.  最後に塩・胡椒で味を調える。

**ポイント:** 出来立てよりも、一度冷まして味が染み込んだ翌日が最高に美味しいです。

## 5. 保存の伝統技法

イタリアの農家にとって、収穫後のレンズ豆は貴重な冬の保存食です。

### 天日干し (Essiccazione naturale)
最も伝統的かつ重要な保存法です。
*   **方法:** 鞘が茶色くなったら株ごと引き抜き、畑や軒下で完全に乾燥させます（シチリアのヴィッラルバでは「Manate」と呼ばれる束にします）。
*   **注意:** 北米産と違い、収穫前の除草剤（乾燥剤）を使わない自然栽培の場合、乾燥ムラが出やすいです。完全に乾燥させてから脱穀し、瓶に入れて冷暗所で保存してください。虫がつかないよう、唐辛子やローリエを一緒に入れるのがイタリアの知恵です。

### レンズ豆のオイル漬け (Lenticchie sott'olio) ※調理後の保存
乾燥豆が基本ですが、茹でた豆が余った場合の保存法です。
1.  固めに茹でたレンズ豆の水気を完全に切る。
2.  清潔な瓶に入れ、ニンニク、ミント、少量の酢、塩を加える。
3.  全体が浸かるようにオリーブオイルを注ぐ。
    *   冷蔵庫で1週間ほど保存可能。そのままアンティパスト（前菜）になります。
