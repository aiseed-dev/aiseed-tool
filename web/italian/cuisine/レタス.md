自然栽培で野菜を育てる皆様へ。

ご自身の手で育てたレタスは、スーパーに並ぶものとは香りの強さも、葉の厚みも違うはずです。日本では「レタス＝生サラダ」のイメージが強いですが、イタリアでは古代ローマの時代から、加熱したり、保存食にしたりと、驚くほど多様な方法で楽しまれてきました。

特に、外葉の苦味や、とう立ち（抽苔）してしまった芯の部分まで美味しく食べる知恵は、栽培者だからこそ実践できる特権です。イタリアの伝統に学ぶ、レタスの魅力を余すことなく味わうためのガイドをお届けします。

# イタリア流・レタスの伝統料理ガイド
**〜畑の恵みを味わい尽くす、栽培者のための食卓〜**

## 1. 日本とイタリア：食べ方の決定的な違い

日本の食卓とイタリアの食卓で、レタスの扱いには大きく2つの違いがあります。

### ① 「生」だけではない、「加熱」の文化
イタリアでは、レタスをほうれん草やキャベツと同じように**「加熱調理用野菜」**としても扱います。特に、硬くなった外葉や、少し苦味が出たものは、加熱することで甘みが引き立ち、トロリとした独特の食感に生まれ変わります。
*   **日本：** シャキシャキ感を重視（生食中心）。
*   **イタリア：** クタクタになるまで煮込む、焼く、リゾットにする（加熱も一般的）。

### ② ドレッシングは使わない、「インサラータ」の儀式
イタリア語でサラダを意味する「インサラータ（Insalata）」の語源は「塩（Sale）」です。あらかじめ混ぜ合わせたドレッシングをかけるのではなく、食卓で食べる直前に**「塩 → 酢（またはレモン） → オリーブオイル」**の順に調味します。
この順序が重要です。最初にオイルをかけると、油膜で塩が溶けず、野菜の味がぼやけてしまうからです。

## 2. 品種別：イタリア流・料理の使い分け

栽培されているレタスのタイプに合わせて、最適な調理法を選びましょう。

| レタスのタイプ | イタリアでの呼び名 | 特徴 | おすすめの調理法 |
| :--- | :--- | :--- | :--- |
| **結球レタス** | **Cappuccia** (カプッチャ) | 肉厚で水分が多い。 | **【煮込み・詰め物】**<br>葉が丈夫なので、詰め物をしてスープ煮にしたり、半分に切ってグリルにするのに最適。 |
| **立ちレタス**<br>(ロメインなど) | **Romana** (ロマーナ) | 葉脈が太く、加熱しても食感が残る。 | **【炒め煮・オーブン焼き】**<br>イタリアの春の煮込み料理には欠かせません。甘みが強く、加熱向き。 |
| **非結球・縮れ**<br>(サニーなど) | **Crispa / Gentile** (クリスパ/ジェンティーレ) | 葉が柔らかく、ドレッシングが絡みやすい。 | **【生食・飾り】**<br>繊細な食感を活かし、シンプルなサラダに。「フォルミアのシニョリネッラ」のような美しい品種は彩りに。 |
| **カッティング**<br>(ベビーリーフ) | **Da taglio** (ダ・ターリョ) | 非常に柔らかく、甘みが強い。 | **【生食専用】**<br>「モンカリエーリのラットゥギーノ」のように、収穫後すぐに生で食べるのが最高。加熱には向きません。 |

## 3. 栽培者のためのイタリア郷土料理レシピ

畑で採れた新鮮なレタス（特に外葉や硬い部分）を活かす、代表的なレシピをご紹介します。

### ① ローマの春の煮込み「ヴィニャローラ (Vignarola)」
**【適した品種：ロメインレタス (Romana)】**

ラツィオ州の伝統料理。春に収穫される野菜をすべて鍋に入れた、農家の知恵が詰まった一皿です。

*   **材料:** ロメインレタス（1株）、グリンピース、そら豆、アーティチョーク（あれば）、新玉ねぎ、パンチェッタ（またはベーコン）、白ワイン、オリーブオイル。
*   **作り方:**
    1.  鍋にオリーブオイルとパンチェッタを入れ、脂が出るまで炒める。
    2.  刻んだ新玉ねぎを加え、透き通るまで炒める。
    3.  ざく切りにしたロメインレタス、豆類をすべて入れる。
    4.  白ワインを回しかけ、アルコールを飛ばしたら、蓋をして弱火で20〜30分「蒸し煮」にする（水は野菜から出る水分だけで十分ですが、焦げそうなら少量を足す）。
    5.  野菜がクタクタになり、甘い香りが立ったら塩胡椒で味を調える。
*   **味わい:** レタスのほろ苦さと豆の甘みが一体となり、驚くほど濃厚な旨味が出ます。

### ② 北イタリアの知恵「レタスのリゾット (Risotto alla Lattuga)」
**【適した品種：結球レタスの外葉 (Cappuccia)】**

サラダには硬すぎる外側の緑色が濃い葉を使います。捨ててしまいがちな部分が、極上のプリモ・ピアットに変わります。

*   **作り方:**
    1.  レタスの外葉を細切りにする（芯に近い硬い部分も細かく刻む）。
    2.  鍋で玉ねぎのみじん切りを炒め、米（洗わない）を加えて透き通るまで炒める。
    3.  レタスを加え、しんなりしたら白ワインを加える。
    4.  温かいブロード（野菜だし）を少しずつ加えながら、アルデンテになるまで煮る。
    5.  火を止め、バターとパルミジャーノチーズをたっぷり加え、激しくかき混ぜて乳化（マンテカーレ）させる。
*   **ポイント:** レタスをクタクタに溶かすように煮込むことで、クリームのような滑らかさが生まれます。

### ③ ナポリ風「レタスのスープ (Zuppa di Lattuga)」
**【適した品種：すべてのレタス】**

古代ローマの医師たちが「夕食に食べて安眠を得た」といわれる、胃腸に優しいスープです。

*   **作り方:**
    1.  レタスをざく切りにし、ジャガイモの薄切りと共にブロードで煮込む。
    2.  柔らかくなったら、ハンドブレンダーで撹拌してポタージュ状にするか、そのままの食感を楽しむ。
    3.  器に盛り、トーストしたパンを添え、上質なオリーブオイルをたっぷりと回しかける。

## 4. 畑の保存食：古代ローマの知恵

冷蔵庫がなかった時代、コルメラなどの農学者が書き残した保存法を現代風にアレンジしました。

### レタスの芯のピクルス (Gambi di Lattuga Sott'aceto)
とう立ち（抽苔）してしまい、茎が伸びてしまったレタス。「失敗した」と捨てないでください。古代ローマでは、この茎こそがご馳走でした。

*   **材料:** とう立ちしたレタスの茎（皮が硬ければ剥く）、白ワインビネガー、水、塩、黒胡椒、ローリエ、ニンニク。
*   **作り方:**
    1.  茎を一口大の長さに切り、塩茹でして少し柔らかくする（茹で過ぎないよう注意）。
    2.  水気をしっかり拭き取り、消毒した瓶に詰める。
    3.  小鍋にビネガーと水（1:1）、香辛料を入れて沸騰させ、熱いまま瓶に注ぐ。
    4.  数日寝かせれば完成。
*   **食べ方:** コリコリとした食感は、まるで山クラゲやアスパラガスのよう。肉料理の付け合わせや、刻んでタルタルソースに入れます。

## 5. 自然栽培レタスを最高に美味しく食べるコツ

**「ルージアのレタス (Insalata di Lusia)」** のように、ミネラル豊富な土壌で育ったレタスは、それ自体に塩気と旨味があります。

自然栽培で育てた皆様のレタスも、肥料過多のエグみがない、清らかな味がするはずです。まずは、**「何もかけずに」** 一枚食べてみてください。
そのあと、ほんの少しの塩と、最高のオリーブオイルだけで味わう。それが、イタリア人が最も愛するレタスの食べ方であり、栽培者への最大のリスペクトです。
