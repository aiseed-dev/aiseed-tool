# イタリアのレタス

*Lactuca sativa L.* — 乳白の雫に宿る地中海の葉

---

## 序論：植物学的定義とイタリア半島における位置づけ

レタス（*Lactuca sativa L.*）は、キク科（Asteraceae）アキノノゲシ属に分類される一年生または二年生の草本植物であり、イタリアの農業史および食文化において極めて中心的な役割を果たしてきた野菜です。イタリア語で「ラットゥーガ（Lattuga）」と呼ばれるこの植物の語源は、ラテン語の「lacteus（乳のような）」または「lac（乳）」に由来します。これは、植物体が成熟期に達した際、茎や葉を切断すると分泌される白色の乳状液（ラテックス）の存在に起因しています。この乳液には「ラクトゥカリウム（Lactucarium）」と呼ばれる苦味成分が含まれており、古くから穏やかな鎮静作用や催眠作用を持つ物質として認識されてきました。

イタリアを中心とする欧州連合（EU）域内において、レタスは単なる葉菜類としての経済的価値を超え、地域固有のテロワール（土壌・気候条件）を反映した生物多様性の象徴となっています。ここでは、古代ローマ時代から現代に至るまでのイタリアにおけるレタスの歴史的変遷、伝統的な栽培技術、地域ごとの固有品種（IGP、PAT）、そしてその文化的・医学的意義について詳述します。

| 植物学的分類 | 詳細 |
| :---- | :---- |
| 界 | 植物界 (Plantae) |
| 科 | キク科 (Asteraceae) |
| 属 | アキノノゲシ属 (*Lactuca*) |
| 種 | *Lactuca sativa* |
| イタリア語名 | Lattuga |
| 主要な伝統的形態 | Cappuccia（結球）、Romana（立ち）、Da taglio（カッティング）、Crispa（縮れ） |

## 古代における起源と伝播：メソポタミアからローマへ

レタスの栽培化の歴史は古く、その起源は古代エジプトやメソポタミアの肥沃な三日月地帯にまで遡ることができます。シュメール人がチグリス・ユーフラテス川流域で最初の栽培を行った可能性が示唆されており、彼らは灌漑農業を通じてタマネギや豆類と共にレタスを生産していました。

### エジプトとローマの文化的対比：媚薬から鎮静剤へ

レタスがイタリア半島へ伝播する過程で、その象徴的意味合いは劇的な転換を遂げました。古代エジプトにおいて、レタスは豊穣の神「ミン（Min）」に捧げられる神聖な植物であり、その直立した形状と白い乳液から、精力増強や媚薬としての効果を持つと信じられていました。

しかし、ギリシャを経由してローマに伝わると、この解釈は逆転しました。古代ローマの医師や博物学者たちは、レタスが持つ「冷やす」性質に注目し、それを情熱を鎮め、睡眠を誘う「鎮静剤」として再定義しました。この医学的見解の変遷は、ローマ社会におけるレタスの消費形態や調理法に深い影響を与えることとなりました。

## 古代ローマにおけるレタス栽培と農学的発展

古代ローマ時代、レタスは「ラクトゥーカ（lactuca）」として広く栽培され、ローマの食卓に欠かせない存在となりました。当時の文献、特に大プリニウスの『博物誌（Naturalis Historia）』やコルメラの『農耕論（De Re Rustica）』には、現代の品種の祖先とも言える多様なレタスに関する詳細な記述が残されています。

### プリニウスの『博物誌』に見る品種の多様性

紀元1世紀の博物学者ガイウス・プリニウス・セクンドゥス（大プリニウス）は、その著書の中で当時のレタス栽培について網羅的な記録を残しています。彼はレタスを播種時期や種子の色によって分類し、以下のような特徴を記述しています。

* 黒い種子（Nigra）: 1月に播種される品種。
* 白い種子（Alba）: 3月に播種される品種で、寒さに強いとされます。
* 赤い種子（Ruffa/Rossa）: 4月に播種される品種。

さらにプリニウスは、地理的起源や形態的特徴に基づく具体的な品種名を挙げています。これらは現代のイタリア地方品種のルーツを探る上で極めて重要な資料です。

| 古代ローマの品種名 | 特徴と記述 |
| :---- | :---- |
| Cappadocia（カッパドキア種） | 葉が厚く、生命力が強い。結球する性質があったとされます。 |
| Graeca（ギリシャ種） | 茎が白く、特定の栽培法を必要としました。 |
| Crispa（クリスパ種） | 葉が縮れており、現代のエンダイブやフリルレタスの前身と考えられます。 |
| Laconian（ラコニア種） | 「Caulodes」とも呼ばれ、低くずんぐりとした形態を持ちます。 |
| Purpurea（紫種） | 赤紫色の葉を持つ品種で、装飾的価値もありました。 |
| Caecilius Metellus（カエキリウス・メテッルス種） | 当時の有力者カエキリウス・メテッルスの名にちなんで名付けられた改良種です。 |

プリニウスはまた、特定の栽培技術についても言及しています。例えば、砂浜の砂を株の中心に盛り上げたり、葉を縛って日光を遮断したりすることで、葉を白く柔らかくする「軟白栽培（ブランチング）」の技術が既に存在していました。これは苦味を抑え、視覚的な美しさを追求するローマ人の美的感覚と美食への探求心を示しています。

### コルメラによる栽培管理の詳述

農学者ルキウス・ユニウス・モデラトゥス・コルメラは、『農耕論』において、より実践的な栽培指針を示しました。彼はレタスが「湿気と肥料（fimo）」を好む作物であることを強調し、特に移植（replantare）の重要性を説きました。苗がある程度の大きさになった段階で間引きや移植を行うことで、より大きく充実した結球（または株）を得ることができるという技術は、現代の農業技術と通底するものです。

また、コルメラはレタスの保存方法についても記述しており、塩水と酢を用いた漬物（Pickling）のような加工が行われていたことが示唆されています。これは、冷蔵技術のない時代において、旬の野菜を年間を通じて利用するための知恵でした。

### アウグストゥス帝とレタスの逸話

ローマ皇帝アウグストゥスが重病を患った際、侍医アントニウス・ムサがレタスを用いた治療を行い、皇帝の命を救ったという逸話は、レタスの地位を決定的に高めました。この出来事以降、レタスは単なる食材を超え、高貴な「薬膳」としての名声を獲得し、貴族の庭園での栽培がさらに奨励されるようになりました。当時の人々にとって、レタスは身体の熱を取り除き、体液のバランスを整える重要な医療資源でもあったのです。

## 古代ローマのガストロノミー：調理法と食習慣

現代においてレタスは主に生食されますが、古代ローマにおける消費形態はより多様でした。アピキウスの料理書『料理帖（De Re Coquinaria）』からは、当時の洗練されたレタス料理の数々を読み取ることができます。

### 「インサラータ」の語源と調理法

「サラダ（Insalata）」という言葉は、ラテン語の「Insalare（塩を振る）」に由来します。ローマ人はレタスの葉に塩、オリーブオイル、そして酢（Aceto）をかけて食しました。これは現代のドレッシングの原点です。しかし、彼らは生食だけでなく、加熱調理も頻繁に行っていました。

特に、成長して硬くなった外葉や、とう立ちした茎の部分は、茹でたり煮込んだりして食されました。アピキウスのレシピには、以下のような調理法が記録されています。

* Agrestes Lactucae（野性味のあるレタスのサラダ）: 酢、少量のガルム（魚醤）、オイルで和えるシンプルな料理。消化を助け、鼓腸（ガス）を防ぐ効果があるとされました。
* Patina（パティーナ）: レタスの芯（Core/Grumolo）を用いたオムレツまたはフランのような料理。レタスの芯を細かく刻み、ガルム、カロエヌム（濃縮ブドウ果汁）、オイル、胡椒、卵と混ぜ合わせ、加熱して固めます。レタスの芯が持つ独特の苦味と香りが、甘いブドウ果汁や塩気のある魚醤と複雑なハーモニーを生み出しました。

### オキシポルム（Oxyporum）と消化促進

ローマ人は、レタスを食べる際に「オキシポルム」と呼ばれる混合調味料を添えることがありました。これは酢、蜂蜜、様々なスパイス（クミン、ジンジャー、ルーなど）を混ぜたものであり、レタスの「冷やす」性質を中和し、胃腸の働きを活発にするために用いられました。この医食同源的なアプローチは、ローマ人の食に対する深い洞察を示しています。

## 中世からルネサンス期：修道院医学と品種の洗練

ローマ帝国の崩壊後、高度な農業技術の多くは失われましたが、レタスの栽培は修道院の壁の中で脈々と受け継がれました。中世において、レタスは主に薬草園（Hortus Simplicium）で栽培され、修道士たちの食事や病人の治療に用いられました。

### 『健康全書（Tacuinum Sanitatis）』における評価

14世紀後半に北イタリアで製作された『健康全書（Tacuinum Sanitatis）』は、中世後期の健康観と食文化を知る上で貴重な資料です。この書物において、レタスは以下のように分類・評価されています。

| 分類項目 | 記述内容 |
| :---- | :---- |
| 性質（Natura） | 冷・湿（第2度）。 |
| 効能 | 胆汁の過剰を抑える、喉の渇きを癒やす、不眠症を改善する、性欲を抑制する（特に遺精や淋病に対して）。 |
| 推奨される対象 | 「熱い」体質の人、若者、夏場。 |
| 調理の注意点 | 消化を良くするために、油や塩、酢と共に摂取することが推奨されます。 |

特筆すべきは、「他の野菜よりも良質な血液を作る」という記述です。これは、中世の人々がレタスを単なる水分補給源ではなく、造血作用のある栄養価の高い食品として認識していたことを示しています。

### ルネサンス期の品種改良とカステルヴェトロの記述

ルネサンス期に入ると、イタリアの園芸技術は再び黄金期を迎えます。16世紀から17世紀にかけて、貴族の庭園ではより美味で美しい野菜の作出が競われました。この時期に、現代の「ロメインレタス（Lattuga Romana）」や「結球レタス（Lattuga Cappuccia）」の原型が確立されたと考えられます。

1614年にジャコモ・カステルヴェトロが著した『イタリアで食べられている全ての根、草、果実についての簡潔な物語（Brieve racconto di tutte le radici...）』は、当時のイタリアのサラダ文化がいかに先進的であったかを伝えています。彼は、イタリア人が多種多様な葉野菜（栽培種および野生種）を組み合わせてサラダ（Insalata）を作り、食事の最初に前菜として、あるいは最後に消化剤として楽しんでいたことを詳述しています。これは、野菜を生食する習慣が乏しかった北ヨーロッパ諸国に対し、イタリアの食文化の優位性を示すものでした。

## イタリアの地域固有品種と地理的表示（IGP/PAT）

イタリアのレタス栽培の真髄は、その驚くべき地域多様性にあります。南北に長いイタリア半島は、異なる気候帯と土壌条件を有しており、それぞれの土地に適応した独自のレタス品種（Ecotipo）が数世紀にわたって選抜・維持されてきました。現代において、これらの品種の一部はEUの地理的表示保護（IGP）や、イタリア政府の伝統的農産食品（PAT）として認定され、その歴史と品質が法的に守られています。

### ヴェネト州：ルージアのインサラータ（Insalata di Lusia IGP）

歴史と背景： ヴェネト州ロヴィーゴ県ルージア（Lusia）地域におけるレタス栽培の歴史は、20世紀初頭に遡ります。当初は自家消費用でしたが、1933年の生産者記録には既に「latuga」や「salata」の文字が見られ、商業化の兆しがありました。1950年代には地域の主要産品となり、1960年代にはヴェローナから導入された「Gentile（ジェンティーレ）」種が定着しました。2009年、その品質と歴史的背景が認められ、ヨーロッパで唯一のIGP（保護地理的表示）認定レタスとなりました。

テロワールと特性： ルージアの土壌は、アディジェ川の氾濫によって運ばれた砂質の堆積土であり、ミネラル分が極めて豊富です。特にカリウムとカルシウムの含有量が高く、これがルージアのレタスに独特の「天然の塩味（Sapidita）」を与えています。このため、消費者は塩を振らなくても十分にその風味を楽しむことができるとされます。

認定品種： IGP規定では、以下の2品種が認められています。

1. Varietà Capitata (Cappuccia): 結球型。重量200-450g。葉は波打ち、コンパクトで、食感はクリスピー（croccante）。
2. Varietà Crispa (Gentile): 非結球・縮れ型。重量150-450g。明るい緑色で、葉は気泡状（bollosa）。食感は非常に柔らかいです。

### ピエモンテ州：モンカリエーリのラットゥギーノ（Lattughino di Moncalieri PAT）

歴史と伝承： トリノ南東部に位置するモンカリエーリおよびニケリーノ周辺では、少なくとも1900年代初頭から特定のレタス品種が栽培されてきました。都市近郊農業として発展したこの地域のレタスは、生産者が自ら最も優れた株を選抜し、種を採る「自家採種」によって遺伝的純度を保ってきました。

品種の詳細： 「ラットゥギーノ（小さなレタス）」の名が示す通り、小ぶりの状態で収穫されるのが特徴です。主に以下の2つのタイプが存在します。

1. Parella（パレッラ）: 「Lattuga Cappuccio」の選抜種。濃い不透明な緑色で、葉は短く気泡が多いです。春の味覚として知られ、直径10-12cm、重量20-40gという極めて小さなサイズで収穫されます。
2. Zuccherina（ズッケリーナ）: 「Lattuga Romana」の選抜種。明るく光沢のある緑色で、葉は長く滑らかです。名前の通り「砂糖（Zucchero）」のような強い甘みを持ち、パレッラよりもさらに柔らかいです。

これらの品種は、大規模流通に適さない繊細さを持つため、地元の市場やレストランでのみ消費される希少な存在です。

### ラツィオ州：フォルミアのシニョリネッラ（Lattuga Signorinella di Formia PAT）

文化的背景： ラツィオ州南部の海岸沿い、フォルミアとガエタの地域で栽培される伝統品種です。「シニョリネッラ（Signorinella）」とは「お嬢さん」や「未婚の女性」を意味する愛称であり、その繊細で美しい外観が、フリルやレースのついたスカートを履いた女性を連想させることに由来します。

形態的特徴：

明るい緑色の葉を持ち、その縁（エッジ）がラディッキオのような赤紫色に色づくのが最大の特徴です。この独特の配色は、フォルミア特有の海洋性気候と土壌条件によって発現します。地元では「Lattuga a gonnella（スカート状のレタス）」とも呼ばれ、柔らかさと見た目の美しさから、祝祭のサラダや飾り付けに重宝されてきました。

### サルデーニャ州：セストゥのレタス（Lattuga di Sestu PAT）

栽培技術の伝統： サルデーニャ州セストゥ（Sestu）周辺で栽培されるこの品種は、800gから1kgにも達する巨大な結球を形成します。カリアリ近郊のこの地域では、「レット・カルド（Letto caldo＝温床）」と呼ばれる伝統的な育苗技術が用いられてきました。これは、ブドウの搾りかす（Vinacce）と家畜の堆肥を発酵させ、その発酵熱を利用して冬場に苗を育てる手法です。

風味と利用：

「Iceberg」のようなカリカリとした食感を持ちながら、「Romana」のような濃厚な甘みを併せ持ちます。地元では、復活祭（Pasqua）の時期に特に好んで食されることから、春の訪れを象徴する野菜となっています。

## 農業生物多様性の保全と現代的課題

イタリアにおける伝統的なレタス品種は、F1種（一代雑種）の普及や流通のグローバル化により、絶滅の危機に瀕しているものも少なくありません。しかし、スローフード協会や各地の農業機関による保全活動（プレシディオ、味の箱舟）が、これらの「生きた文化遺産」を守る防波堤となっています。

### 「味の箱舟（Arca del Gusto）」による保護品種

スローフード財団は、消滅の危機にある品種を「味の箱舟」に登録し、その種子と食文化の保存を推進しています。

* Lattuga Ubriacona（酔っぱらいレタス）: トレンティーノ地方や北イタリアの一部で見られる品種です。葉が赤ワインのような色を帯びることから名付けられました。鉄分やビタミンが豊富で、不眠症に対する民間療法としても利用されてきました。
* Lattuga di Albino（アルビーノのレタス）: ロンバルディア州ベルガモ県アルビーノで発見された品種です。一人の高齢女性によって守られていました。特筆すべきは、苦味成分であるラクトゥカリウム（乳液）が極めて少なく、非常に繊細で消化が良い点です。
* Lattuga Regina di Maggio（5月の女王）: 春の早い時期に収穫される品種で、葉の縁にピンクから紫の美しい色が入ります。成長が早く、柔らかいため、家庭菜園での栽培に適しています。

### 伝統的種子交換システム

イタリアの伝統野菜生産において重要なのは、種子が商品としてではなく、農家間の信頼関係に基づく交換物として扱われてきた歴史です。セストゥやモンカリエーリの農家は、最も優れた株を収穫せずに畑に残し、花を咲かせて種を採ります。こうして得られた種子は、地域の気候変動や病害虫に対する適応力を世代ごとに獲得していきます。これは、遺伝的多様性を維持する上で、現代のバイオテクノロジーにも劣らない重要なプロセスです。

### イノベーション：垂直農法と伝統品種の融合

一方で、イタリアはレタス栽培の技術革新においてもEUをリードしています。ロンバルディア州ヴェローラヌオーヴァなどで導入されている垂直農法（Vertical Farming）は、水耕栽培とLED照明を組み合わせ、農薬不使用、節水（従来比95%削減）、土壌不使用での生産を実現しています。興味深いのは、この最先端施設において、生産効率の高い品種だけでなく、「Multifoglia variegata（斑入り多葉）」などの伝統的な特徴を持つ品種も栽培されている点です。これは、伝統的な味と最新の持続可能技術の融合事例として注目されます。

## 結論：イタリア文化の鏡としてのレタス

イタリアにおけるレタスの歴史を紐解くと、それは単なる農業史にとどまらず、イタリア半島の文明史そのものを映し出していることがわかります。古代エジプトからもたらされた「異国の媚薬」は、ローマ人の合理精神によって「医学的な鎮静剤」へと転換され、中世の修道院で「癒やしの草」として守り抜かれました。そしてルネサンス以降、美食の対象として洗練され、地域ごとの多様な品種へと分化していきました。

インサラータ・ディ・ルージアのミネラル感、モンカリエーリの甘み、フォルミアの美しさ、それらすべてはイタリアの多様なテロワールと、それを守り続けてきた農民たちの知恵の結晶です。現代において、IGPやPATといった認証制度、あるいはスローフードの活動は、この豊かな生物多様性を未来へ継承するための重要な枠組みとして機能しています。

イタリアのレタス文化は、「Insalare（塩を振る）」という単純な行為から始まり、数千年の時を経て、医学、農学、そしてガストロノミーが交差する深遠な領域へと発展しました。その一枚の葉には、地中海の太陽と歴史の重みが凝縮されているのです。
