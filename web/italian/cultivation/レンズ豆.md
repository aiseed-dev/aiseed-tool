日本の自然栽培・有機栽培に取り組む生産者の皆様へ向けた、イタリア伝統野菜「レンズ豆」のガイドを作成しました。

イタリアでは、レンズ豆は単なる「豆」ではなく、土地の記憶（テロワール）を宿す文化遺産として扱われています。日本の一般的な市場にあるレンズ豆とは一線を画す、その個性と栽培の視点をご紹介します。

---

# イタリアの伝統レンズ豆：地中海の「金貨」を日本の畑へ

レンズ豆（*Lens culinaris*）は、人類が最も古くから栽培してきた作物の一つです。イタリアでは、その形が硬貨に似ていることから「富と繁栄の象徴」とされ、特に大晦日に食べると翌年お金に困らないという言い伝えがあります。

日本のスーパーで見かけるレンズ豆の多くは北米産などの大量生産品ですが、イタリアの在来種は「品種」と「土地」が結びつき、驚くほど多様な風味と食感を持っています。

## 1. イタリア在来種と一般流通品種の違い

日本の一般的な流通品（皮なしの赤レンズ豆や、北米産の茶色いレンズ豆）と、イタリアの伝統品種（特にIGPやプレシディオ認定のもの）には、決定的な違いが3つあります。

### ① 「皮」の薄さと食感
*   **一般流通品:** 皮が厚く口に残るか、あるいは皮が除去（脱穀）されており煮崩れしやすいものが主流です。
*   **イタリア在来種:** 特にカステッルッチョ産などに代表される品種は、**皮が極めて薄い**のが特徴です。そのため、**調理前の浸水（水戻し）が不要**で、そのまま鍋に入れて煮ることができます。火の通りが早いにもかかわらず、煮崩れしにくく、パスタのような「アルデンテ」の食感を楽しめます。

### ② 風味の複雑さ（テロワール）
*   **一般流通品:** 味付けのベースとしては優秀ですが、豆自体の主張は穏やかです。
*   **イタリア在来種:** 栗（カスターニャ）のような甘み、土の香り、ミネラル感など、品種ごとに明確な個性があります。「豚の餌」ではなく「美食」として改良されてきた歴史があるため、塩とオリーブオイルだけで一品料理になるポテンシャルを持っています。

### ③ 収穫後の処理（グリホサート不使用）
*   イタリアの伝統的な産地では、乾燥した夏の日差しを利用して**自然乾燥**させます。大規模農業で見られるような、収穫前に除草剤（乾燥剤）を散布して強制的に枯らす手法を取らないため、自然派の消費者にとって安心できる食材です。

## 2. イタリアの主要品種とその特徴

イタリアには大きく分けて「小粒種（ミクロスペルマ）」と「大粒種（マクロスペルマ）」が存在します。

### カステッルッチョ・ディ・ノルチャ (Lenticchia di Castelluccio di Norcia IGP)
*   **産地:** ウンブリア州（標高約1450m）
*   **タイプ:** 小粒種
*   **特徴:** **「レンズ豆の王様」**。皮が非常に薄く、浸水不要で約20〜30分で煮えます。小粒で丸く、色は緑や茶色が混ざったモザイク模様。
*   **味:** 繊細でナッツのような風味。煮崩れしにくいので、煮込み料理だけでなくサラダにも最適です。

### アルタムーラ (Lenticchia di Altamura IGP)
*   **産地:** プーリア州
*   **タイプ:** 大粒種
*   **特徴:** **「ジャイアント・グリーン」**と呼ばれる大型種。平べったく、緑色がかった色味をしています。
*   **味:** 大粒ならではの「肉感」があり、食べ応え抜群。鉄分が豊富で、メインディッシュの付け合わせとして存在感を発揮します。

### ウスティカ島 (Lenticchia di Ustica - Presidio Slow Food)
*   **産地:** シチリア州ウスティカ島（火山島）
*   **タイプ:** 極小粒種
*   **特徴:** **イタリアで最も小さい**とされるレンズ豆。火山性土壌で無灌漑栽培されます。
*   **味:** 濃厚でクリーミー。皮が柔らかく、45分ほど煮込むと素晴らしいスープになります。機械化が難しく、手作業の工程が多い希少品です。

### ヴィッラルバ (Lenticchia di Villalba - Presidio Slow Food)
*   **産地:** シチリア州内陸部
*   **タイプ:** 大粒種
*   **特徴:** かつてイタリア生産の3割を占めた品種。特筆すべきは**圧倒的な鉄分含有量**（100g中10mg超）です。
*   **味:** 味が濃く、栄養価が高い。伝統的に手刈りされ、畑で束にして乾燥されます。

### オナーノ (Lenticchia di Onano - PAT)
*   **産地:** ラツィオ州（火山性土壌）
*   **タイプ:** 中〜大粒
*   **特徴:** 「法王のレンズ豆」と呼ばれ、ピオ9世が愛したことで有名。
*   **味:** 火山灰土壌由来のミネラル感と、繊細で甘みのある風味が特徴です。

## 3. 日本の自然栽培におけるポイント

日本の気候でイタリアのレンズ豆を栽培する場合、最大の敵は「湿気」です。イタリアの産地（特に南部や高地）は乾燥していますが、日本には梅雨があります。

### 品種選びの視点
*   **小粒種（カステッルッチョ系）:** 一般的に寒さや痩せた土地に強い傾向があります。日本の山間部や冷涼地での栽培に向いている可能性があります。
*   **大粒種（アルタムーラ系）:** 温暖な地域に向きますが、収穫期が梅雨と重なるとカビのリスクが高まります。

### 栽培環境と土壌
*   **痩せた土地を好む:** レンズ豆はマメ科特有の根粒菌により窒素固定を行うため、**肥料（特に窒素）はほとんど不要**です。肥沃すぎる土地では「つるボケ（葉ばかり茂って実がつかない）」を起こしやすくなります。
*   **水はけが命:** 過湿を極端に嫌います。高畝にするか、傾斜地、あるいは水はけの良い砂質土壌を選んでください。
*   **輪作の活用:** 麦（小麦・大麦）の収穫後の、養分が少なくなった畑や、地力を回復させたいタイミングでの作付けに適しています。イタリアでも麦との輪作が基本です。

### 収穫のタイミング
*   日本の自然栽培では、**「梅雨入り前に収穫できるか」**が勝負です。秋蒔きで越冬させ、初夏に収穫するサイクルが一般的ですが、地域の気候に合わせて播種時期を調整する必要があります。

## 栽培データ

レンズ豆は、エンドウ豆やソラマメに近い栽培サイクルを持ちますが、より乾燥を好みます。

| 項目 | データ | 備考 |
| :--- | :--- | :--- |
| **収穫期** | 5月下旬 〜 7月上旬 | 梅雨入り前の乾燥した時期に収穫するのが理想。 |
| **播種期** | **秋まき:** 10月中旬〜11月<br>**春まき:** 3月〜4月 | 温暖地は秋まき（越冬）、寒冷地は春まきが一般的。 |
| **発芽適温** | 15 〜 20℃ | 涼しい気候を好みます。 |
| **播種法** | 点まき または 条まき | 株間15〜20cm程度。支柱は基本不要ですが、密植すると倒れやすくなります。 |
| **覆土** | 2 〜 3cm | 種の大きさの2〜3倍程度。 |
| **生育適温** | 10 〜 25℃ | 耐寒性はありますが、極端な寒風や霜柱には注意（必要なら不織布などで防寒）。高温多湿には弱いです。 |
| **栽培法** | 放任栽培が可能 | 肥料は控えめ（特に窒素）。土寄せをすると倒伏防止になります。混植（コンパニオンプランツ）として麦類と相性が良いです。 |
| **採種法** | 自家採種 | 鞘が茶色く乾燥したら株ごと引き抜き、雨の当たらない場所で追熟・乾燥させてから脱穀します。交雑しにくい（自家受粉）ため、初心者でも採種しやすいです。 |
| **種子寿命** | 2 〜 3年 | 条件が良ければ長持ちします。 |
| **休眠** | ほとんどなし | 収穫後、次のシーズンにすぐに蒔けます。 |
| **種子保存法** | 乾燥・冷暗所 | 湿気を吸うとカビたり発芽率が落ちるため、乾燥剤と共に密閉容器に入れ、冷蔵庫（野菜室）などで保管するのがベストです。虫（ゾウムシ）がつきやすいので注意。 |
