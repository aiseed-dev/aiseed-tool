日本の自然栽培・有機栽培の実践者の皆様へ。

かつてはガラスや石鹸を作るための「灰」の原料として、地中海経済を支えた戦略物資でありながら、現在は春を告げる美食の野菜として愛される「アグレッティ（*Salsola soda*）」。

日本では「オカヒジキ」の親戚として知られていますが、その歴史的背景や本来の味わいは非常に奥深いものがあります。イタリアの伝統品種の特性と、日本で栽培する際のポイントについて解説します。

# アグレッティ（*Salsola soda*）栽培ガイド
〜地中海の歴史が生んだミネラルの味〜

## 1. イタリア伝統品種と日本流通品種の違い

日本で一般的に流通している「アグレッティ」の種子は、品種名が特定されていない輸入種子であることがほとんどです。また、日本の食卓には近縁種の「オカヒジキ（*Salsola komarovii*）」が定着しています。まずはこの違いを理解することが重要です。

### オカヒジキとの違い
*   **食感と形状:** 日本のオカヒジキよりも、アグレッティ（*Salsola soda*）は**葉が肉厚で、茎が太く、より多肉植物に近い**見た目をしています。食感はオカヒジキ同様にシャキシャキしていますが、より「ジューシー」で、噛むと水分が弾ける感覚があります。
*   **風味:** オカヒジキが比較的淡白であるのに対し、アグレッティは**「土の香り」と「ミネラル感（ほのかな塩味）」、そして独特の「酸味」**を持っています。名前の由来がイタリア語の「Agro（酸っぱい）」にある通り、ほうれん草のような鉄分を感じさせる濃厚な味わいが特徴です。

### 日本での流通状況
日本国内で入手できる種子は、イタリアの一般種（Commercial variety）が主です。イタリア現地では、地域ごとの食文化に根ざした「在来の系統」が存在しますが、種子の寿命が極端に短い（後述）ため、特定の地方品種が日本へ安定して入ってくることは稀です。したがって、自家採種を繰り返して日本の風土に馴染ませていくことが、自然栽培における「品種作り」の第一歩となります。

## 2. イタリア認定品種（PAT）の特徴と違い

イタリア農林食糧政策省が認定する伝統的農産物（PAT）のリストには、以下の2つのアグレッティが登録されています。これらは植物学的には同じ *Salsola soda* ですが、**「誰が、どう食べてきたか」という文化的文脈（テロワール）**によって、求められる味や形状が異なります。

### ① Agretti di Roma (ローマのアグレッティ) [ラツィオ州]
*   **背景:** ローマのユダヤ人コミュニティ（ゲットー）や修道院で守られてきた系統です。
*   **特徴:** **「洗練された酸味と柔らかさ」**が重視されます。
*   **用途:** さっと茹でて、レモンとオリーブオイルだけで食べるシンプルな料理が主流です。アンチョビの塩気や、内臓料理の脂っこさを中和するための「口直し」としての役割が強いため、若く柔らかい、酸味の効いた芽が好まれます。

### ② Agretti (Liscari) (リスカリ) [エミリア＝ロマーニャ州]
*   **背景:** 北イタリアの農村地帯で「リスキ（Lischi）」と呼ばれ、農民の貴重な栄養源や、戦時中の救荒植物として利用されてきた系統です。
*   **特徴:** **「野生味と力強さ」**が特徴です。ローマのものより少し成長した、しっかりとした食感のものが使われる傾向があります。
*   **用途:** 単なる茹で野菜ではなく、ニンニクやバター（またはラード）、ソーセージと共に炒め合わせるなど、動物性脂肪と合わせた高カロリーな料理に使われます。油と合わせても負けない、しっかりとした軸の太さが求められます。

## 3. 自然栽培におけるポイント

アグレッティは、肥料や農薬がなくとも育つ強健な植物ですが、「美味しいアグレッティ」を作るにはコツがいります。

### ① 「塩」を意識する（ミネラル栽培）
アグレッティは塩生植物（ハロファイト）です。塩分のない土壌でも育ちますが、**塩分ストレスがある環境のほうが、風味が濃厚になり、食感が良くなる（シャキシャキ感が増す）**という研究結果があります。
*   **自然栽培での工夫:** 海岸沿いの畑であれば最高ですが、内陸の場合、海藻由来の資材や、希釈した海水（または天然塩をごく薄く溶いた水）を時折与えることで、原産地に近いミネラル感と甘みを引き出せる可能性があります。

### ② 種子の鮮度が命
栽培における最大のハードルは**「種子の寿命が極めて短い」**ことです。
*   **1年で寿命が尽きる:** 採取から1〜2年で発芽能力をほぼ失います。輸入種子を購入する場合、古い在庫だと発芽率が極端に低い（時には0%）ことがあります。
*   **自家採種の推奨:** 一度発芽に成功したら、必ず数株を残して大きく育て、自分で種を採ってください。その秋〜冬に採れた種を、翌春に撒くサイクルを作ることが、最も確実な栽培法です。

### ③ 収穫のタイミング
茎が木質化（硬くなる）する前に収穫する必要があります。
*   **春の味覚:** 3月〜5月が旬です。トスカーナ地方では「ロスカーノ（赤くなるもの）」とも呼ばれるように、成長しすぎると茎が赤くなり硬くなります。緑色が鮮やかで、指で簡単に折れる柔らかいうちに収穫します。

## 栽培データ

| 項目 | 内容 | 備考 |
| :--- | :--- | :--- |
| **収穫期** | 3月下旬 〜 5月 | 茎が硬くなる前、草丈15〜20cm程度で収穫。 |
| **播種期** | 12月 〜 3月 | 寒さに強いため、冬の間に撒くことも可能（トンネル等）。 |
| **発芽適温** | 10℃ 〜 20℃ | 比較的好冷性。地温が高すぎると発芽しにくい。 |
| **播種法** | 直播き（すじまき） | 移植を嫌うため、畑に直接撒くのが基本。 |
| **覆土** | 1cm 〜 2cm | 種が比較的大きいため、しっかり覆土して鎮圧する。 |
| **生育適温** | 15℃ 〜 25℃ | 春の涼しい気候を好む。夏の暑さには弱い。 |
| **栽培法** | 密植栽培 | 株間5〜10cm程度。密植気味に育てたほうが柔らかく育つ。乾燥には強いが、適度な湿り気がある方が良品になる。 |
| **採種法** | 放置して結実させる | 収穫せずに残すと、夏〜秋に開花し（地味な花）、トゲのある種ができる。株が枯れたら刈り取って乾燥させる。 |
| **種子寿命** | **極短（1年〜2年）** | **最重要項目。** 翌年には発芽率が激減する。 |
| **休眠** | あり | 採種直後は休眠している場合があるが、冬の寒さに当たることで打破される。 |
| **種子保存法** | 冷蔵保存 | 乾燥剤と共に密閉容器に入れ、冷蔵庫の野菜室で保管するが、それでも長期保存は不可。 |
