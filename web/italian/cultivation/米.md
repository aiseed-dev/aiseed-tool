日本の自然栽培家や有機農業者の皆様へ。

イタリアは欧州最大の米生産国であり、その歴史はレオナルド・ダ・ヴィンチの水利工学や、労働者たちの権利闘争と共に歩んできました。日本の「ご飯」とは異なる進化を遂げた、イタリアの「リゾット用米」。その魅力と、日本での栽培・導入に向けた視点を解説します。

# イタリア伝統野菜ガイド：米（Riso）

**学名**：*Oryza sativa*  
**分類**：イネ科イネ属  
**原産**：アジア（イタリアへの伝来は中世以降）

## 1. 日本の米とイタリア米の決定的な違い

日本の米もイタリアの米も、植物学的には同じ「ジャポニカ種（*Oryza sativa japonica*）」に属します。しかし、目指す食感と用途が全く異なるため、品種改良の方向性が大きく異なります。

| 特徴 | 日本の一般的な米（うるち米） | イタリアの伝統米（リゾット米） |
| :--- | :--- | :--- |
| **粒の形** | 短粒で丸い | **大粒で長い**（カルナローリ等）または中粒（ヴィアローネ・ナノ） |
| **デンプン構造** | アミロースが少なめ（粘りが強い） | **アミロースが多め**（粘りが少なく、煮崩れしにくい） |
| **中心部** | 透明であることが良しとされる | **「ペルラ（真珠）」**と呼ばれる不透明な芯があることが重要 |
| **調理のゴール** | ふっくら、もちもち、甘み | **アルデンテ**（芯を残す）、ソースとの乳化 |
| **食べ方** | おかずと共に食べる「主食」 | 米自体に味を含ませる「独立した料理」 |

### 「ペルラ（真珠）」の役割
イタリア米の最大の特徴は、生米の状態で中心に見える白い不透明な部分「ペルラ」です。ここが多孔質（穴が多い構造）になっており、調理時にブロード（出汁）をたっぷり吸収します。一方で、外側は煮崩れずにしっかりとした歯ごたえを保ちます。これが「アルデンテ」を生み出す秘密です。

## 2. 主要なイタリア伝統品種と特徴

イタリアには原産地名称保護（DOP）や地理的表示保護（IGP）で守られた、地域ごとの名品があります。栽培を検討する際は、それぞれの「個性」を知ることが重要です。

### ① カルナローリ (Carnaroli)
*   **別名**：「米の王（Re dei risi）」
*   **特徴**：1945年生まれ。粒が長く、デンプン含有量が高い品種です。調理しても最も煮崩れしにくく、アルデンテの状態を長く保てるため、プロのシェフに愛されます。
*   **栽培の視点**：晩生で背が高くなりやすいため、倒伏に注意が必要ですが、その分、根を深く張る力強さがあります。

### ② アルボリオ (Arborio)
*   **特徴**：1946年生まれ。イタリア米の中で最大級の粒の大きさを誇ります。アミロペクチンが多く、調理すると表面が溶け出して濃厚なクリーム状（マンテカータ）になりやすい一方、中心にはしっかり芯が残ります。
*   **栽培の視点**：海外での知名度が最も高く、「リゾット米」の代名詞的存在です。

### ③ ヴィアローネ・ナノ (Vialone Nano)
*   **認証**：**Riso Nano Vialone Veronese IGP**（ヴェネト州）
*   **特徴**：1937年生まれ。「ナノ（小人）」の名がつく通り、上記2種に比べると丸く中粒です。吸水率が非常に高く、自身の体積の2倍ものスープを吸うと言われます。
*   **栽培の視点**：ヴェローナ地方の湧き水（Risorgive）で育てられる伝統があり、清らかな水を好みます。

### ④ バラッジャの米 (Riso di Baraggia Biellese e Vercellese DOP)
*   **特徴**：特定の品種名ではなく、ピエモンテ州の「バラッジャ」と呼ばれる地域（粘土質土壌・冷涼な気候）で育った米の総称（カルナローリやアルボリオなどが含まれます）。
*   **栽培の視点**：寒暖差と粘土質土壌により、粒はやや小ぶりになりますが、密度が高く、極めて調理耐性が高い（煮崩れない）米になります。「環境が品質を作る」好例です。

## 3. 自然栽培・有機栽培におけるポイント

日本の自然栽培家がイタリア米を手掛ける場合、以下の点がアドバンテージや注意点となります。

### 品種選びは「物語」と「食味」で
日本の一般市場では「コシヒカリ」などの粘りのある米が主流ですが、イタリア米は「洋食・リゾット専用」としての明確な差別化が可能です。
*   **レストラン向け**: シェフは本物の「カルナローリ」や「ヴィアローネ・ナノ」を探しています。日本国内で鮮度の良いイタリア品種が手に入れば、大きな強みになります。
*   **直売向け**: レオナルド・ダ・ヴィンチが水路設計に関わった歴史や、女性労働者（モンディーネ）の歌と共に、「物語のある米」として提案できます。

### 背丈と肥料設計
イタリアの古い品種（カルナローリや、復活品種のマラテッリなど）は、近代的な多収量品種に比べて**背丈が高くなる傾向**があります。
*   慣行栽培（多肥）では倒伏しやすいというデメリットがありますが、**無肥料・少肥の自然栽培においては、この草丈の高さが雑草との競合に勝つ強み**になり得ます。
*   肥料を切り詰め、根を張らせる日本の自然栽培技術は、これら伝統品種のポテンシャルを引き出すのに適している可能性があります。

### 水管理と気候
イタリアの主産地（ポー川流域）は北緯45度以北で、アルプスの冷たい雪解け水を利用しています。そのため、本来は冷涼な気候や冷水に対する適応力を持っています（カヴール運河などは水温を上げる工夫もされていますが）。
*   日本の寒冷地や、水温が低い地域での栽培とも相性が良い可能性があります。
*   近年イタリアでは干ばつ対策として「乾田直播（Dry seeding）」も増えていますが、日本の水田環境（湛水）であれば、本来の品質を十分に発揮できるでしょう。

## 栽培データ

日本の一般的な水稲栽培に準じますが、晩生種が多いため登熟期の気温確保に注意してください。

| 項目 | データ | 備考 |
| :--- | :--- | :--- |
| **播種期** | 4月〜5月 | 日本の一般品種と同様、桜の咲く頃を目安に。 |
| **発芽適温** | 10〜13℃（最低） | 25〜30℃で揃いやすくなります。 |
| **生育適温** | 25〜30℃ | 開花・登熟期には十分な日照と温度が必要です。 |
| **栽培法** | 水田（湛水栽培） | 畑での陸稲栽培も可能ですが、品質・収量は水田が優ります。近年は乾田直播も可。 |
| **播種法** | 苗箱育苗・移植 または 直播 | 自然栽培では、初期除草の手間を減らすため移植栽培（田植え）が一般的です。 |
| **覆土** | 薄く（種子が隠れる程度） | 育苗箱の場合。 |
| **収穫期** | 9月〜10月 | **晩生種（奥手）が多い**傾向があります。早刈りは避け、完熟を待ちます。 |
| **採種法** | 自家採種可能 | 他の品種と交雑しないよう、隔離するか開花時期をずらします。充実した穂を選抜して次年度用にします。 |
| **種子寿命** | 2〜3年 | 常温でも持ちますが、発芽率維持のためには低温低湿が望ましいです。 |
| **休眠** | あり | 収穫直後は発芽しにくい場合があります。翌春には覚めます。 |
| **種子保存法** | 籾（もみ）の状態で保存 | 籾殻がついたまま、通気性の良い袋に入れ、冷暗所（15℃以下・湿度低め）で保管するのがベストです。 |

**注意点**: イタリア品種は脱粒（収穫前に粒が落ちること）しやすい性質を持つものがあります。収穫適期を逃さないよう観察してください。
