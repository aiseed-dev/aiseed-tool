日本の自然栽培・有機栽培に取り組む皆様へ。

イタリア野菜、特にエンドウマメ（Piselli）は、単なる「甘い春の味覚」にとどまらず、1万年以上の農業史と各地のテロワール（風土）が凝縮された作物です。日本の市場で一般的な品種とは一線を画す、イタリア伝統品種の魅力と選び方、そして栽培のポイントをご紹介します。

# イタリア伝統エンドウマメ栽培ガイド：歴史を味わう「種」の選び方

イタリアのエンドウマメは、古代ローマの「貧者の肉」からルネサンス期の「王侯の珍味」まで、劇的な歴史を歩んできました。
F1種（一代交配種）が主流の現代において、イタリアの地方品種（ランドレース）を育てることは、その土地の気候風土に適応してきた「強さ」と「物語」を畑に取り入れることを意味します。

## 1. 日本の一般品種とイタリア伝統品種の違い

日本のスーパーで見かけるエンドウ（特にスナップエンドウや絹さや）と、イタリアの伝統的なエンドウには、味や利用法に明確な違いがあります。

| 特徴 | 日本の一般品種 (F1主流) | イタリア伝統品種 (固定種・在来種) |
| :--- | :--- | :--- |
| **味の方向性** | **糖度重視**。フルーツのような甘さと、プリッとした食感が好まれる。 | **風味（コク）重視**。甘みもあるが、豆本来の旨味やデンプン質、香りが豊か。 |
| **主な用途** | 軽く茹でてサラダ、炒め物。鞘ごと食べる（スナップ・絹さや）が多い。 | **加熱調理が前提**。リゾット、スープ、煮込み。鞘から豆を出して使う（実エンドウ）が基本。乾燥豆としても多用。 |
| **草姿** | 矮性（背が低い）や、揃いが良い品種が多い。 | 高性（1.5m以上）や、環境適応力が高い暴れる品種も多い。 |
| **強み** | 収穫時期が揃う、見た目が綺麗、誰が食べても甘い。 | 乾燥や痩せ地に強い、病気に強い、料理に深みが出る。 |

**ここがポイント：**
日本のエンドウは「野菜（ビタミン源）」としての側面が強いですが、イタリア（特に南部）のエンドウは「穀物・タンパク源（主食の一部）」としての性格を色濃く残しています。そのため、**「煮込んでも煮崩れず、クリーミーになる」「出汁が出る」**といった特性が重視されます。

## 2. 品種別ガイド：あなたの畑に合うのはどれ？

イタリアの品種は、大きく「北部の生食・甘味系」と「南部の乾燥・煮込み系」に分かれます。ご自身の畑の環境や、食べたい料理に合わせて選んでみてください。

### 【北部・春の味覚系】甘みと鮮度を楽しむ
湿潤で豊かな土壌、あるいは春の穏やかな気候に向いています。

*   **Biso di Borso (ボルソ・デル・グラッパのビーゾ)** [ヴェネト州]
    *   **特徴:** 「緑の黄金」と呼ばれ、かつてはヴェネツィアのドージェ（総督）に献上された最高級品。皮が薄く、非常に甘いです。
    *   **用途:** ヴェネトの伝統料理「リジ・エ・ビジ（豆入りリゾット）」には必須。生食も可能。
    *   **栽培:** 繊細な味を持つため、過度な肥料は避け、石灰質の土壌を好みます。

*   **Verdone Nano (ヴェルドーネ・ナーノ)** [ヴェネト州]
    *   **特徴:** 「矮性の濃い緑」の名の通り、草丈が低く管理しやすい品種。鮮やかな緑色が特徴。
    *   **用途:** 甘みが強く、パスタやサイドディッシュに最適。
    *   **栽培:** 矮性なので支柱が低くて済み、風の強い地域でも育てやすいです。

*   **Pisello di Lavagna (ラヴァーニャのエンドウ)** [リグーリア州]
    *   **特徴:** 海沿いの温暖な気候で育まれた品種。草丈が高く（1.5m程）、早生（早く収穫できる）です。
    *   **用途:** 魚料理との相性が抜群。生のまま食べられるほど甘い。
    *   **栽培:** 蔓が伸びるのでしっかりした支柱が必要。

*   **Pisello di Casalborgone (カザルボルゴーネのエンドウ)** [ピエモンテ州]
    *   **特徴:** 小粒で甘みが強い。3つの伝統的な系統が混ざり合って維持されています。
    *   **用途:** サラダやフリッタータ（オムレツ）に。

### 【南部・野生味系】乾燥と過酷な環境に強い
日差しが強く、水はけの良い場所、あるいは不耕起栽培などのラフな環境に向いています。

*   **Pisello Centogiorni (チェントジョルニ / 100日エンドウ)** [カンパニア州]
    *   **特徴:** ヴェズヴィオ火山の麓で育つ。播種から収穫まで約100日と早い。無灌漑でも育つ強健種。
    *   **用途:** 皮が薄いので生でもいけますが、乾燥豆としても優秀。
    *   **栽培:** 根張りが良く、乾燥に非常に強いです。

*   **Pisello Nano di Zollino (ゾッリーノの矮性エンドウ)** [プーリア州]
    *   **特徴:** 「乾燥農法（Arido coltura）」の象徴。水やりなしで育つ極めてタフな矮性種。
    *   **用途:** 煮込むとクリーミーになるため、スープやピューレに。
    *   **栽培:** 痩せ地でも育ちます。過湿を嫌うため、水はけの良い畝で。

*   **Pisello Secco di Vitigliano (ヴィティリアーノの乾燥エンドウ)** [プーリア州]
    *   **特徴:** 「クチウロ（煮えやすい）」の別名を持ち、調理時間の短さが自慢。
    *   **用途:** 乾燥豆専用。消化が良いとされています。
    *   **栽培:** 完熟させて乾燥豆として収穫します。

*   **Pisello Riccio di Sannicola (サンニコラの縮れエンドウ)** [プーリア州]
    *   **特徴:** 巻きひげ（蔓）が強くカールするのが特徴。
    *   **用途:** 乾燥豆として伝統料理に使用。

### 【番外編・古代の味】
*   **Roveja (ロヴェジャ)** [ウンブリア州]
    *   **特徴:** エンドウの野生種、あるいは原種に近いとされる希少種。豆は赤茶色や灰色などカラフル。
    *   **用途:** 粉に挽いてポレンタにするか、スープに。味はエンドウとレンズマメの中間のようで、野性味があります。
    *   **栽培:** 標高の高い冷涼地でも育ちますが、茎が倒れやすく収穫は手作業必須。自然農法の実践者には非常に面白い品種です。

## 3. 日本で自然栽培・有機栽培する場合のポイント

イタリアの在来種を日本で育てる場合、気候の違い（特に日本の梅雨と湿気）を考慮する必要があります。

### 1. 「窒素」を切り、「ミネラル」を意識する
イタリア品種、特に南部のものは痩せ地に適応しています。日本の肥沃な畑や、動物性堆肥を多用した土では「つるぼけ（葉ばかり茂って実がつかない）」しやすくなります。
*   **対策:** 前作で肥料分を吸わせた後の区画を選ぶか、無施肥でスタートします。代わりに、イタリアの土壌（石灰岩質や火山灰土）に近づけるため、カキ殻石灰や草木灰などでミネラルを補給すると味がのります。

### 2. 「支柱」は高く、風通し良く
「Verdone Nano」などの矮性種以外、イタリアのエンドウは背が高くなります（1.5m〜2m）。
*   **対策:** 日本のキュウリネットなどを使って、しっかりと高さを確保してください。また、春先のうどんこ病や湿気による腐敗を防ぐため、株間は日本の標準よりも広め（30cm以上）に取り、風通しを確保することが成功の鍵です。

### 3. 「完熟」まで待つ勇気
日本のエンドウは若採りが基本ですが、イタリア品種（特にリゾットやスープ用）は、鞘の中の実がパンパンに膨らみ、デンプン質が蓄積されるまで待つことで真価を発揮します。
*   **食べ方:** 収穫時期を少し遅らせて、ホクホクとした食感を楽しんでみてください。採り遅れて硬くなっても、スープにしてしまえば絶品です。

### 4. 自家採種で「和製イタリアン」へ
固定種・在来種の最大のメリットは自家採種ができることです。
*   **適応:** 初年度は日本の気候に合わず、病気が出たり生育が悪かったりするかもしれません。しかし、その中で生き残った株から種を採り続けることで、3〜4年後には「あなたの畑の気候に合ったイタリアエンドウ」へと進化します。

## 栽培データ

果樹ではなく一年草のマメ科植物ですので、種子から容易に増やせます。

*   **収穫期:**
    *   暖地・一般地：4月下旬 〜 6月上旬
    *   寒冷地：6月 〜 7月
    *   ※乾燥豆として利用する場合は、鞘が茶色く枯れるまで待ちます。
*   **播種期（種まき）:**
    *   暖地・一般地：10月中旬 〜 11月中旬（越冬栽培）
    *   寒冷地：3月 〜 4月（春まき）
    *   ※イタリア品種は寒さに強いものが多いですが、幼苗期に厳寒に当たると傷むため、本格的な寒さの前に「本葉2〜3枚」の状態にしておくのがコツです。早まきしすぎて大きくなりすぎると、かえって寒害を受けやすくなります。
*   **発芽適温:** 15℃ 〜 20℃（最低4℃でも発芽はしますが時間がかかります）
*   **播種法:** 点まき（1箇所に3粒ほど）。深さは2〜3cm。
*   **覆土:** 種の厚みの2〜3倍程度（2〜3cm）。マメ科は酸素を好むため、鎮圧しすぎないこと。鳥に食べられやすいため、発芽までは不織布などをかけると安心です。
*   **生育適温:** 15℃ 〜 20℃（冷涼な気候を好みます。25℃を超えると生育が止まりやすくなります）。
*   **栽培法:**
    *   **矮性種（Nano）:** 支柱なし、または短い笹竹などを添える程度で栽培可能。
    *   **高性種（蔓あり）:** 1.5m以上の支柱とネットが必須。
    *   **土寄せ:** 越冬前と春先に株元へ土を寄せると、耐寒性が増し、倒伏を防げます。
*   **採種法:**
    *   最も生育が良く、病気にかかっていない株を選抜します。
    *   食用に収穫せず、鞘が茶色くなりカラカラに乾くまで畑に置いておきます。
    *   天気の良い日に収穫し、さらに数日間天日干しして完全に乾燥させます。
    *   鞘から豆を取り出し、さらに乾燥させます。
*   **種子寿命:** 2 〜 3年（条件が良ければ長持ちしますが、発芽率は徐々に落ちます）。
*   **休眠:** ほとんどありません。
*   **種子保存法:**
    *   よく乾燥させた後、紙袋や密閉容器に入れます。
    *   マメゾウムシの被害を防ぐため、乾燥剤と共に瓶に入れて冷蔵庫の野菜室で保管するのがベストです。
