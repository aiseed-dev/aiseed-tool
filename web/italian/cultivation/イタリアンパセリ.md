# イタリアの魂を育てる：伝統種イタリアンパセリ栽培ガイド

イタリア語で「プレッツェーモロ（Prezzemolo）」と呼ばれるイタリアンパセリ。
日本では単なる「お皿の飾り」として扱われがちですが、イタリア本国では料理の味を決定づける重要な「食材」です。古代ローマから愛され、岩場にルーツを持つこのハーブを、日本の風土でたくましく、香り高く育てるためのポイントを解説します。

## 1. 日本の「パセリ」とイタリアの「プレッツェーモロ」の違い

日本のスーパーで見かける縮れ葉のパセリ（カールパセリ）と、イタリアで愛される平葉種（イタリアンパセリ）は、植物学的には同じ種ですが、その**「在り方」**は全く異なります。

| 特徴 | 日本で一般的な縮れ葉種 (var. *crispum*) | イタリア伝統の平葉種 (var. *neapolitanum*) |
| :--- | :--- | :--- |
| **見た目** | 葉が細かく縮れている。装飾的。 | 葉が平らで切れ込みがある（三つ葉やセロリに似る）。濃緑色。 |
| **食感** | 硬く、ゴワゴワして口に残る。 | 柔らかく、加熱しても食感が馴染む。 |
| **香り・味** | 青臭さが目立ち、香りの広がりは弱い。 | **芳醇で甘みのある強い香り**。精油成分が豊富。 |
| **用途** | 料理の添え物（飾り）、彩り。 | **料理の主役・ベース**。ソース、煮込み、ソテーの味付け。 |

イタリアの料理人にとって、縮れ葉種は香りが弱く硬いため、ほとんど使われません。本場の味を再現するには、平葉種、それもイタリアの風土で選抜された品種を選ぶことがスタートラインです。

## 2. 自然栽培における品種選びと栽培のポイント

日本の自然栽培環境で、イタリアンパセリの野性味と香りを引き出すための鍵は**「品種」**と**「水分管理」**にあります。

### 品種選び：「イタリアの巨人」を探せ
種を選ぶ際は、単に「イタリアンパセリ」と書かれたものではなく、イタリアで圧倒的なシェアを誇る伝統品種**「Gigante d'Italia（ジガンテ・ディ・イタリア／イタリアの巨人）」**の系統を選ぶことを強くおすすめします。

*   **特徴:** 名前通り葉が大きく肉厚で、草丈も高くなります（50〜60cm）。
*   **メリット:** 香りが非常に強く甘みがあり、再生力が旺盛です。日本の気候でも作りやすく、収量が多いのが魅力です。

### 栽培のコツ：岩場の記憶と日本の湿気
学名が「岩場のセロリ」を意味するように、本来は排水性の良い場所を好みますが、乾燥しすぎると葉が硬くなります。

1.  **発芽の忍耐:**
    古くから「悪魔と往復してから発芽する」と言われるほど、発芽には時間がかかります（2週間〜1ヶ月）。土を乾かさないよう、じっくり待つ忍耐が必要です。
2.  **夏場の管理:**
    イタリアンパセリは日当たりを好みますが、日本の真夏の湿気と直射日光は苦手です。夏場は半日陰になる場所や、背の高い野菜の影になる場所（コンパニオンプランツ）での栽培が適しています。
3.  **味のコントロール:**
    水分を控えめに育てると香りと味が凝縮されますが、葉は硬くなります。逆に水分を多くすると葉は柔らかくなりますが、香りは穏やかになります。料理の用途に合わせて水やり（あるいは植え場所）を調整してください。

## 栽培データ

### 収穫期
**春まき：6月〜11月 / 秋まき：翌3月〜6月**
本葉が15枚以上になったら、外側の葉から順次かき取って収穫します。成長点は中心にあるため、中心の若葉を残しておけば長期間収穫し続けられます。

### 播種期
**春まき：3月〜5月 / 秋まき：9月〜10月**
秋まきの方が、翌春にトウ立ち（開花）するまでの期間が長く、長く収穫を楽しめます。

### 発芽適温
**15〜20℃**
高温（25℃以上）では極端に発芽率が落ちるため、夏まきは避けます。

### 播種法
**条まき、またはバラまき**
移植を嫌う（直根性）ため、直播きが理想ですが、育苗する場合は根鉢を崩さないように若苗のうちに定植します。
種皮に発芽抑制物質が含まれているため、**一晩水に浸してから**まくと発芽が揃いやすくなります。「種を揉んで傷をつける」という昔ながらの方法も有効です。

### 覆土
**ごく薄く（種が隠れる程度）**
好光性種子（光を好む）の傾向がありますが、乾燥には極端に弱いため、薄く土をかけ、発芽までは新聞紙や不織布で保湿するのが成功の秘訣です。

### 生育適温
**15〜20℃**
冷涼な気候を好みます。耐寒性は強いですが、霜に当たると葉が傷むため、冬場は寒冷紗やトンネルで保護すると良質な葉が収穫できます。

### 栽培法
株間は20〜30cm確保します。「Gigante d'Italia」のような大型種は株が張るため、広めにとります。
乾燥を防ぐため、株元に草マルチや敷き藁をすると良いでしょう。トウ立ち（花芽がついた茎が伸びること）すると葉が硬くなり風味が落ちるため、種を採らない場合は早めに花茎を摘み取ります。

### 採種法
イタリアンパセリは二年草です。秋にまいた株を冬越しさせ、翌年の初夏に開花させます（セリ科特有の傘のような花が咲きます）。
花が終わり、種が茶色く完熟して乾燥してくるのを待ちます。熟すと自然に落ちやすくなるので、花茎ごと刈り取り、袋の中で叩くなどして種を落とします。
※他のセリ科植物と交雑しやすいため、近くに人参などを咲かせないよう注意してください。

### 種子寿命
**短命（1〜2年）**
発芽率が落ちやすいので、できるだけ新しい種を使うか、毎年採種することをおすすめします。

### 休眠
深い休眠はありませんが、発芽抑制物質の影響で発芽には時間がかかります。

### 種子保存法
よく乾燥させた後、紙袋や密閉容器に入れ、冷蔵庫などの低温低湿な場所で保管します。
