Cloudflare設定ガイド

あなたの農園ホームページを Cloudflare Pages で公開する手順

はじめに

Growアプリの「ホームページ作成」機能を使うと、あなたの農園・作物の紹介サイトを生成できます。 生成したサイトは あなた自身のCloudflareアカウント にホスティングされるため、 独自のURLを持ち、完全にあなたの管理下に置かれます。

2つの公開方法があります
A. HTMLをコピーして手動アップロード（かんたん）
B. APIトークンで自動デプロイ（上級者向け）

方法A: 手動アップロード（かんたん）

1 Cloudflareアカウントを作成

Cloudflare にアクセスして無料アカウントを作成します。 メールアドレスとパスワードだけで登録できます。

2 GrowアプリでHTMLを生成

Growアプリの作物詳細画面から「ホームページを作成」をタップし、 農園情報・掲載する作物・販売情報を入力して「HTMLを生成」を押します。 生成されたHTMLを「HTMLをコピー」ボタンでコピーしてください。

3 HTMLファイルを保存

PCでテキストエディタを開き、コピーしたHTMLを貼り付けて index.html というファイル名で保存します。

4 Cloudflare Pagesにアップロード

  1. Cloudflareダッシュボードにログイン
  2. 左メニューから Workers & Pages を選択
  3. 「ページを作成」→「直接アップロード」を選択
  4. プロジェクト名を入力（例: watashi-no-nouen）
  5. index.html をドラッグ＆ドロップでアップロード
  6. 「サイトを公開」をクリック

5 公開完了

数秒でサイトが公開されます。URLは以下の形式になります：

https://watashi-no-nouen.pages.dev

このURLを共有すれば、誰でもあなたの農園ページを見ることができます。

方法B: アプリから自動デプロイ（上級者向け）

Growアプリにあなたの Cloudflare APIトークンを設定すると、 アプリのボタンひとつでサイトの公開・更新ができます。

必要な情報

項目取得場所例
Account ID ダッシュボード右サイドバー a1b2c3d4e5f6...
API Token プロフィール → APIトークン → 作成 xxxx-yyyy-zzzz
プロジェクト名 自分で決める（URLの一部になる） my-farm

1 Account ID を確認

  1. Cloudflareダッシュボード にログイン
  2. 左メニューの Workers & Pages を選択
  3. ページ右サイドバーに Account ID が表示されています
  4. この文字列をコピーしてください

2 APIトークンを作成

  1. ダッシュボード右上のアイコン → マイプロフィール
  2. 左メニューから APIトークン を選択
  3. 「トークンを作成する」をクリック
  4. 「Cloudflare Pages を編集する」テンプレートを選択
  5. アカウントリソースで自分のアカウントを選択
  6. 「概要に進む」→「トークンを作成する」
重要: 作成されたトークンは一度しか表示されません。 必ずコピーして安全な場所に保存してください。

3 Growアプリに設定

  1. Growアプリの作物詳細画面 →「ホームページを作成」
  2. 画面下部の「Cloudflareに自動デプロイ」セクションまでスクロール
  3. Account ID、APIトークン、プロジェクト名を入力
  4. 「デプロイ」ボタンをタップ

数秒でサイトが公開されます。更新時は同じボタンを押すだけです。

独自ドメインの設定（任意）

Cloudflare Pages では、独自ドメインを設定することもできます。 例えば www.watashi-no-nouen.com のようなURLで公開できます。

  1. Cloudflareダッシュボードで対象のPagesプロジェクトを開く
  2. 「カスタムドメイン」タブを選択
  3. 「カスタムドメインを設定」をクリック
  4. ドメイン名を入力して設定完了
独自ドメインを使うにはドメインの購入が別途必要です。 Cloudflareでもドメインを購入できます（年間約1,000〜2,000円程度）。

料金について

項目無料プラン
サイト数無制限
月間リクエスト無制限
帯域無制限
ビルド回数500回/月
カスタムドメイン対応（ドメイン代のみ別途）
SSL証明書自動・無料

通常の農園サイトであれば、完全無料 で運用できます。

よくある質問

Q. スマホだけで公開できますか？

方法B（自動デプロイ）を使えば、スマホのGrowアプリだけで公開できます。 方法A（手動アップロード）の場合は、PCでの作業が必要です。

Q. サイトを更新したいときは？

Growアプリで内容を変更して再度「デプロイ」するだけです。 手動の場合は新しいHTMLをアップロードしてください。

Q. 写真はどこに保存されますか？

同期済みの写真はGrowサーバー（Cloudflare R2）に保存されています。 生成されたサイトからはこれらの写真が参照されます。 写真を表示するには、先にGrowアプリの「データ同期」で写真をサーバーに同期してください。

Q. APIトークンは安全ですか？

APIトークンはGrowアプリのローカルストレージにのみ保存され、 デプロイ時にのみサーバー経由でCloudflareに送信されます。 不安な場合は方法A（手動アップロード）をご利用ください。
