カッペリ料理ガイド

～栽培者が楽しむ、本場の味と保存の知恵～

1. 日本とイタリア：使い方の決定的な違い

日本の家庭料理とイタリアの伝統料理における最大の違いは、「塩漬け」の使い方と「加熱」にあります。

  • 日本（一般的）： 酢漬け（ピクルス）をそのままサラダやサーモンに散らす。「酸味」のアクセントとして使用。
  • イタリア（伝統的）： 塩漬け（Sotto sale）が基本。塩抜きをしてから、刻んでソースに溶かし込んだり、油で炒めて香りを引き出したりします。カッペリは「具」であると同時に、昆布や鰹節のような「出汁（だし）」の役割を果たします。

2. 品種と産地で使い分ける：パンテッレリア vs エオリエ

イタリアの二大産地、パンテッレリア島とエオリエ諸島。それぞれの特徴に合わせた使い分けを知ると、料理の完成度が一段と上がります。

3. まずはここから：「塩抜き」の作法

自家製またはイタリア産の「塩漬けカッペリ」を使う場合、この工程が味を左右します。

  1. 洗い流す: 表面の塩を水でさっと洗い流します。
  2. 浸す: ボウルに水を張り、カッペリを入れて10分〜20分ほど浸します（古漬けの場合はぬるま湯で何度か水を変えます）。
  3. 絞る: 手で軽く水気を絞ります。

栽培者のポイント: 完全に塩を抜き切らず、「薄い塩味が残る程度」で止めるのがコツです。この残った塩気が、料理の調味料代わりになります。

4. イタリア各地の郷土料理とレシピ

カッペリの個性を活かした、イタリアの代表的な郷土料理をご紹介します。

【シチリア州】インサラータ・パンテスカ (Insalata Pantesca)

パンテッレリア島の農民料理。素材の味がダイレクトに伝わる、栽培者にこそ食べてほしい一皿です。

  • 主な材料: 茹でたジャガイモ、トマト、赤玉ねぎ、オリーブ、塩漬けカッペリ（パンテッレリア産推奨）、オレガノ、オリーブオイル。
  • 作り方:
    1. ジャガイモは茹でて一口大に、トマトと玉ねぎはスライスします。
    2. ボウルに全ての野菜と塩抜きしたカッペリを入れます。
    3. たっぷりのオリーブオイルとオレガノ（乾燥または生）で和えます。
    4. 冷蔵庫で少し冷やし、味が馴染んでから食べます。
  • 味の秘訣: カッペリの塩気と旨味がジャガイモに染み込み、素朴ながら驚くほど豊かな味になります。

【カンパニア州】スパゲッティ・アッラ・プッタネスカ (Spaghetti alla Puttanesca)

「娼婦風スパゲッティ」として有名ですが、本質は「保存食だけで作れる旨味の爆弾」です。

  • 主な材料: トマト缶、アンチョビ、黒オリーブ、カッペリ、ニンニク、唐辛子。
  • 作り方:
    1. オリーブオイルでニンニクと唐辛子、アンチョビを熱します。
    2. ここでカッペリを加えます。 軽く炒めることで香りが油に移ります。
    3. トマトを加え、ソースを煮詰めます。
    4. 茹でたパスタを和え、仕上げに生のパセリを散らします。

【ピエモンテ州】サルサ・ヴェルデ (Salsa Verde)

北イタリアの伝統的な「緑のソース」。茹でた肉や魚、野菜スティックに最高です。

  • 作り方: イタリアンパセリ（大量）、カッペリ（多め）、アンチョビ、ゆで卵の黄身、パン粉（酢に浸したもの）、オリーブオイルをミキサーにかけるか、包丁で細かく叩きます。
  • 活用: 収穫した野菜のディップソースとして、あるいはBBQのソースとして万能です。

5. 栽培者ならではの楽しみ：収穫と保存の伝統技法

解説記事にあるように、カッペリは「蕾（カッペリ）」と「果実（ククンチ）」の両方を楽しめます。自分で育てたからこそできる、伝統的な保存方法をご紹介します。

A. 塩漬け発酵 (Sotto Sale) - パンテッレリア方式

最も香りが良く、長期保存に適した伝統製法です。

  1. 収穫: 蕾が硬く閉じた状態で収穫し、涼しい場所で数時間広げて熱をとります（萎れさせないよう注意）。
  2. 塩漬け: カッペリの重量の40%の海塩（粗塩）を用意します。
  3. 発酵: 容器（陶器やガラス）にカッペリと塩を混ぜて入れます。最初の数日は水分が出るので、毎日かき混ぜます。
  4. 熟成: 約10日後、出てきた水分（塩水）を捨て、さらに20%程度の新しい塩を加えます。
  5. 完成: そのまま冷暗所で3〜4ヶ月熟成させます。乳酸発酵により、独特のフローラルな香りが生まれます。

B. 酢漬け (Sotto Aceto) - エオリエ風ククンチ

実（ククンチ）や、少し開いてしまった蕾におすすめの方法です。

  1. 下処理: 収穫した実を、塩水（水1Lに対し塩100g程度）に2〜3日漬け、苦味を抜きます。
  2. 乾燥: 水から上げ、風通しの良い場所で数時間乾かします。
  3. 漬け込み: 煮沸消毒した瓶に詰め、良質の白ワインビネガーを注ぎます。お好みで月桂樹の葉や胡椒の粒を入れます。
  4. 食べ頃: 1ヶ月後から食べられます。カリッとした食感が残り、アペリティーボに最適です。

C. オイル漬け (Sott'olio)

塩漬けにしたものを塩抜きし、水分をよく拭き取ってからオリーブオイルに漬け込みます。
※生のままオイルに漬けるとボツリヌス菌のリスクがあるため、必ず「塩漬け→塩抜き→酢洗い→乾燥→オイル」または「酢漬け→オイル」の手順を踏んでください。

おわりに

イタリアの諺に「カッペリのない料理は、太陽のない夏のようなもの」という言葉はありませんが、そう言いたくなるほど、彼らの食卓には欠かせない存在です。

ご自身の畑で育ったカッペリを、パンテッレリアの農民のように塩で発酵させ、エオリエの漁師のようにパスタに絡める。その一粒には、地中海の歴史と、あなたの畑のテロワール（風土）が凝縮されています。ぜひ、本場の食べ方でその力強い生命力を味わってください。
