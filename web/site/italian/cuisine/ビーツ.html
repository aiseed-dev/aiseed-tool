ビーツ料理ガイド

〜葉も根も、余すことなく大地の恵みを〜

1. イタリア流と日本流：調理アプローチの違い

日本の家庭では「皮をむいて茹でる」ことが多いですが、イタリアの伝統的なアプローチは異なります。

  • 茹でずに「焼く」が基本
    イタリア、特に南部では伝統的に「竈（かまど）の灰の中（Sotto la cenere）」でじっくり焼く調理法が愛されてきました。水で茹でると水溶性のビタミンや糖分が流れ出してしまいますが、皮付きのままオーブンでローストすることで、甘みと風味を内部に凝縮させます。
  • 「葉」は捨てない（主役級の野菜）
    歴史的にビーツは「葉を食べる野菜」でした。若い葉はサラダに、大きく育った葉や茎は「ビエートラ（フダンソウ）」として、ほうれん草以上に重宝されます。
  • 「酸味」と「ハーブ」で土の香りを手なずける
    ビーツ特有の土の香り（ゲオスミン）は、酢（ビネガー）やミント、ガーリックと合わせることで、不快な臭みから「芳醇な風味」へと変わります。

2. 品種で使い分ける：キオッジャと赤ビーツ

お手元のビーツはどちらのタイプでしょうか？ 品種に合わせた調理で、そのポテンシャルを最大限に引き出せます。

🔴 赤ビーツ（一般的なビーツ）

  • 特徴: 濃厚な甘みと強い赤色。加熱するとねっとりとした食感になる。
  • おすすめ料理: ロースト、ニョッキ、リゾット、煮込み。
  • 栽培者の特権: 収穫したての若い根は、皮が薄く香りもフレッシュなので、薄くスライスして生でも楽しめます。

🎯 キオッジャ・ビーツ（Barbabietola di Chioggia）

  • 特徴: ヴェネト州キオッジャ発祥の伝統品種（P.A.T.認定）。切ると赤と白の美しい年輪模様が現れます。赤ビーツより土臭さが少なく、マイルドで甘みが強いのが特徴。
  • 注意点: 加熱すると縞模様が消えて全体が薄ピンク色になります。 模様を活かしたい場合は「生」または「ごく短時間の加熱」が鉄則です。
  • おすすめ料理: カルパッチョ（生）、サラダ、軽いピクルス。

3. イタリア各地の郷土料理レシピ

その土地の風土に根ざした食べ方は、素材の味を活かすヒントに満ちています。

【北部・ドロミテ地方】カスンツィエイ (Casunziei)

〜ビーツの甘みとポピーシードの香ばしさ〜
北イタリアの山岳地帯で愛される、ビーツを詰めたラビオリです。

  • 作り方:
    1. ローストして柔らかくした赤ビーツを裏ごしし、少量のジャガイモと合わせてペーストにします。
    2. パスタ生地（または餃子の皮で代用可）で包み、半月型にします。
    3. 茹で上げたものに、溶かしバターとポピーシード（ケシの実）をたっぷりかけます。
  • 味わい: バターのコクがビーツの甘みを引き立て、まるでデザートのようなリッチな味わいです。

【南部・プーリア州】バルバビエートラ・ア・サルサ (Barbabietola a salsa)

〜暑さを吹き飛ばす、ミントと酢の清涼感〜
南イタリア、サレント地方の夏の定番。土の香りとミントの相性が抜群です。

  • 作り方:
    1. ビーツは皮付きのまま柔らかくなるまで加熱し、皮をむいてスライスします。
    2. ボウルに入れ、刻んだ生ニンニク、フレッシュミント、オリーブオイル、塩、そしてワインビネガーで和えます。
    3. 冷蔵庫で数時間冷やし、味を馴染ませます。
  • 味わい: ビネガーとミントがビーツのクセを爽やかに中和し、いくらでも食べられるサラダになります。

【全土・冬の定番】ビーツのオーブン焼き (Al Forno)

〜最もシンプルで、最も甘い食べ方〜

  • 作り方:
    1. ビーツを皮付きのままよく洗い、一つずつアルミホイルで包みます（塩とオリーブオイルを少々振っても良い）。
    2. 180〜200℃のオーブンで40分〜1時間、竹串がスッと通るまで焼きます。
    3. 粗熱が取れたら、手でつるりと皮をむき、塩と極上のオリーブオイルだけでいただきます。

4. 葉と茎（Bietola）を楽しむレシピ

自然栽培農家だからこそ手に入る、新鮮な葉と茎。捨てずに美味しくいただきましょう。

葉のソテー「リパッサータ」 (Bietola ripassata)

ローマなどで定番の、一度茹でてから炒める調理法です。

  1. 葉と茎をざく切りにし、さっと塩茹でして水気をよく絞ります。
  2. フライパンにオリーブオイル、潰したニンニク、唐辛子（お好みで）を入れて熱し、香りを立たせます。
  3. 茹でた葉を加え、オイルを吸わせるように強火で炒め合わせます。
  4. ポイント: 茎のシャキシャキ感と葉の旨味が、肉料理の付け合わせやパスタの具に最適です。

5. 保存食の伝統技法 (Conservazione)

たくさん収穫できた時は、イタリアのマンマの知恵で保存食にしましょう。

ビーツのソットアチェト（酢漬け）

冬の間のビタミン源として作られてきた保存法です。

  1. ビーツを丸ごと茹でるかローストして皮をむき、一口大やスティック状にカットします。
  2. 鍋に白ワインビネガー、水（2:1の割合）、砂糖少々、塩、粒胡椒、ローリエを入れて沸騰させます。
  3. 煮沸消毒した瓶にビーツを詰め、熱いピクルス液を注ぎます。
  4. 活用: そのままアンティパスト（前菜）にするほか、刻んでポテトサラダに入れたり、肉料理の口直しに添えます。

セミドライ・ビーツ（Chips）

天日干しやオーブンを使った保存法です。

  1. 生のビーツをスライサーで極薄にスライスします。
  2. 天日で干すか、低温のオーブン（100〜120℃）でカリッとするまで乾燥させます。
  3. 活用: 自然な甘みが凝縮されたチップスは、お子様のおやつや、ワインのおつまみに最適です。

最後に

イタリアのことわざに「自然は急がない（La natura non ha fretta）」という言葉があります。じっくりと時間をかけて育ったビーツは、じっくりと火を通すことでその真価を発揮します。

土作りからこだわった皆様のビーツが、イタリアの知恵と出会い、食卓を鮮やかに彩ることを願っています。ぜひ、赤と白のコントラストや、大地の香りを楽しんでください。
