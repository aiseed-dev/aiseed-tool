コールラビ料理ガイド

1. 日本の食べ方 vs イタリアの伝統

日本の家庭では、コールラビを「カブや大根の代用」として扱いがちです。しかしイタリア、特に南部では「キャベツの甘み」と「カブの食感」を併せ持つ独特の野菜として扱います。

大きな違いは以下の2点です。

  1. 葉と脇芽（フィッギウレ）を捨てない:
    日本のスーパーでは葉が落とされた状態で売られていますが、栽培者なら新鮮な葉が手に入ります。イタリア南部では、葉は茎以上に旨味が詰まった「ご馳走」です。
  2. 「茹でる」より「蒸し煮（Stufato）」:
    お湯で茹でこぼすと甘みが逃げます。イタリアでは少量の水分とオリーブオイルで蒸し煮にし、野菜自身の水分で甘みを凝縮させる調理法が好まれます。

2. 品種別：料理への適性マップ

育てている品種に合わせて調理法を変えるのが、イタリアのマンマ（お母さん）の知恵です。

品種タイプ 特徴 おすすめの調理法
Trunzo di Aci 系
(紫皮・小型)		 繊維が柔らかく、甘みが非常に強い。 【生食・パスタ】
まずは生でサラダに。加熱するとトロリとするので、トマト煮込みやパスタソースに最適。
Chepe de Murte 系
(白/緑皮・中〜大型)		 肉質がしっかりしており、煮崩れしにくい。 【煮込み・付け合わせ】
ジャガイモの代わりにスープに入れたり、クタクタになるまで蒸し煮にして肉料理の付け合わせに。
Grucciolo 系
(野生的・脇芽が多い)		 風味が強く、野趣あふれる味。 【オイル蒸し・葉料理】
脇芽（Figghiule）と葉を主役に、ニンニクと唐辛子で炒め煮にするプーリア風の食べ方がベスト。

3. 産地を旅する郷土料理レシピ

【シチリア風】トロンゾのパスタ (Pasta co' Trunzu)

対象：紫色の品種（Trunzoなど）、または甘みの強い品種

アチレアーレ地方の看板料理です。紫色の皮の色素が溶け出し、パスタ全体がほんのり色づくのが正解です。

  • 材料: コールラビ（茎と葉）、トマト（ホール缶または生）、ニンニク、オリーブオイル、パスタ（ショートパスタ推奨）、塩、ペコリーノチーズ（あれば）。
  • 作り方:
    1. コールラビの茎は皮を剥かずに（硬い部分だけ取り除く）サイコロ状に切り、葉はざく切りにします。
    2. フライパンに潰したニンニクと多めのオリーブオイルを熱し、香りが立ったらコールラビの茎と葉を入れて炒めます。
    3. トマトを加え、蓋をして弱火で「蒸し煮」にします。水は加えず、野菜の水分だけで煮込むのがコツです。
    4. 別の鍋でパスタを茹でます。
    5. パスタが茹で上がる直前にソースと合わせ、茹で汁で濃度を調整します。
    6. 皿に盛り、チーズをかけて完成。

【プーリア風】カボロ・ラパと脇芽のビアンコ (Orecchiette con Cime di Rapa style)

対象：脇芽の多い品種（Gruccioloなど）、または葉が元気なもの

トマトを使わない「ビアンコ（白）」仕立て。素材の味がダイレクトに伝わります。

  • 材料: コールラビ（特に葉と脇芽）、ニンニク、鷹の爪、アンチョビ（数枚）、オレキエッテ（または耳たぶ型のパスタ）、最高級のオリーブオイル。
  • 作り方:
    1. 茎は薄切り、葉と脇芽は食べやすい大きさに切ります。
    2. パスタを茹でるお湯に塩を入れ、沸騰したらパスタとコールラビ（茎・葉すべて）を一緒に鍋に入れて茹でます。これがプーリア流です。
    3. フライパンにオリーブオイル、ニンニク、鷹の爪、アンチョビを入れて弱火でじっくり熱し、アンチョビを溶かします。
    4. 茹で上がったパスタと野菜を湯切りしてフライパンに移し、オイルとよく絡めます（マンテカーレ）。
    5. 仕上げに、さらに新鮮なオリーブオイルを回しかけます。

【ルネサンス風】古典的ピンツィモニオ (Pinzimonio)

対象：採れたての全品種

1554年にマッティオリが記述した、最も古く、最も贅沢な食べ方です。

  • 作り方:
    1. 収穫したてのコールラビの皮を厚めに剥き、スティック状にスライスします。
    2. 小皿に以下の調味料を合わせます。
      • 極上のオリーブオイル
      • 粗塩
      • 挽きたての黒胡椒
      • レモン汁（数滴）
    3. これに浸して、コリコリとかじります。レモンの酸味が鉄分の吸収を助け、栽培の疲れを癒やしてくれます。

4. 栽培者の特権：保存食の知恵

収穫期にたくさん採れすぎてしまった場合は、イタリアの伝統的な保存法「ソットーリオ（Sott'olio）」で瓶詰めにしましょう。冬の間の貴重なアンティパスト（前菜）になります。

コールラビのオイル漬け (Cavolo Rapa Sott'olio)

  1. 下準備: コールラビの皮を剥き、拍子木切りにします。
  2. 茹でる: 鍋に水と白ワインビネガーを「2:1」の割合で入れ、塩を加えて沸騰させます。コールラビを入れ、3〜4分（歯ごたえが残る程度）茹でます。
  3. 乾かす: 清潔な布巾の上に広げ、数時間〜半日置いて表面の水分を完全に飛ばします（重要！水分が残るとカビの原因になります）。
  4. 詰める: 煮沸消毒した瓶に、コールラビ、ニンニク片、唐辛子、オレガノやミントなどのハーブを詰めます。
  5. 注ぐ: 具材が完全に隠れるまでオリーブオイルを注ぎます。気泡が入らないように竹串などでつつき、蓋をして冷暗所で保存します。
    • 食べ頃：1ヶ月後から。

最後に：栽培者へのアドバイス

イタリアの伝統品種（特にTrunzoやGrucciolo）を育てている方は、「不揃いであること」を誇りに思ってください。
記事にもある通り、均一なF1品種よりも成長が遅かったり、形がいびつだったりするかもしれません。しかし、その「野性味」こそが、イタリア料理において最も重要な調味料となります。

  • 収穫のタイミング: 肥大茎がテニスボール大（約6-8cm）になった頃がベストです。大きすぎると木質化（スが入る）して食味が落ちます。
  • 水やり: 収穫前の数週間、水を控えめにすると（イタリア南部の環境を模倣）、糖度がグッと増します。

ぜひ今夜は、畑から採ってきたばかりのコールラビの葉も茎も丸ごと使って、イタリアの風を感じてみてください。ボナペティート！
