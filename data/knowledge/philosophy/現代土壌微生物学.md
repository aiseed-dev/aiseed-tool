# 現代土壌微生物学 ― 自然農法を裏付ける科学

## 概要

土壌微生物学は、21世紀に入り飛躍的な発展を遂げ、福岡正信が直観と観察で到達した自然農法の原理を科学的に裏付けている。本資料では、土壌微生物の世界、特に菌根菌ネットワーク、炭素隔離メカニズム、土壌の健康と農業生産性の関係について、最新の研究成果をまとめる。

---

## 1. 土壌微生物の世界

### 1.1 土壌は「生きている」

土壌1グラムには、10億個以上の細菌、数十万個の真菌（カビ・キノコ類）の菌糸、数万個の原生動物が生息している。これらの微生物は、有機物の分解、養分の循環、植物への栄養供給、病原菌の抑制など、生態系の基盤となる機能を担っている。

### 1.2 土壌マイクロバイオーム

人間の腸内細菌叢（腸内フローラ）が健康に重要であるように、土壌の微生物群集（土壌マイクロバイオーム）は植物の健康と生産性に決定的な影響を与える。

| 微生物グループ | 主な機能 |
|---------------|---------|
| **細菌** | 有機物分解、窒素固定、養分循環 |
| **真菌（糸状菌）** | 難分解性有機物の分解、養分輸送、土壌構造形成 |
| **菌根菌** | 植物との共生、リン酸吸収促進、水分供給 |
| **放線菌** | 抗生物質産生、病原菌抑制 |
| **原生動物** | 細菌の捕食による養分放出 |

## 2. アーバスキュラー菌根菌（AM菌）

### 2.1 AM菌とは

アーバスキュラー菌根菌（Arbuscular Mycorrhizal Fungi, AM菌）は、陸上植物の80%以上と共生できる糸状菌である。植物の根の内部に侵入し、「樹枝状体（arbuscule）」と呼ばれる特徴的な構造を形成して、植物と養分を交換する。

**共生のメカニズム：**

1. 植物はリン酸が不足すると「ストリゴラクトン」という物質を分泌
2. AM菌がストリゴラクトンを認知し、「リポキチンオリゴサッカライド」を分泌
3. お互いに存在を認識し合い、共生関係が成立
4. AM菌は土壌から吸収したリン酸や水を植物に供給
5. 植物は光合成産物（糖分・脂肪酸）をAM菌に供給

### 2.2 AM菌の農業的意義

AM菌との共生は、以下の農業的メリットをもたらす：

- **リン酸吸収の促進**：AM菌の菌糸は根毛より細く（直径2-5μm）、土壌中を広範囲に伸び、植物だけでは届かないリン酸を吸収
- **減肥効果**：宿主植物を栽培した跡地では、リン酸を半分以下に減肥しても収量がほとんど変わらない
- **乾燥耐性の向上**：菌糸ネットワークを通じた水分供給
- **病害抵抗性の向上**：根圏の生物相バランス改善

### 2.3 最新研究動向

#### 純粋培養の成功（2019年）
大阪府立大学などの共同研究チームが、AM菌の純粋培養に世界で初めて成功（Nature Microbiology掲載）。これにより、AM菌を大量生産し、微生物資材として農業に本格活用できる可能性が開かれた。

#### ゲノム解読（2018年）
基礎生物学研究所の研究グループが、代表的なAM菌（*Rhizophagus irregularis*）のゲノムを高精度に解読。AM菌が脂肪酸やビタミンB1の合成遺伝子を欠損しており、これらを宿主植物から得ていることが明確化された。

## 3. グロマリンと土壌団粒構造

### 3.1 グロマリンとは

グロマリン（Glomalin）は、AM菌が分泌する糖タンパク質で、1996年に発見された。土壌科学における最も重要な発見の一つとされる。

**グロマリンの特性：**

- 土壌粒子を接着させる「天然の接着剤」として機能
- 土壌中で7〜42年間安定して存在
- 土壌有機炭素の15〜30%を占める場合もある
- 疎水性があり、土壌の保水力を向上

### 3.2 団粒構造の重要性

グロマリンによって形成される土壌団粒構造は、以下の機能を持つ：

| 機能 | 説明 |
|------|------|
| **通気性** | 団粒間の隙間が空気の通り道となる |
| **排水性** | 過剰な水分が排出されやすい |
| **保水性** | 団粒内部に水分を保持 |
| **根の伸長** | 根が伸びやすい環境を提供 |
| **微生物の住処** | 団粒内部が微生物の生息場所となる |

### 3.3 耕起が破壊するもの

福岡正信が「不耕起」を主張した科学的根拠がここにある：

- **物理的破壊**：耕起は団粒構造を粉砕し、土壌の物理性を悪化させる
- **菌糸ネットワークの切断**：AM菌の菌糸は耕起により寸断され、再構築に時間がかかる
- **炭素の放出**：団粒が壊れると、内部に保護されていた有機炭素が酸化してCO2として大気中に放出

## 4. 菌根菌ネットワーク（Wood Wide Web）

### 4.1 地下のインターネット

菌根菌は、複数の植物の根を菌糸で連結し、巨大なネットワークを形成する。このネットワークは「Wood Wide Web（森のインターネット）」と呼ばれ、森林の木の80%以上が接続されていると推定される。

### 4.2 ネットワークの機能

#### 養分・水分の再分配
菌根菌ネットワークを通じて、余裕のある植物から不足している植物へ養分や水分が移動する。2025年の研究では、ネットワークに接続された植物は、乾燥ストレス下での死亡率が20%低下し、光合成活動を長期間維持できることが示された。

#### シグナル伝達
菌糸を通じた電気信号は、1秒あたり約1mmの速度で伝達される。これは生体組織としては驚くほど速く、一種の「原始的な神経系」として機能している可能性がある。

#### 化学物質による情報伝達
害虫に攻撃された植物は、菌根菌ネットワークを通じて周囲の植物に「警告」を発し、防御物質の生産を促すことが確認されている。

### 4.3 2025年の新発見

従来の研究はAM菌や外生菌根菌に集中していたが、2025年の*Communications Biology*の研究で、Dark Septate Endophyte（DSE）と呼ばれる非菌根菌も植物間ネットワークを形成し、水を移動させることが発見された。菌根菌ネットワークは、これまで考えられていたよりもはるかに複雑で広範であることが明らかになりつつある。

## 5. 根圏浸出物と炭素隔離

### 5.1 液体炭素経路（Liquid Carbon Pathway）

植物は光合成で固定した炭素の20〜40%を、根から「根圏浸出物（Root Exudates）」として土壌中に放出する。これは「液体炭素経路」と呼ばれ、土壌炭素蓄積の主要なメカニズムである。

**根圏浸出物の成分：**
- 糖類（グルコース、フルクトースなど）
- アミノ酸
- 有機酸（クエン酸、リンゴ酸など）
- 二次代謝産物

### 5.2 炭素隔離のメカニズム

根圏浸出物は、以下の経路で土壌有機炭素（SOC）として固定される：

1. **直接経路**：浸出物が土壌鉱物と結合し、MAOC（鉱物結合有機炭素）を形成
2. **間接経路**：微生物が浸出物を代謝し、微生物バイオマスとして蓄積後、死んでMAOCとなる

### 5.3 微生物炭素利用効率（CUE）

2025年の最新研究では、**微生物炭素利用効率（Carbon Use Efficiency, CUE）**が土壌炭素蓄積の鍵であることが明らかになっている。

- **CUEとは**：微生物が取り込んだ炭素のうち、呼吸でCO2として放出される割合と、バイオマスとして蓄積される割合の比率
- **高いCUE**：微生物が効率的に炭素をバイオマスに変換 → 土壌有機物が増加
- **低いCUE**：微生物が炭素を呼吸で放出 → 土壌有機物が減少

**CUEを高める要因：**
- 多様な植物種による多様な根圏浸出物
- 不耕起による微生物群集の安定化
- 適切な炭素・窒素比

### 5.4 土壌タイプによる違い（2025年研究）

2025年の研究では、根からのカルボン酸浸出が土壌タイプによって異なる効果をもたらすことが発見された：

| 土壌タイプ | メカニズム | 結果 |
|-----------|-----------|------|
| **アルカリ性土壌**（乾燥・半乾燥地域） | カルボン酸がCa結合リンを溶解 | 正味の炭素隔離 |
| **酸性土壌** | 有機リンの鉱化に伴うCO2放出 | 炭素損失の可能性 |

## 6. 自然農法との関連

### 6.1 福岡正信の直観と現代科学

福岡正信が提唱した自然農法の四大原則は、現代の土壌微生物学によって科学的に裏付けられている：

| 福岡の原則 | 土壌微生物学的裏付け |
|-----------|---------------------|
| **不耕起** | AM菌ネットワークの保全、団粒構造の維持、土壌炭素の保持 |
| **無肥料** | AM菌によるリン酸供給、根圏浸出物を介した養分循環 |
| **無除草（クローバー）** | マメ科植物の窒素固定、多様な根圏浸出物による微生物活性化 |
| **無農薬** | 土壌微生物の多様性維持、天敵微生物による病害抑制 |

### 6.2 「何もしない」ことの科学

福岡の「何もしない農法」は、実際には「微生物に仕事をさせる農法」と言い換えることができる。人間が機械や化学物質で行おうとしていた仕事（耕耘、施肥、防除）を、土壌微生物が無償で行ってくれるシステムを構築するものである。

## 7. 気候変動対策としての土壌微生物

### 7.1 土壌の炭素貯留ポテンシャル

土壌は、大気中のCO2の2倍以上の炭素を貯蔵している。リジェネラティブ農業や自然農法の普及により、土壌の炭素貯留量を増加させることは、気候変動対策として極めて有効である。

### 7.2 2025年の戦略的アプローチ

2025年の*Cell*誌の論文では、「気候スマート作物（Climate-Smart Crops）」の実現に向けて、以下の統合的アプローチが提案されている：

1. **光合成同化の改善**：より多くのCO2を固定
2. **作物残渣と根圏浸出物の質の向上**：微生物が利用しやすい炭素供給
3. **地上部と地下部の炭素配分の最適化**：より多くの炭素を根に送る
4. **微生物炭素利用効率の向上**：土壌への炭素蓄積を最大化

### 7.3 将来の技術展望

- **植物育種とマイクロバイオーム工学の統合**：炭素隔離を促進する根圏浸出物を産生する作物品種の開発
- **土壌微生物のCRISPR編集**：CUEを高める遺伝子改変（研究段階）
- **ターゲット微生物の接種**：炭素固定能力の高い微生物群の意図的導入

## 8. 農業応用の現状

### 8.1 微生物資材（バイオスティミュラント）

AM菌や有用微生物を含む資材が、農業現場で使用され始めている：

| 資材タイプ | 効果 | 課題 |
|-----------|------|------|
| **AM菌接種剤** | リン酸吸収促進、乾燥耐性向上 | 土着菌との競合、効果の不安定性 |
| **根圏細菌** | 窒素固定、成長促進 | 土壌環境による効果変動 |
| **バイオ炭** | 微生物の住処提供、炭素固定 | 品質のばらつき |

### 8.2 前作効果の活用

農研機構の研究によると、AM菌の宿主となる作物（トウモロコシ、ダイズなど）を前作として栽培した跡地では、後作の初期生育が促進され、リン酸施肥を大幅に削減できる。これは既存の輪作体系に組み込める実践的な手法である。

### 8.3 不耕起栽培の普及

日本ではまだ普及が限られているが、世界的には不耕起栽培が急速に拡大している。米国では農地の約40%、ブラジルでは約50%で不耕起またはMinimum Tillage（最小耕起）が採用されている。

## 9. 土壌診断の革新

### 9.1 従来の化学分析の限界

従来の土壌診断は、pH、EC（電気伝導度）、窒素・リン酸・カリウム含量など、化学的指標に偏っていた。しかし、これらは土壌の「生物学的健康」を反映していない。

### 9.2 生物学的土壌診断

新たな土壌診断手法が開発されつつある：

| 手法 | 測定対象 | 意義 |
|------|---------|------|
| **ヘイニーテスト** | 土壌呼吸量、水溶性有機炭素・窒素 | 微生物活性と養分供給力の評価 |
| **PLFA分析** | リン脂質脂肪酸組成 | 微生物群集構造の評価 |
| **メタゲノム解析** | 土壌DNA | 微生物種多様性の網羅的評価 |
| **グロマリン定量** | GRSP（グロマリン関連土壌タンパク） | AM菌活性と団粒構造の評価 |

### 9.3 理化学研究所の取り組み

理化学研究所バイオリソース研究センターでは、生物化学的要素である微生物を加えた新しい土壌診断技術の開発を目指している。これにより、「肥料をどれだけ入れるか」ではなく、「微生物がどれだけ働いているか」を基準とした農業管理が可能になる。

## 10. まとめ：土壌微生物学が示す農業の未来

### 10.1 パラダイムシフト

現代土壌微生物学は、農業に対する根本的なパラダイムシフトを促している：

| 従来の農業観 | 新しい農業観 |
|-------------|-------------|
| 土壌は作物を支える培地 | 土壌は生きた生態系 |
| 養分は外部から投入するもの | 養分は微生物が循環させるもの |
| 雑草・害虫は排除すべき敵 | 生物多様性は生態系サービスの源泉 |
| 耕起は土を柔らかくする | 耕起は土壌生態系を破壊する |

### 10.2 福岡正信の先見性

福岡正信は、現代の土壌微生物学が明らかにした事実を、半世紀以上前に直観と観察によって把握していた。彼の「無の哲学」は、科学的には「微生物に任せる農法」として再解釈できる。

### 10.3 今後の課題

1. **知識の普及**：土壌微生物学の知見を農家に分かりやすく伝える
2. **経済的インセンティブ**：土壌健康の改善を経済的価値として評価する仕組み（炭素クレジット等）
3. **技術開発**：微生物資材、診断技術、精密農業との統合
4. **文化的変革**：「きれいに耕された畑」から「生きている土壌」へ価値観の転換
